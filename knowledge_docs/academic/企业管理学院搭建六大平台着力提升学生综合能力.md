# 企业管理学院搭建六大平台着力提升学生综合能力

> 来源: https://jwc.hhu.edu.cn/2016/1219/c3147a40517/page.htm

近年来，企业管理学院围绕“立德树人”根本任务，紧密结合学生成长成才需求，积极搭建六大平台，着力提升学生综合能力，取得了显著成效。

学习力提升平台——学院持续开展“若水讲堂”学业帮扶活动，以朋辈互助帮扶，形成良好的学风为目标，为学生在课余时间进行自我学习、自我服务搭建平台，同时通过空间公众号和河海企管小微等新媒体发布，在线下开展学长团授课答疑解惑、四级模拟考场、“奋斗的青春最美丽” 保研经验交流会等活动，营造良好的朋辈学习氛围。

实践力提升平台——引导学生将科技基金项目、大学生创新创业训练计划项目与暑期实践相结合，积极开展“企管雨露，爱满龙城”志愿服务活动，搭建了传统文化志愿服务基地、红色教育志愿服务基地、环保志愿服务基地等平台，逐步建成“项目+团队+基地+管理（线上管理、线下活动的“互联网+管理”模式）”工作模式，学生在奉献社会的同时，能力得到进一步提高。

创新力提升平台——学院通过开展品牌活动“创业大讲堂”邀请校内外创业精英与大学生们开展交流互动活动，同时积极组织学生参加“挑战杯”全国大学生课外学术科技作品竞赛、“创青春”全国大学生创业大赛、中国江苏省创新创业大赛等活动，指导学生申报科技基金、大学生创新创业训练计划，为学生创新力提升提供了积极的引导和帮助。

自护力提升平台——学院积极加强对大学生的安全教育，提高其安全防范意识和自我保护能力，教授基本的防身技能，同时积极推荐学生关注公安系统微信公众号，通过信息推送等形式提高学生防盗防骗的防范意识。

文体力提升平台——学院每年开展摄影、书法、剪纸培训、篆刻培训等培训活动，举办班级合唱比赛、传统文化创作大赛、书画大赛、“新生杯”辩论赛等赛事，创新性开展新生杯篮球赛、“青春三走，凝聚力量”素质拓展活动、趣味性体育锻炼等活动，形成了“人人有体育项目、班班有体育活动”的良好局面，进一步提高了学生的身体素质。

就业力提升平台——连续四年组织“创赢”暑期实习招聘会，紧贴学生需求举办大学生求职培训班，通过专题讲座、分享交流会、模拟面试沙龙等活动有力促进学生就业能力的提升，也进一步提高了就业质量。（胡井军、左丽）
