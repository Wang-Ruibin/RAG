# 公共管理学院多举措提升学生实践能力

> 来源: https://jwc.hhu.edu.cn/2016/1111/c3147a40487/page.htm

在多年教学实践中，公共理学院通过凝练理论教学内涵、强化实践教学环节、拓展课外学习空间等多种举措，全面提升学生实践能力，取得了丰硕成果。

把握时代趋势，凝练理论教学内涵。学院经过多年思考与实践，“与时俱进”的理念已融入到本科理论教学之中，在调整原有课程内容与授课方式的同时，积极调整培养方案，开设媒介融合、新媒体等课程。与此同时，强化经典著作的阅读，提高理论修养，夯实基础。

强化实践教学，增强学生动手能力。学生的实践能力，并不是某项简单的专业操作技能，而是具有理论基础、带有创新意识和开放思维的业务能力。在多年探索中，各个专业通过实践教学基地建设，引导学生接触社会实际，将理论知识融入到实践课程中加以运用，分析解决实践活动中面临的问题。学院积极鼓励教师进行教学改革和研讨，以“翻转课程”为抓手，将创新意识、开放思维和实践能力引入到课程之中。

拓展课外空间，提升学生专业水平。学院努力搭建平台，有效拓展学生在课堂之外的学习空间，形成学院特色。”吸纳学生参加教师的科研项目，鼓励学生积极申报挑战杯等科研项目。出版学生自己组稿编排的刊物《拓荒者》、《小星》，并围绕刊物举行学生学术沙龙等活动。组建从事视频制作的“万象工作室”、致力于无人机航拍的“天空视觉文化工作室”等专业性社团。举行“播音盛典”、“播音毕业汇演”，丰富校园生活。学生在“齐越朗诵节”、“阳光校园 DV 大赛”、“环太平洋大学生微纪录作品大赛”等比赛中屡获奖项，充分展示了河海大学学生风采。 （张纯）
