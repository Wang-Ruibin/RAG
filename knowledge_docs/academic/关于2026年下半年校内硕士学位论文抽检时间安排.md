# 关于2026年下半年校内硕士学位论文抽检时间安排

## 

发布时间：2026-06-18浏览次数：76

各学院（系）：

为促进研究生学位论文质量的提高，我校对硕士研究生学位论文进行校内抽检工作。抽检硕士研究生学位论文范围包括学术型、全日制专业学位和非全日制专业学位。

现就2026年下半年校内硕士学位论文抽检工作做如下通知：

|  |  |
| --- | --- |
| 月份 | 报送时间 |
| 6月 | 25日 |
| 7月 | 2日 |
| 8月 | 10日 |
| 9月 | 1日、16日、28日 |
| 10月 | 12日、26日 |
| 11月 | 9日、23日 |

请各学院（系）务必通知到相关导师和每一个毕业研究生，避免因时间安排不当而影响正常毕业，同时请各学院（系）做好学位论文把关工作，提高研究生培养质量。
