# 力材院建立产教融合高素质创新人才培养机制

> 来源: https://jwc.hhu.edu.cn/2016/1026/c3147a40461/page.htm

力学与材料学院以本科教学审核评估及工程力学品牌专业建设为契机，在继承力学专业优良教学传统的基础上，坚持全面深化教育教学改革，面向社会需求，不断深化本科教育教学改革，形成了具有工程特色的高素质创新人才培养体系。

依托雄厚的师资力量与丰富的教学资源，学院工程力学国家重点学科牵头建设了江苏高校重大基础设施安全保障协同创新中心，并承担了培养和输送高素质创新人才的任务，优化调整专业培养方案，强化实践教学环节，采取多元、多样的产学研联合培养模式，形成了规范性协同创新培养方案，建立了协同创新的人才培养教学理念，选拔优秀本科生进入创新平台或团队，参与科研实践，使人才培养与科研活动紧密结合，同时组建了跨地域的开放式人才培养基地，学生可以根据个人特长和兴趣，申请到不同单位参与到各种科研活动和工程现场。

通过建立产教结合人才培养模式，学生创新创业与实践能力明显提升，学院本科生培养成果显著，在近两届国际力学竞赛、全国周培源大学生力学竞赛、江苏省大学生力学竞赛、全国大学生金相技能大赛中，多名学生取得优异成绩，累计获奖项目90余项；自2013年以来，学生获得国家级大学生创新训练项目30项，省级创新训练项目16项，授权或申请专利11项，发表论文13篇。（李静）
