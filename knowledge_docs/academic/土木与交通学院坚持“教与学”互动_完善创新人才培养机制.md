# 土木与交通学院坚持“教与学”互动 完善创新人才培养机制

> 来源: https://jwc.hhu.edu.cn/2016/0406/c3147a40449/page.htm

土木与交通学院围绕师生互动、教学互动、学研互动的新型“教与学”关系，积极推进本科教学教育改革，在“双主”特色人才培养模式引领下，以创新能力培育为导向，注重课堂教学与课外拓展相结合，开设土木大讲堂、土木微讲堂，开通微信平台，通过线上线下联动，寓教于学，教学相长，实现教与学良性互动，营造出良好的创新人才培育氛围。 土木大讲堂，专家水准引领风范。学院开设了土木大讲堂系列活动，邀请知名教授、行业专家、校园名师走进讲堂与学生互动，让学生感受名师风范、接触专家视野，为学生搭建起与名家大拿沟通交流的平台。目前大讲堂已连续举办40期，参与学生6000余人次，讲座内容涵盖土木与交通行业多方面的专业前沿知识，在学生中引起良好反响，有力地推动了学生专业素养提升，创新能力提高。 土木微讲堂，寓学于教共享得失。学院以优秀本科生为主讲人开设了多期土木微讲堂，为全院学生分享学业、创新、竞赛、就业等经验，提供科创竞赛辅导，提升学生课外拓展活动的参与度，增强学生科技创新意识。目前微讲堂已成功举办22期，覆盖本科全部年级，对课堂内教学形成了有力补充和支撑。 网络新媒体，宣传展示多向互动。学院积极打造本科教学和学院团委微信平台，不断强化新媒体阵地建设，将线上线下宣传相结合，极大的激发了学生参与科研创新的热情。平台累计推送102期，涵盖本科教学、科学技术、学科竞赛、考试、留学、就业、学生活动等多方面内容，及时传播业界最新资讯，促进教学、科研、实践紧密集合，增强学生科研创新意识，营造了较好的教学互动氛围。 以土木大讲堂为依托，以土木微讲堂为纽带，以微信新媒体为平台，三方联动全面推进第二课堂建设，丰富教学互动模式，土木与交通学院在实践中探索出一条创新型人才培育之路。三年来，本科生论文专利发表数屡创新高，如2015年我院本科学生授权和申请专利78项，为历年最高。专业学科竞赛屡获殊荣，省部级以上获奖数百人次，夺得江苏高校历史上第一个全国大学生结构设计竞赛冠军、获得全国大学生交通科技大赛一等奖（专业组第一名），连续三次蝉联全国土木工程大学生创新实践竞赛一等奖，诸多学科竞赛中均刷新历史最高佳绩。（吴昊）
