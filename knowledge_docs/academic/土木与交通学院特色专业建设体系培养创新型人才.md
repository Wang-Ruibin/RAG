# 土木与交通学院特色专业建设体系培养创新型人才

> 来源: https://jwc.hhu.edu.cn/2016/1021/c3147a40457/page.htm

土木与交通学院以本科教学审核评估及土木工程专业认证为契机，继续大力推进本科教育教学改革，在以教师为主导、学生为主体的“双主”特色人才培养模式引领下，致力于打造系列专业建设体制机制，为学院专业建设发展及创新型人才培养提供具有鲜明特色的保障体系。“双主”特色人才培养模式的突出成果《河海大学土木与交通学院本科教学全程导引》更是作为范本于本科教学审核评估年在全校得到推广和应用。

致力打造一流专业类质量保障体系。学院近几年陆续出版了一套六部150多万字的《河海大学土木类本科教学系列蓝皮书》，对学生从学业规划到创新训练以及职业规划等内容作了全方面的引导，不仅使得7届本科生受益，在全校得到推广，另有20多家高校纷纷学习借鉴蓝皮书的做法。

致力打造一流专业类创新训练和学科竞赛培育体系。学院注重加强创新能力实训，多视角多层面引导学生开展相关活动，近年来专业类学科竞赛成绩突出，获得了全国大学生结构设计竞赛特等奖、全国土木专业大学生创新实践成果竞赛特等奖、全国大学生交通设计竞赛一等奖、美国土木工程大学生竞赛钢桥赛季军等荣誉。

致力打造考核明确、互动灵活、实效突出的新型导师制。学院积极推行学业及专业导师制，对学生创新能力实行点对点培养，2013年以来共有4届近400名学生经历导师制培养，取得了大量有显示度的创新成果，同时也增强了学生对学院和专业的认同感和向心力。 （陈鹏）
