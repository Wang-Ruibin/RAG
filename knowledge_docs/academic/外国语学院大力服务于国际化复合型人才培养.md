# 外国语学院大力服务于国际化复合型人才培养

> 来源: https://jwc.hhu.edu.cn/2015/0505/c3147a40445/page.htm

外国语学院围绕学校“水利特色、世界一流”发展目标框架下人才培养国际化的要求，结合学院专业特色，多措并举，积极为学校国际化复合型人才培养提供服务。

立足专业辐射全校。学院利用自身学科及师资优势，在实施好大学外语教学和英语专业教学的同时，自 2014 年秋季起，针对 2013 级非英语专业学生开设英语双学位培养项目，制定了相应的应用型、复合型人才的培养方案。首批参加英语双学位项目的学生近 100 人，计划经过几年发展，将该项目培养规模提高至 500 人以上。

针对需求个性培养。针对不同学科学生的专业需求，开设了亚非拉文化、国际工程谈判、商务英语、外贸函电、外事礼仪、西班牙语和法语等个性化选修课程。在提高学生英语基本能力的同时，着力加强学生对于语言文化知识的认知及双语转换和运用的能力，使学生具备从事各类涉外工作的能力。

加强对接订制服务。学院不断加大与其他兄弟院系的合作与交流，根据不同学院的需求提供专业化语言培训。如，承担环境工程国际合作班的雅思培训和写作培训课程，为参加赴法项目的学生提供法语培训等，做好学生出国深造、国际交流语言培训的服务工作。 （储晨雪）
