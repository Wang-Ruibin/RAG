# 学位点建设风采 | 智能科学与技术：智承四十载 交叉筑根基

## 

发布时间：2026-07-01浏览次数：13

河海大学智能科学与技术学位点，始终紧跟数字中国、智慧水利、国家水网、国家水安全等战略需求，面向水利工程安全监测、数字孪生水利、水资源优化配置等典型场景，以“人工智能 + 水利”交叉融合为核心发展主线，历经四十余年积淀，形成体系完备、平台一流、队伍精锐的鲜明特色。

**溯源逐梦：薪火相传，踏浪而行**

河海大学是全国最早布局信息技术领域的水利院校之一。1978年，学校设立电子计算机专业，聚焦大型水利工程科学计算，为行业输送首批信息化技术人才。

四十余载砥砺前行，紧跟行业转型步伐，率先布局水利信息化、数字化、智能化赛道：2003年设立国内首个 “水信息学” 二级学科，推动国家自然科学基金委增设 “水力学与水信息学” 专项研究领域；依托学校水利学科顶尖优势，逐步搭建人工智能交叉学科体系，2024年获批建设江苏省水安全学科交叉中心、2025年入选首批江苏省人工智能学院。从工程水利、数字水利到如今的智慧水利，始终以技术革新引领行业变革，发展根基愈发坚实。

**交叉立科：打破学科壁垒，打造水利AI 独有优势**

多学科深度交叉是学位点最核心的办学特色与核心竞争力。立足学校水利、土木、环境等传统优势学科，融合计算机科学、人工智能、自动控制、大数据、电子信息等信息技术学科，构建起“AI+ 水利”的交叉融合发展生态。

在顶层设计上，建立跨院系、跨学科协同机制，整合校内水利工程、水文水资源、电气自动化等学科资源，设置交叉研究方向、交叉课题、联合培养项目，彻底打破传统学科边界；在发展定位上，不做通用型人工智能研究，而是聚焦水利行业特有场景，针对水文预报、工程运维、流域调控、数字孪生等行业痛点开展定向攻关，形成“别人不可复制、行业不可替代” 的交叉学科优势，也为学位点建设筑牢特色根基。

**平台赋能：立体平台矩阵，支撑交叉创新研究**

高能级平台是学位点发展、交叉科研的重要载体。目前已建成国家级+省部级多层级、全覆盖的科研平台集群，形成水利领域独具优势的创新阵地：

拥有水资源高效利用与工程安全国家工程研究中心等国家级平台，以及水利部水利大数据重点实验室等一批省部级重点平台，支撑水利工程智能感知、工程安全监测、水文智能预报与数字孪生流域建设。

搭建国际联合实验室、111 引智基地等国际合作平台，汇聚全球智力开展交叉课题研究。

建成水利领域少有的“全国重点实验室+国家工程研究中心+国际联合实验室” 一体化平台体系，可全面支撑基础理论、技术研发、工程应用全链条交叉研究，为师生开展科研创新提供一流硬件与环境保障。

**师资聚力：引育并举，建强梯队与特色团队**

坚持“人才是第一资源”理念，构建 “引-育-联” 三位一体人才建设体系，打造了一支懂水利、精智能、善交叉的高水平师资队伍，并形成多支结构稳定、战斗力强的教学科研团队。
