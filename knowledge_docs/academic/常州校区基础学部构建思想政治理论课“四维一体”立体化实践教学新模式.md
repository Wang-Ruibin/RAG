# 常州校区基础学部构建思想政治理论课“四维一体”立体化实践教学新模式

> 来源: https://jwc.hhu.edu.cn/2016/1208/c3147a40509/page.htm

近年来，常州校区基础学部立足 多维立体，强化整体协同，坚持 问题导向，促进有效互动， 构建了以“课堂实践+校园实践+社会实践+在线实践”为特征的“四维一体”思想政治理论课立体化实践教学模式， 充分发挥教师的主导作用和学生的主体作用， 实现了大学生思政课实践过程的全员参与、实践成果的科学评价目标。

以“大班小组法”为手段开展思政课课堂实践教学。教师将“大班小组法”运用于思政课课内实践环节。在课堂实践教学中，教师发挥主体引导作用紧扣理论课程内容，采用包括围绕问题的主题辩论、专题讨论、即兴演讲、情景展示和案例分析等多种实践教学环节相结合的方式，引导学生自主探索和合作交流。

以“主题探究式”为目标开展思政课校园实践教学。“主题探究式”实践教学法以时事主题为教学中心，围绕主题，通过讨论、思辨、研究性学习等多环节和形式来开展的一种实践教学形式。如，2015上半年围绕“纪念中国人民抗日战争胜利70周年”这一时事主题开展系列实践活动。2016年围绕“孙中山诞辰150周年”和“中国工农红军长征胜利80周年”主题，开展了“孙中山与近代中国”、“中国梦 新征程 长征精神代代传”主题探究演讲赛。

以“现场教学法”为载体开展思政课社会实践教学。思政课教师组织学生在常州市三杰纪念地廉政教育馆开展的“廉洁诚信宣誓仪式”现场教学，该实践教学活动获得光明网、江苏省纪委清风扬帆网和常州日报等媒体广泛关注。教师还开展了“铭记历史，关爱老兵”校外现场教学实践活动，活动获得中国新闻网、扬子晚报等十多家媒体广泛报道。

以“翻转互动法”为手段开展思政课在线实践教学。在线实践教学是指以现代网络技术手段为平台的实践教学活动，是新媒体环境下对思想政治理论课传统实践教学的一种创新。2015年以来，思政课程组申报了３门翻转课堂教学改革项目。《马克思主义基本原理概论》课教师利用河海在线课堂平台提供的“讨论”功能，使学生们有了畅所欲言的场所。（赵蓉、张金城）
