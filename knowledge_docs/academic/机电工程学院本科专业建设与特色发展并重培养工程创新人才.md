# 机电工程学院本科专业建设与特色发展并重培养工程创新人才

> 来源: https://jwc.hhu.edu.cn/2016/1219/c3147a40512/page.htm

近年来，机电工程学院坚持本科专业建设和特色发展并重，培养工程创新人才，取得了丰硕成果。

学院以专业建设为龙头，持续优化教学资源，改革人才培养模式，形成了“创新型工程人才培养体系的构建与实践”、“依托机械基础实验中心建设培养机械类拔尖创新人才”等十余项省、校级教学成果，机械工程专业被评为国家综合改革试点专业，教育部“卓越工程师”试点专业，江苏省品牌专业，能源与动力工程专业被评为国家特色专业。

学院以卓越计划为契机，加强校企协同，创新人才培养，形成了“校企协同，五措并举”机械类卓越工程师人才培养模式，机械工程专业入选首批教育部“卓越工程师”试点专业，先后建设校外基地２个，获得省优毕业设计团队１项，省优本科毕业设计２项，省级教学成果奖２项，获批省级教学改革项目２项。188名卓越工程师计划学员顺利毕业。

学院以创新创业为导向，推进点面结合，提升创新能力，足球机器人“e龙队”获得2项世界冠军、5项世界亚军，50余项全国冠军，学生设计作品曾获得德国“红点”设计大奖、美国“Idea设计大奖”、德国“IF”设计奖，学院学生还在“挑战杯”全国大学生课外科技作品竞赛、全国大学生机械创新设计大赛、全国大学生英语竞赛、全国大学生数学建模竞赛中取得优异成绩。

学院以水利行业为背景，建设特色方向，融合工程意识，其中特色专业金属结构和焊接着力培养兼具工程设计开发能力和管理能力的高等专门人才，600余名毕业生有一半在在行业内就业，多人次获得“大禹奖”、“鲁班奖”等国家级高层次行业奖项。（丁坤、楼力律）
