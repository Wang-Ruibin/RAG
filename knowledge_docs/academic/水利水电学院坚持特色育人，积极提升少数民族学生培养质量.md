# 水利水电学院坚持特色育人，积极提升少数民族学生培养质量

> 来源: https://jwc.hhu.edu.cn/2016/0407/c3147a40452/page.htm

水利水电学院有48名新疆生，28名西藏生，占据全校民族生近1/4，为促进民族学生适应并融入大学生活，顺利完成学业，学院以提升学业成绩为抓手，以培养综合素质为导向，注重学生体育锻炼，加强民族文化建设，通过建立“乐水学堂”互助平台、实施“私人订制”导师工程、举办“民族风采”体育赛事、打造“水育明珠”社区品牌，不断提升少数民族学生培养质量。

“乐水学堂”，仁者乐水立本学堂。 学院成立“乐水学堂”，在全校范围内聘请“教学小能手”为讲师，以讲座的形式为全院少数民族同学讲解难题、分享学习经验；在课程设置中，不仅有“高数专题班”、“物理专题班”、“力学专题班”，还特别设置水文化课程、管理课程、人文课程等，通过聆听和交流，使学生潜移默化中感受人水和谐的重要意义，促使民族生与汉族同学之间互帮互助,共同提升自己的学业水平。

“私人定制”，三位一体助推成长。 学院积极推动“私人订制”导师工程，通过导师自主申报、少数民族学生自主选择、学院少数民族工作组审核，构建“三位一体”的少数民族学生结对政策，以 实施“灵魂塑造师”，加强少数民族学生思想教育；以实施“校园引导师”，强化少数民族学生社会责任感；以实施“课业指导师”，提升少数民族学生学习能力和综合素质；以实施“前景规划师”，提升少数民族学生就业水平， 切实加强少数民族学生管理和服务工作，助推少数民族学生健康成长。

“民族风采”，加强锻炼共展风采。 每年举办“民族风采活动月“系列体育活动，通过开展足球赛、篮球赛、羽毛球赛等，鼓励少数民族同学走出课堂、走出宿舍、走向操场，锻炼自身的体魄，提高身体素质。

“水育明珠”，弘扬文化共建家园。 在社区文化建设方面，学院重点打造“水育明珠”系列品牌活动，针对民族生宿舍，重点开展民族生美化宿舍项目，通过网络平台将线上线下宣传相结合，极大的激发了少数民族同学弘扬民族文化的热情，同时也鼓励民族同学将自己民族的风俗特色融入到宿舍的装饰中来，美化自己日常生活休息的一片天地，让他们更好地融入校园生活中。

近三年来，水电学院的新疆、西藏少数民族学生培养质量不断提升，在顺利完成学业的基础上，每年都有民族生被免试推荐为研究生， 9位民族生在各类学生组织中担任学生干部，20多位民族生取得各类优秀学生奖学金及校级以上荣誉称号；2011级马锐同学被新疆水利设计院录用；2012级杰德尔别克同学荣获“全国水利院校未来之星”、“中国大学生自强之星”、“江苏省优秀学生干部”等各类荣誉称号；2013级胡安什别克同学学习成绩突出，绩点高达4.5等等。（刘津铭）
