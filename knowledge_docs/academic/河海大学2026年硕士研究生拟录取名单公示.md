# 河海大学2026年硕士研究生拟录取名单公示

> 来源: https://gs.hhu.edu.cn/2026/0509/c3624a326366/page.htm

各相关考生：

根据教育部、江苏省教育考试院和学校有关文件要求，经各单位组织复试、各单位公示、学校审核，现将我校 2026年硕士研究生拟录取名单统一公示（见附件）。

1.本次公示名单为拟录取名单。若对公示名单有异议，请于2026年5月15日17:00前实名向研究生招生办公室进行书面反映。联系电话：025-83786303，邮箱：hhuyzb@hhu.edu.cn。

2.拟录取为全日制定向和非全日制的考生须与学校、用人单位签订定向培养协议。 材料提交相关事项另行通知。

3.拟录取考生经学校公示、教育部批准后方可正式录取。

4.我校将对拟录取考生进行思想政治素质和品德考核，不合格者将不予录取，具体安排另行通知。体检工作入学时统一开展，体检不合格者按有关规定处理。

5.我校后续将根据思想政治素质和品德考核、定向协议签订、档案寄送等情况陆续发放录取通知书，请考生及时关注相关通知。
