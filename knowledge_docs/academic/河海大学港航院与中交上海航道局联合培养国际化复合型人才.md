# 河海大学港航院与中交上海航道局联合培养国际化复合型人才

> 来源: https://jwc.hhu.edu.cn/2017/0517/c3147a40524/page.htm

近日，中交上海航道局有限公司副总经理曹湘喜一行访问我校，党委副书记郭继超接待了曹湘喜副总经理一行。双方洽谈了联合培养学生事宜。

根据协议，双方将依托港航学院，通过校企深度合作、依托企业行业优势和学校师资优势，推出工程技术模块、现代企业管理模块、葡萄牙语言模块、中西方文化模块、工匠技能模块、社会实践模块等六大能力提升方案，通过一年左右的时间，联合培养具有国际视野、通晓商务知识、精通所在国语言、熟知所在国文化等的高层次国际化复合型人才。这次深度合作是港航院积极服务国家“一带一路” 倡议 的行动之一，也是学院认真落实本科教学工作审核评估整改意见，推进建设“中国特色、世界一流”学科的重要举措之一。

学生处、港航学院等单位负责人参加了活动。（张继生、胡忠华）
