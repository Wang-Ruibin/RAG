# 港口海岸与近海工程学院创新校企、军地协同的卓越工程师培养模式

> 来源: https://jwc.hhu.edu.cn/2016/1115/c3147a40490/page.htm

港口海岸与近海工程学院始终以学生发展、社会需求作为人才培养的目标和导向，密切跟踪行业发展，推进人才培养模式改革和实践，取得丰硕成果。港口与航道工程专业先后入选教育部和海军国防生卓越工程师教育培养计划，现有国家级工程实践教育中心、实验教学示范中心、虚拟仿真实验中心等一系列国家级实验教学平台，2012年全国首家通过工程教育专业认证，2015、2016连续两年在中国大学最佳专业排行榜排名第一。

以行业企业和国防建设需求为导向，校企军地共同构建新时期港航领域创新型卓越工程技术人才的知识体系。与行业专家共同组建专业建设指导委员会，制订人才培养方案。校企合作培养学生工程实践能力贯穿四年教学过程。一年级开设“创新思维训练与实践”课程；二、三年级选择企业技术培训课程纳入课外拓展，实施模块化选修；四年级实施企业和学校双导师制，结合企业生产科研项目开展各类综合训练。

以实现高质量工程教育为目标，校企共同组建满足卓越工程师培养要求的高水平多元化师资队伍。强调校内教师的工程背景，提高教师的工程实践能力，并积极组建企业教师队伍，培养工程技术人员的教学能力。利用企业教师到校授课、指导毕业设计等机会，组织课堂教学观摩，利用远程教育平台开展教学培训等活动，提高工程技术人员的教学能力。

以理论实践交叉融合为主线，校企共同实施从基础到专业的多元立体化课程建设。在传统教材中增加典型工程案例强化基于项目和案例的研究型教学，针对企业工程实践课程新编相关讲义。发挥校企合作优势，将课件演示、课堂讲授、实验教学有机结合。建设各类工程多媒体教学资源库及工程物理模型试验视频资源库辅助专业课教学。

以强化工程实践能力培养为目的，校企共同开展多模块层次化的实验实践条件建设。工程概念贯穿基础实验、专业实验和研究型实验三大模块，实现基本型、提高型和创新型三个层次的实验训练。与企业和军队共建实习实践基地，满足专业各方向参观实习、讲座培训和动手实践的需要。（陈达、戴菲）
