# 物联网工程学院全面深化人才培养改革持续提高人才培养质量

> 来源: https://jwc.hhu.edu.cn/2016/1206/c3147a40501/page.htm

物联网工程学院高度重视本科人才培养工作，紧密结合国家战略和经济社会发展新形势，全面深化人才培养模式改革，不断提高人才培养质量，取得了丰硕成果。

积极开展人才培养模式改革探索。学院通过“卓越计划”开展工程型卓越人才培养，强化校企协同，将学校的培养方案与企业培养有机融合，全面提升学生实践操作和解决问题的能力，也为他们踏上工作岗位打下基础。通过“英才计划”开展学术型英才培养，加强导师队伍建设，注重将理论学习与科学研究、实践能力有机结合。

以校企合作为核心进行课程改革。学院与相关领域知名企业IBM、EMC、赛灵思（Xilinx）公司等技术背景和成熟平台联合建设和开发相关课程，并结合实际工程问题设计案例教学，其中云计算技术及应用课程和数字电路Verilog HDL及系统设计课程建设分获教育部－IBM产学综合改革项目和教育部-Xilinx公司产学合作专业综合改革项目。

加强创新创业教育。加强科教融合，形成了本科生实践创新能力团队培养模式，包括科教协同的团队建设理念、整体协同的团队集结模式、分层协同的团队管理制度、多元协同的团队发展平台、双向协同的团队互动机制等方面，并结合学院专业特色制定了大创基地科技创新攀登计划，成立了8支计划团队，全方位提升学生实践能力。

深入开展教学研究。2013-2015年申请成功多项国家和部省级教学学术研究、教学质量工程项目，经费1000多万。在国家核心期刊《高等工程教育研究》、《实验室研究与探索》、《实验技术与管理》等杂志上发表高水平教学学术论文35篇。

全面加强专业建设。目前，学院5个本科专业中有3个国家特色专业、2个教育部首批“卓越计划”专业与专业综合改革试点专业、1个江苏省高校品牌建设专业、3个江苏省“卓越计划”专业和1个江苏省特色专业。

经过多年持续改革和努力，学院在人才培养方面捷报频传。教师倾心本科教学、专注教学研究，学生奋发成才、比学赶超已蔚然成风，学院本科人才培养质量全面提升，教研成果获江苏省教学成果奖一等奖2项，二等奖2项。本科生升学出国率保持在35%左右，其中2016届升学出国率近38%；学生在包括“挑战杯”在内的省部级以上科技创新竞赛中获奖239项，并涌现出以冯全为代表的一大批创业典型。（朱金秀、周燕、邱华）
