# 理学院多措并举筑牢学生理学基础

> 来源: https://jwc.hhu.edu.cn/2017/0327/c3147a40521/page.htm

日前，从第八届全国大学生数学竞赛现场传来捷报，由理学院郑苏娟、朱永忠、杨永富等教师指导，大禹学院龚正、史立地同学组成的团队双获总决赛一等奖。

近年来，我校学生在代表数学最高水平的全国大学生数学竞赛和全国大学生数学建模竞赛中屡获佳绩，这与理学院坚持为打牢学生理学基础提供一流的教学分不开的。理学院紧紧围绕服务学校“双一流”建设的中心工作，外引青年才俊，内强师风师德，教学质量常抓不懈，为人才培养予助力导航，给优势学科予人才支撑，助推学校“双一流”建设，努力使数学、物理类学科成为实现学校“双一流”建设的智力库、技术库、人才库。

持之以恒抓好师德师风教育。 一方面，学院将师德教育摆在教师培养的首位，贯穿教师职业生涯全过程。利用全院大会、教学学术研讨会、思想政治教育活动、党支部组织生活会等开设师德教育专题，积极引导广大教师做有理想信念，有道德情操，有扎实学识，有仁爱之心的党和人民满意的好教师；另一方面，把握正确舆论导向，坚持师德宣传制度化、常态化，充分利用教师节重大节日、纪念日，利用校报、网站及微博、微信等新媒体集中宣传优秀教师的先进事迹，把培养良好的师德作风作为校园文化建设的核心内容。

切实加强教学管理质量和教风、学风建设。 学院结合“两学一做”专题学习教育实际，强化教学质量管理，营造浓郁的学风、教风。第一，实施青年教师导师制，为青年教师配备导师进行传、帮、带，指导和帮助青年教师过好教学关和科研起步关，帮助年轻教师提高思想认识与教学水平；第二，学院始终坚持教育与研究相结合的原则。充分发挥自己的教学优势，深化教学改革。第三，以建立“全员育人、全过程育人、全方位育人”的机制为保障，引导学生树立明确学习目标，端正学习态度，增强学习动力，着力提高学习能力、实践能力和创新能力。

提供强有力的学科支撑。 长期以来，理学院广大教师兢兢业业，为学校优势学科培养出一大批宽基础、具有扎实理学根底的优秀人才。一是通过转专业、保研、考研为我校特色优势学科输送了大量的具有扎实理学基础知识的优秀学生；二是鼓励年青教师进入优势学科博士后流动站，开展跨学科科研合作和学术交流，为学校的科研工作注入新活力，提升科研成果的层次；三是鼓励教师积极参与挂职锻炼、教学答疑等志愿服务活动，以“学无止境，教亦无止境”的教风感染人。（应海华）
