# 能电院推进“三个强化” 以高质量党建引领高质量发展

> 来源: https://jwc.hhu.edu.cn/2022/0810/c3147a239604/page.htm

近五年来，能电院“电气工程及其自动化”“能源与动力工程”专业被评为国家一流本科专业，“电气工程”学科入选“十四五”江苏省重点学科，交叉学科“新能源科学与工程”二级博士点获批，自动化专业通过电信类专业认证；学院生师比由19.21优化至15.03，新增科研经费从2016年的1900.14万元，上升到2020年的6012.11万元，高质量论文从2016年的31篇，上升到2020年的141篇，2021年学院获批国家自然科学基金项目16项；自动化专业获首批河海大学课程思政示范专业，并入选学校2021年课程思政示范课程建设项目；获“创青春”全国大学生创业大赛金奖、“挑战杯”全国大学生课外学术科技作品竞赛一等奖、“挑战杯”红色专项活动全国二等奖、江苏省特等奖；获江苏省优秀博士论文、优秀硕士论文；留学生培养质量不断提升，老挝班毕业率100%……2018年学院入选首批“全国党建工作标杆院系”，并于2020年顺利通过验收。学院交出了一份亮丽的“成绩单”，实现以高质量党建引领学院高质量发展。

能电院以习近平新时代中国特色社会主义思想为指导，以政治建设为统领，聚焦立德树人根本任务，实施“全国标杆院系建设工程”，统筹开展“领航计划、强基计划、先锋计划、廉心计划、聚能计划”五项计划，持续强化政治领导力、阵地聚合力、发展推动力，把政治标准和政治要求切实贯彻到党的各项建设中，促进学院建设质量全面提升。

强化政治领导力 对标对表同向同行

忠诚源于理论清醒，追随来自信仰坚定。学院建立并严格落实政治理论学习“第一议题”制度，坚持把习近平新时代中国特色社会主义思想、习近平总书记重要指示批示精神、重要讲话精神作为学院党委中心组学习“第一议题”，全面系统学、及时跟进学、深入思考学、联系实际学。坚持和加强党的全面领导，修订《能源与电气学院党委会议事规则》《能源与电气学院党政联席会议议事规则》，严格党委会先行政治把关程序要求，严格党政共同负责，明晰党政权责边界。

旗帜指引方向，方向决定道路，学院坚持把学习成果转化成为学院高质量发展“金钥匙”，推动实施“一个中心三项联动”学习机制，始终坚持“学习贯彻习近平新时代中国特色社会主义思想”这个中心，推进“党内-党外联学”，学院民主党派人士积极参与学院政治理论学习，院长作为致公党河海大学支部副主委全程列席中心组学习，党政共同推动总书记重要指示批示和党中央决策部署在学院落地生根。推进“干部-师生联学”，制定《能电理论学习月刊》，党员领导干部定期列席支部组织生活会，干部师生理论学习同步推进，干事创业同频共振。推进“党员-群众联学”，学院将支部组织生活会和教职工理论学习结合起来，将党日活动与党群联建结合起来，推动干部师生凝心聚力、同向同行。

强化阵地聚合力 联动联建提质增效

根深才能叶茂，本固才能枝荣。能电院坚持“支部建在连上”，工作在哪里，党组织就建在哪里，党组织作用就发挥到哪里。实施“纵向联动”，将学生党支部从传统的“班级建制式”转为“课题组式”“专业式”党支部设置，实现党建工作与人才培养协同联动、深度融合，支部建设工作法获得江苏省高校党建工作创新奖一等奖。推行“师生联建”，广泛建立教工系所之间、师生之间、年级之间的支部联建互动网络，将教师丰富的专业理论和学生多样的活动形式有机结合，实现优势互补、经验互通、成果共享。组织“头雁联盟”，学院制定实施《能源与电气学院特色党日活动、党建工作创新、党支部书记工作室建设立项管理实施细则》等，凝练支部特色，总结推广经验做法，不断提升党建工作质量；同时，通过定期开展党支部书记交流培训，每半年进行一次支部总结考核，选派优秀青年教师担任研究生党支部书记等，搭建党支部书记党务、业务能力双提升的载体平台。打造“先锋讲堂”，邀请行业优秀党员代表、校友，以及学生中自强不息、具有先锋模范作用的党员学生分享奋斗经历，传承先锋精神。以创建学校“样板党支部”“特色党支部”“优质党支部”为导向，基层党组织建设质量全面提升，支部战斗堡垒作用不断增强。

强化发展推动力 党建引领聚能攻坚

红色基因里深藏着共产党人的精神密码。能电院主动服务国家“碳达峰、碳中和”等重大战略需求，以强化育人育才为根本，以整合队伍资源为抓手，开展“党建引领，聚能攻坚”行动，一批“党建引领+项目”的实施，使各级党组织和广大党员的战斗堡垒和先锋模范作用充分彰显。

实施“党建引领+队伍建设”，汇集师资队伍潜能。学院以理想信念教育为引领，将学科特点与服务国家重大战略需求紧密融合，整合师资队伍资源，引导广大教师有大格局、建大团队、做大项目。以师德师风建设为根本，在教师入职聘用、职称评审、人才推荐、评优评先、选拔任用等方面严把政治关。以先进典型示范为抓手，综合运用荣誉称号、事迹报告、媒体宣传等方法，将思想品德好、教学能力强、科研水平高的教师作为先进典型，引导广大教师不忘立德树人初心，牢记为党育人、为国育才使命，积极探索新时代教育教学方法，培养德智体美劳全面发展的社会主义建设者和接班人。

实施“党建引领+人才培养”，汇集教师学生才能。把好社会主义育人方向，强化全员联动育人，打造院党委-党支部-党员“纵向到底”、辅导员-任课教师-班导师-同学朋辈“横向到边”的育人网络。改革推进“新能源-水电-电网（新-水-电）协同”的课程思政联合培养模式，建立学生专业思政导师四年一贯制培养机制，构建“发电站-电网公司-电力制造企业”全程一体化综合实践平台，将学习研讨与专题辅导、理论学习与调查研究、“走出去”与“请进来”结合，让学生在大平台、大实践中提升服务本领，实现政治素养与专业素养“双提升”。

实施“党建引领+科技创新”，汇集行业单位动能。学院定期邀请校内外专家围绕基金项目申报、科研论文撰写等开展专题讲座，组织引导青年教师瞄准国家重大战略需求，充分发挥学科特色优势，着力解决关键核心技术问题，开展科技攻关和工程应用。在此基础上，主动对接发电集团、流域机构，与国网无锡供电公司、南瑞集体等单位结成支部共建单位和创新实践基地，开展党建和业务交流，争取优质资源，积极开展科研成果转化合作。

学院把事业发展成果作为检验党建工作成效的重要标准，通过不断加强政治建设，不断提升治理体系和治理能力，形成党委工作统领，学科建设强基，科学研究引导，人才培养核心，师资队伍保障，国际合作助力，内部管理激励的高质量特色发展态势。（能电院）
