# 计信院立足专业 强化创新创业

> 来源: https://jwc.hhu.edu.cn/2015/0325/c3147a40442/page.htm

计信院在学生科技创新工作中立足专业特色，紧跟时代步伐，注重科技创新、开展学创结合，多措并举，构建符合专业特色的创新创业之路。

瞄准目标齐奋起。 学院以“多创优创”为目标，从政策、措施、团队及环境建设上精心保障。认定并计算创新创业指导教师工作量鼓励教师参与；组建创新实践团队，形成院领导直接牵头，凝聚指导教师、评审专家、创新实训学生和创新实践组织部门的联动团队；通过校企合作平台，打造创新创业优良的软硬件环境。

立足专业求创新。 学院以国家、省级重点学科与优势学科为依托，加强平台与专业指导，支持与鼓励学生立足专业参与技术与思维创新，参加各级各类科学技术类、创业类竞赛与创新实践活动，使学生在动脑与动手中夯实专业理论基础，熟悉行业发展趋势。近五年来，学生获省部级奖项逐年增加，共计近 500 人次， 2014 年创历史新高 153 人次。

强化特色共创业。 学院继续强化计算机科学与技术、通信工程国家级特色专业，创建互帮、互助、互学的创业机制，开展个性化培育。近年来，学院定期开展创业培训与创新先锋经验传教活动，选树典型，以成功者激发未动者，以经验者带动起步者，相互学习，共同创业。 2014 年，学院博士生傅晓带领的团队获 “创青春 2014 全国大学生创业大赛”挑战杯金奖，激发了更多学生的创业热情。

近年来，学院毕业生创业人数不断增加，在读生有创业意向者较多，目前创业的方向主要是软件系统研发，移动端 APP 开发等。（金明珠）
