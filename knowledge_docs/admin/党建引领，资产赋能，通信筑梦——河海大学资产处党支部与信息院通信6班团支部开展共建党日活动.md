# 党建引领，资产赋能，通信筑梦——河海大学资产处党支部与信息院通信6班团支部开展共建党日活动

党建引领，资产赋能，通信筑梦——河海大学资产处党支部与信息院通信6班团支部开展共建党日活动 来源：资产处 发布时间：2026-06-11 为深化党建引领，牢固树立以师生为中心的发展思想，践行正确政绩观，促进管理与育人深度融合，6月11日，河海大学资产处党支部与信息科学与工程学院通信6班团支部联合顺利开展共建党日活动。本次活动通过座谈交流与实践研学相结合的方式，搭建师生沟通桥梁，聚焦师生所需所盼，扎实办好实事、破解现实难题。 活动第一阶段在夏金仙学生服务中心会议室举行。资产处处长康红强、副处长李志强及部门党员代表，信息科学与工程学院党委书记张伟建、2025级辅导员曹滟梅及通信6班团支部学生共同参加。活动由张伟建主持。 康红强指出，本次共建活动是落实学校党委“党建带团建”要求、推动机关部处中层干部与学生班级结对联系机制的重要实践。他介绍了资产处工作的核心目标是通过制度优化与服务升级，支撑学生健康成长。他强调，通信工程专业的学科优势可为资产管理提供青年视角与技术建议，双方要坚持“资源共享、优势互补、共同提升”的原则，畅通学生意见反馈渠道，搭建实践育人平台，真正实现管理实践与专业学习的双向赋能。 张伟建强调，支部共建不仅是形式上的联谊，更是育人资源的深度链接与一堂行走的思政课。资产处管理着学校的“家底”，通信学子未来要搭建信息的“桥梁”，联结二者的是共同的服务初心和红色基因。 交流发言环节，资产处青年教师代表焦雅，通信6班学生代表刘子淇、郭高瑾分别从优化服务工作、学习生活和学习成长等方面进行发言。与会师生围绕学习空间管理、宿舍生活设施和场馆管理细节进行深入交流，实现了“供给”与“需求”的精准对接。 活动第二阶段，全体人员乘车前往渡江胜利纪念馆。本次参观以“传承渡江红色精神，筑牢为民实干政绩观”为主题，旨在通过重温革命历史，引导党员和团员青年坚定理想信念，强化使命担当。在纪念馆内，师生们认真参观了“天翻地覆慨而慷——渡江战役胜利暨南京解放展”。一幅幅珍贵的历史图片、一件件真实的革命文物、一段段感人的影像资料，生动再现了百万雄师突破长江天险、解放南京的壮阔历程。大家纷纷表示，要弘扬“渡江精神”，摒弃浮躁功利心态，脚踏实地、履职尽责、学好本领、报效祖国。 本次共建活动内容丰富、形式新颖，既有面对面的坦诚交流，又有沉浸式的精神洗礼，切实将党建工作成效体现在服务师生、赋能发展上。双方将以此次活动为契机，持续深化共建机制，在资源共享中凝聚合力，在优势互补中激发活力，在共同提高中增强动力，推动党建带团建走深走实、形成长效、结出硕果，共同为培养担当民族复兴大任的时代新人贡献力量。

为深化党建引领，牢固树立以师生为中心的发展思想，践行正确政绩观，促进管理与育人深度融合，6月11日，河海大学资产处党支部与信息科学与工程学院通信6班团支部联合顺利开展共建党日活动。本次活动通过座谈交流与实践研学相结合的方式，搭建师生沟通桥梁，聚焦师生所需所盼，扎实办好实事、破解现实难题。 活动第一阶段在夏金仙学生服务中心会议室举行。资产处处长康红强、副处长李志强及部门党员代表，信息科学与工程学院党委书记张伟建、2025级辅导员曹滟梅及通信6班团支部学生共同参加。活动由张伟建主持。 康红强指出，本次共建活动是落实学校党委“党建带团建”要求、推动机关部处中层干部与学生班级结对联系机制的重要实践。他介绍了资产处工作的核心目标是通过制度优化与服务升级，支撑学生健康成长。他强调，通信工程专业的学科优势可为资产管理提供青年视角与技术建议，双方要坚持“资源共享、优势互补、共同提升”的原则，畅通学生意见反馈渠道，搭建实践育人平台，真正实现管理实践与专业学习的双向赋能。 张伟建强调，支部共建不仅是形式上的联谊，更是育人资源的深度链接与一堂行走的思政课。资产处管理着学校的“家底”，通信学子未来要搭建信息的“桥梁”，联结二者的是共同的服务初心和红色基因。 交流发言环节，资产处青年教师代表焦雅，通信6班学生代表刘子淇、郭高瑾分别从优化服务工作、学习生活和学习成长等方面进行发言。与会师生围绕学习空间管理、宿舍生活设施和场馆管理细节进行深入交流，实现了“供给”与“需求”的精准对接。 活动第二阶段，全体人员乘车前往渡江胜利纪念馆。本次参观以“传承渡江红色精神，筑牢为民实干政绩观”为主题，旨在通过重温革命历史，引导党员和团员青年坚定理想信念，强化使命担当。在纪念馆内，师生们认真参观了“天翻地覆慨而慷——渡江战役胜利暨南京解放展”。一幅幅珍贵的历史图片、一件件真实的革命文物、一段段感人的影像资料，生动再现了百万雄师突破长江天险、解放南京的壮阔历程。大家纷纷表示，要弘扬“渡江精神”，摒弃浮躁功利心态，脚踏实地、履职尽责、学好本领、报效祖国。 本次共建活动内容丰富、形式新颖，既有面对面的坦诚交流，又有沉浸式的精神洗礼，切实将党建工作成效体现在服务师生、赋能发展上。双方将以此次活动为契机，持续深化共建机制，在资源共享中凝聚合力，在优势互补中激发活力，在共同提高中增强动力，推动党建带团建走深走实、形成长效、结出硕果，共同为培养担当民族复兴大任的时代新人贡献力量。
