# 关于发布《河海大学2025年社会招生专题培训计划项目》的通知

关于发布《河海大学 2025 年社会招生专题培训计划项目》的通知

各单位：

河海大学是一所具有百余年办学历史，以水利为特色，工科为主，多学科协调发展的教育部直属全国重点大学，是中国水利高等教育的发源地，引领我国水利高等教育和科技发展，承载着治水兴邦的历史使命。学校为水利部水利干部培训机构、教育部“全国高等学校继续教育示范基地”、人社部“国家级专业技术人员继续教育基地”、人社部“数字技术工程师培训机构”、江苏高校银龄学习中心、江苏省本科院校教师培训基地、江苏省高等职业教育教师培训基地、江苏省中等职业教育教师培训基地。

长期以来，学校紧紧围绕国家战略、行业及区域发展需要，积极承担国家级、省部级高级别、多层次专业技术人才和管理人才的培训工作，为国家和社会培养了大批德才兼备的技术与管理人才，赢得了良好的社会信誉。根据社会需要，学校发布《河海大学 2025 年社会招生专题培训计划项目》，如有相关需求可咨询终身教育学院，联系电话： 025-83786362 ，联系网站： https://jjy.hhu.edu.cn/ 。
