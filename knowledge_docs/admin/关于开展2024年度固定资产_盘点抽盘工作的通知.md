# 关于开展2024年度固定资产 盘点抽盘工作的通知

> 来源: https://zcc.hhu.edu.cn/2025/0325/c15113a300040/page.htm

关于开展2024年度固定资产 盘点抽盘工作的通知 来源：资产处 发布时间：2025-03-25 各单位： 根据《财政部关于加强行政事业单位固定资产管理的通知》（财资〔2020〕97号）、《河海大学国有资产管理办法（修订）》（河海校政〔2023〕135号）、《河海大学固定资产盘点实施细则》（河海资产〔2021〕4号）等文件要求，为规范国有资产管理，进一步加强和规范固定资产盘点工作，切实保证固定资产账账相符、账实相符、账卡相符，资产管理处将根据2024年度固定资产盘点结果开展抽盘工作，抽查面不低于百分之二十。按二十万以上资产100%与便携设备随机抽取的原则，相关管理部门已形成并确定了各单位抽盘资产清单，抽盘任务将以系统指令的方式从资产管理系统上发出，请各单位及领用人注意查收。 抽盘工作将采用线上与线下相结合的方式开展，具体线上操作步骤与盘点工作基本相似，只是在原有核实资产信息操作步骤上增加了“上传资产照片”必填项（上传每台/件抽盘资产的“资产照片”和“资产标签照片”等两张照片，且标签照片能清晰显示已按要求贴于资产上，上传相关信息需真实可靠）。请各单位资产管理员及时通知资产领用人完成线上上传资产信息的相关工作，上传资产信息工作请于4月11日前按要求完成。 如有需要，各单位也可采用传统的线下抽盘方式，请于4月3日前及时将此情况告知资产管理处，学校将组成抽查专项组到现场抽盘，资产占有使用单位需按要求参与并配合完成现场抽盘工作。 为便于及时沟通与协调资产抽盘工作，请各单位资产管理员及时加入资产管理工作群（215850138）。如有疑问也可及时联系资产管理处。 联系人及联系方式： 资产管理处资产科 赵中强，电话 0519-85195815、13616116868，常州校区6号楼东辅楼115； 资产管理处资产科 焦雅，电话83786036，西康路校区河海馆921。 资产管理处 2025年3月25日
