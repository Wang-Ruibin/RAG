# 关于河海大学高等学历继续教育2026届优秀毕业生评选结果的公示

> 来源: https://jjy.hhu.edu.cn/2026/0622/c14570a328271/page.htm

各相关单位：

根据《河海大学高等学历继续教育优秀毕业生评选办法（试行）》（河海校科教〔 2022〕39号）、《关于评选河海大学2026届高等学历继续教育优秀毕业生的通知》(河海终身教育〔2026〕30号)文件精神，在各教学点及自考专接本对接院校申报的基础上，经专家评审，推荐王玉波等24名同学为河海大学高等学历继续教育2026届优秀毕业生。

现将名单予以公示（详见附件）。公示时间： 2026年6月22日-2026年6月26日。

如有异议，请于 6月26日下午17点前将意见反馈给终身教育学院学历教育部或自学考试办公室。
