# 哈尔滨工业大学实验室与设备管理处张杰主任一行来我处调研座谈

> 来源: https://zcc.hhu.edu.cn/2023/0329/c15112a258816/page.htm

哈尔滨工业大学实验室与设备管理处张杰主任一行来我处调研座谈 来源：资产处 发布时间：2023-03-13 3 月 13 日下午，哈尔滨工业大学实验与设备管理处、分析测试与计算中心张杰主任一行来我处调研座谈，处长何文科、副处长陈红以及相关科室负责人出席调研座谈会。 何文科代表资产处对哈尔滨工业大学代表团一行的到来表示欢迎，并主要介绍了我校资产与实验室管理处的机构设置、工作职责及分工。陈红副处长就大型仪器设备共享、实验室安全管理、实验技术队伍激励等方面进行了详细的介绍。双方围绕大型仪器开放共享、实验室安全管理、校公共服务平台建设、实验技术队伍建设等方面的做法与经验进行了深入探讨。 双方表示要加强交流与学习，共同分享先进的做法和理念。
