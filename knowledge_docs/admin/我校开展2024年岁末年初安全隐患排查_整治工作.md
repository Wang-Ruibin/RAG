# 我校开展2024年岁末年初安全隐患排查 整治工作

我校开展2024年岁末年初安全隐患排查 整治工作 来源：资产处 发布时间：2024-12-31 为了进一步加强校园安全管理，确保广大师生的生命财产安全，校安委会于12月30日至31日分别在西康路校区和江宁校区开展岁末年初安全隐患排查整治工作。此次安全检查由校领导带队，资产处、保卫处、后勤处、实验平台处、基建处、学生处等安委会成员单位共同参与，全面排查校园安全隐患，营造安全稳定的校园环境。 12月30日下午，陆国宾副书记带领检查组在西康路校区开展安全检查。检查组先后来到水利馆、图书馆、食堂、学生宿舍西区工地等地进行查看，对检查中发现的安全隐患及时下发了整改通知。陆校长强调，校园安全是学校安全工作的重中之重，关乎师生的生命财产安全，容不得丝毫马虎和懈怠。 12月31日下午，沈扬副校长带队在江宁校区行安全检查。检查组先后对乐学楼、图书馆、学生7舍、8舍、食堂以及文体中心等安全情况进行检查。沈校长强调，安全无小事，校园安全关系到每一位师生的切身利益，要求相关单位对检查中发现的问题立行立改，建立健全校园安全长效机制，切实提高校园安全管理水平。 此次岁末年初校园安全隐患排查，进一步强化了学校的安全管理工作，及时发现并消除了潜在的安全隐患，为全校师生营造了一个安全、和谐的学习与工作环境。学校将一如既往把校园安全工作放在首位，建立健全安全管理长效机制，定期开展安全检查与隐患排查治理，为学校的高质量发展保驾护航。
