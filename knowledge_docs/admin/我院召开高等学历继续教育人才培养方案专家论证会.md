# 我院召开高等学历继续教育人才培养方案专家论证会

> 来源: https://jjy.hhu.edu.cn/2023/1026/c14556a266659/page.htm

10月24日，远程与继续教育学院召开高等学历继续教育人才培养方案专家论证会。论证会邀请了来自南京水利科学研究院、安徽水利水电职业技术学院、江苏省秦淮河管理处、华能江苏分公司相关校外专家，以及学校教务处、水文院、土木院、能电院相关专家。学院院长朱亚、副院长于伟出席论证会，正在开展的十五个专业方案修订负责人列席此次会议。学院书记叶鸿蔚主持论证会。

论证会上，朱亚院长代表学院向参加专家论证会的专家们表示热烈的欢迎和衷心的感谢，希望与会专家不吝指导，多提宝贵意见和建议。叶鸿蔚书记希望专家们结合成人学习特点、产业行业发展前沿、行业企业人才培养需求，以及学科专业群建设等方面，对人才培养方案进行论证指导。

本次论证会分成水利类、土木与交通类、能源与电气类、管理类四组分别进行。各组专家对专业培养方案整体框架给予了充分肯定，并在人才培养目标、培养要求、课程体系设置等方面提出了意见和建议。水文院陈元芳教授提出人才培养目标定位应适应产业发展要求和专业发展趋势，符合学校办学定位；同时毕业要求描述清晰，体现专业能力的培养。安徽水电学院继教院胡慨院长认为，培养目标与要求需简炼、逻辑清晰，课程设置应与培养目标相统一；行业企业专家认为，专业人培养方案的制定也要关注行业发展情况，应与行业最新热点相结合，培养学生实践能力与创新视野等。同时专业修订负责人针对人才培养方案修订过程中遇到的问题和专家们进行了深入的沟通和探讨。

为深入贯彻落实教育部高等学校学历继续教育改革精神，推动学校学历继续教育规范性、内涵式、高质量发展，学院在2022年启动了高等学历继续教育人才培养方案（2024版）修订工作。一年多来，经过广泛意见征求、文献研究、行业企业走访调研、学生调研以及两次人才培养方案修订负责人研讨会，目前此项工作已初步完成。本次邀请校内外专家对各专业方案进行论证，是确保人才培养方案修订工作高水平完成的重要一步。学院和各专业修订负责人将认真总结和分析本次论证会的成果，结合专家意见，进一步优化和完善人才培养方案，推动学院教育教学和专业建设高质量发展 。
