# 教育部财务司来校开展无形资产、数据资产暨科技成果转化管理情况调研

> 来源: https://zcc.hhu.edu.cn/2025/1225/c15112a315357/page.htm

教育部财务司来校开展无形资产、数据资产暨科技成果转化管理情况调研 来源：资产处 发布时间：2025-12-22 12月22日，教育部财务司二级巡视员迟玉收一行来到河海大学，就无形资产数据资产暨科技成果转化管理情况展开了深入调研。副校长吴峰出席座谈会并主持会议。 吴峰代表学校对教育部财务司调研组的到来表示欢迎，对教育部财务司长期以来给予河海大学的关心与指导表示感谢。他介绍了学校的基本情况，特别是在无形资产、数据资产管理和科技成果转化方面的一些探索和成效。他表示，学校将以此次调研为新起点，持续提升管理水平，推动学校无形资产、数据资产和科技成果转化管理迈向新高度，为学校高质量发展筑牢根基。 迟玉收指出，在数字经济蓬勃发展的时代背景下，无形资产与数据资产已成为高校发展的重要战略资源，要快速转变思维，重视无形资产、数据资产和科技成果转化管理工作，充分发挥学校优势，为数据资产发展和提高高校科技成果转化成效贡献力量。 科技处、实验室与平台处、计财处、法律事务办、资产管理处、网信办、产业技术研究院、资产经营有限公司等部门相关负责人参加会议。

教育部财务司来校开展无形资产、数据资产暨科技成果转化管理情况调研

来源：资产处 发布时间：2025-12-22

12月22日，教育部财务司二级巡视员迟玉收一行来到河海大学，就无形资产数据资产暨科技成果转化管理情况展开了深入调研。副校长吴峰出席座谈会并主持会议。 吴峰代表学校对教育部财务司调研组的到来表示欢迎，对教育部财务司长期以来给予河海大学的关心与指导表示感谢。他介绍了学校的基本情况，特别是在无形资产、数据资产管理和科技成果转化方面的一些探索和成效。他表示，学校将以此次调研为新起点，持续提升管理水平，推动学校无形资产、数据资产和科技成果转化管理迈向新高度，为学校高质量发展筑牢根基。 迟玉收指出，在数字经济蓬勃发展的时代背景下，无形资产与数据资产已成为高校发展的重要战略资源，要快速转变思维，重视无形资产、数据资产和科技成果转化管理工作，充分发挥学校优势，为数据资产发展和提高高校科技成果转化成效贡献力量。 科技处、实验室与平台处、计财处、法律事务办、资产管理处、网信办、产业技术研究院、资产经营有限公司等部门相关负责人参加会议。

12月22日，教育部财务司二级巡视员迟玉收一行来到河海大学，就无形资产数据资产暨科技成果转化管理情况展开了深入调研。副校长吴峰出席座谈会并主持会议。

科技处、实验室与平台处、计财处、法律事务办、资产管理处、网信办、产业技术研究院、资产经营有限公司等部门相关负责人参加会议。
