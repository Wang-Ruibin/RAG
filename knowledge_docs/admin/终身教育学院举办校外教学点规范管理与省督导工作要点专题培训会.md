# 终身教育学院举办校外教学点规范管理与省督导工作要点专题培训会

> 来源: https://jjy.hhu.edu.cn/2025/0327/c14556a300130/page.htm

为有效落实合格评估整改工作任务，加大教学质量监控力度，提高教学督导业务水平，3月25日，河海大学终身教育学院召开高等学历继续教育校外教学点规范管理与省督导工作要点专题培训会。学院党总支书记、院长朱亚出席会议，教研与质量保障部主任程巧玲主持会议。

朱亚在讲话中指出，党的二十大对继续教育高质量发展作出重要部署，《教育强国建设规划纲要（2024－2035年）》将构建“泛在可及的终身教育体系”列为教育强国建设“八大体系”之一。近年来，教育主管部门以“全面规范、提高质量”为主线推进继续教育新政，对普通高校学历继续教育改革作出全面部署，旨在推动形成办学结构合理、质量标准完善、办学行为规范、监管措施有效、保障机制健全的新格局。朱亚强调，高等学历继续教育必须深化教育改革，推动由规模扩张向规范性、内涵式、高质量发展转变。自2021年以来，学院持续完善校外教学点管理机制，通过四年不懈努力，建立了较为完备的监督评估体系。各教学点要严格对照江苏省督导要求，深入开展自查自纠，切实提升教学质量和管理水平。

此次培训会特邀江苏省督导专家、江苏开放大学发展规划处处长唐荣作专题讲座。唐荣结合丰富的督导经验，紧扣“江苏省高校学历继续教育专项督导（实地督察）政策解读”主题，依据教育部政策导向、江苏省发展现状与需求，高屋建瓴地剖析了高等教育学历继续教育未来趋势。她围绕制度设计、平台监管、专项整治和专项督导四个维度，全方位介绍了当前的政策背景，并对照《江苏省高校学历继续教育专项督导实施方案》，从主体责任、基本条件、教学管理、师资队伍、经费管理五个核心方面，逐一拆解了专项督导的各项指标、观测点与常见问题，描绘出教学质量提升的清晰路径。针对实地督察的准备工作要点，她从强化顶层设计的前瞻视角出发，系统梳理了加强专业建设、规范教学管理、汇编规范有序三大举措。本次讲座有理论、有方法、有实例，内容丰富，指导性强，为参会者带来了全新的思考维度与实践方向，提供了兼具理论高度与实践价值的重要指引。

培训过程中，参会人员积极互动，唐荣针对提问逐一解答，现场氛围热烈。大家纷纷表示，此次培训内容丰富、针对性强，不仅对提升教学点规范管理水平具有重要促进作用，更为今后工作开展提供了明确方向。

近年来，学院持续强化高等学历继续教育质量监管。结合2024年江苏省督导和“双随机”检查的反馈意见，今年学院将定期开展校外教学点规范管理培训，并成立教学督导组，加大监督检查力度，严把教学质量关，努力成为规范办学的“守护者”、质量革命的“先锋队”、终身教育的“摆渡人”。

学院相关部门代表、特聘督导员以及17个校外教学点的60余名代表，通过线上线下的方式参加培训。
