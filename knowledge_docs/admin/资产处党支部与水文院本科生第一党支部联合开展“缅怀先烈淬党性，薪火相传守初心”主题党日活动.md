# 资产处党支部与水文院本科生第一党支部联合开展“缅怀先烈淬党性，薪火相传守初心”主题党日活动

> 来源: https://zcc.hhu.edu.cn/2025/0527/c15112a303449/page.htm

资产处党支部与水文院本科生第一党支部联合开展“缅怀先烈淬党性，薪火相传守初心”主题党日活动 来源：资产处 发布时间：2025-05-21 为深入学习贯彻习近平总书记重要讲话精神和中央八项规定精神，持续巩固深化学习教育成果，5月21日，学校资产处党支部与水文院本科生第一党支部联合赴雨花台烈士陵园开展“缅怀先烈淬党性，薪火相传守初心”主题党日活动。通过实地研学、交流研讨、师生对话等形式，将政治理论学习与业务实践深度融合，推动党建引领服务师生效能全面提升。 在雨花台烈士陵园，全体党员参观《信仰的力量——雨花英烈生平事迹展》，通过历史图片、文献实物、多媒体影像等，重温邓中夏、恽代英等革命先烈为民族独立和人民解放英勇斗争的壮烈事迹。党员们驻足于烈士遗物前，深刻感悟“崇高理想信念、高尚道德情操、为民牺牲的大无畏精神”的雨花英烈精神内涵。交流研讨环节中，党员们围绕“新时代大学生党员如何传承红色基因”“作风建设在高校服务工作中的实践路径”等主题展开讨论。水文院本科生第一党支部党员结合自身学习生活经历，分享了对革命先辈“忠诚、担当、奉献”精神的理解；资产处党支部党员则从业务角度，探讨如何将红色教育成果转化为服务师生的实际行动。 参观结束后，两个党支部围绕党的作风建设和学校资产管理领域师生关切问题开展面对面座谈交流。水文院本科生第一党支部党员代表结合实际体验提出了相关问题，资产处相关科室党员逐一回应，并详细介绍改进措施。学校资产处党支部书记康红强表示党的作风建设是党的生命线，雨花英烈用生命诠释了“为人民服务”的初心，资产处党支部将以师生需求为导向，将作风建设与业务工作深度融合，确保资源分配公开透明、高效有序。 此次联合主题党日活动不仅是一次生动的党性教育课，更是一次“党建+业务”的实践探索。两个党支部将继续深化合作，建立“问题收集-整改反馈-效果评估”的闭环机制，推动学校资产管理服务提质增效，为师生办实事、解难题。 （水文院 资产处 供稿）
