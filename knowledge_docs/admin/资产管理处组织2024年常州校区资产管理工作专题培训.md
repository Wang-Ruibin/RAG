# 资产管理处组织2024年常州校区资产管理工作专题培训

> 来源: https://zcc.hhu.edu.cn/2024/1017/c15112a290046/page.htm

资产管理处组织2024年常州校区资产管理工作专题培训 来源：资产处 发布时间：2024-10-15 10月15日上午，资产管理处在金坛校区33号楼101报告厅室组织我校常州校区2024年资产管理工作专题培训。资产管理处领导班子和部分相关科室成员、金坛校区17个单位资产管理人员共30余人参加了此次培训会。 资产管理处处长康红强主持会议并做了重要发言，他指出高校国有资产是高校开展科研教学管理工作的重要物质基础和保障，加强高校资产管理，有利于优化资产规划、配置、使用、处置等各个环节的管理措施，能够确保国有资产的安全、完整和高效利用，优化资产管理主要做好三方面的工作：一是要靠理念转变、认识提升；二是要靠责任落实、规范管理；三是要靠改进手段、强化培训。 宣讲培训活动围绕以上“三个目标”，分为“五个模块”展开：程晓芳处长从宏观层面介绍国有资产基础理论知识、高校资产管理制度与体系；李均熙处长结合当前各学院高度关注的公用房定额配置管理办法的执行要点做政策解读；房产科庞剑科长介绍2024年全成本核算工作的具体要求、公用房管理系统基本功能、公用房、周转房申请的流程；李志强处长、设备科成健科长介绍了资产配置、建账、变动、报废业务的政策文件、工作流程和注意事项；资产科赵中强科长汇报了固定资产验收流程和注意事项，以及近期正在推进的资产盘点工作流程和盘点情况。 本次培训内容丰富、实操性强，既有资产管理理论知识的普及，又有管理政策文件和制度的解读；既有具体管理业务内容、核心流程的讲解，又有主要信息系统功能的演示说明，得到了参加人员的一致好评。 下周资产管理处将继续举办2024年西康路校区资产管理工作专题培训，不断提高资产管理人员业务水平。（供稿：焦雅）
