# 资产管理处组织2024年西康路校区资产管理工作专题培训

> 来源: https://zcc.hhu.edu.cn/2024/1028/c15112a290559/page.htm

资产管理处组织2024年西康路校区资产管理工作专题培训 来源：资产处 发布时间：2024-10-23 10月23日上午，资产管理处在西康路校区闻天馆113报告厅室组织我校2024年资产管理工作专题培训。资产管理处领导班子和部分相关科室成员、南京各校区46个单位资产管理人员共80余人参加了此次培训会。 资产管理处处长康红强主持会议并做了重要发言，他指出科学合理的高校国有资产管理有利于保障教学科研活动顺利开展，提高资源利用效率，促进高校可持续发展。优化资产管理主要做好三方面的工作：一是要提升资产管理意识；二是要靠明晰管理责任落实；三是要开展资产管理培训。 宣讲培训活动围绕以上“三个目标”，分为“五个模块”展开：程晓芳处长从宏观层面介绍国有资产基础理论知识、高校资产管理制度与体系；李均熙处长结合当前各学院高度关注的公用房定额配置管理办法的执行要点做政策解读；房产科庞剑科长介绍2024年全成本核算工作的具体要求、公用房管理系统基本功能、公用房、周转房申请的流程；设备科成健科长介绍了资产配置、建账、变动、报废业务的政策文件、工作流程和注意事项；资产科赵中强科长汇报了固定资产验收流程和注意事项，以及近期正在推进的资产盘点工作流程和盘点情况。 本次培训内容丰富、安排紧凑，有效结合了理论和实践。既有资产管理理论知识的普及，又有管理政策文件和制度的解读；既有具体管理业务内容、核心流程的讲解，又有主要信息系统功能的演示说明，得到了参加人员的一致好评。至此，2024年各校区资产管理专题培训均圆满结束。
