# 金坛区应急管理局与河海大学召开校地合作交流座谈会

> 来源: https://zcc.hhu.edu.cn/2023/0826/c15112a264493/page.htm

金坛区应急管理局与河海大学召开校地合作交流座谈会 来源：资产处 发布时间：2023-08-24 8月24日下午，金坛区应急管理局与河海大学召开校地合作交流座谈会，双方围绕加强项目合作和人才培养等进行深入交流。河海大学资产与实验室管理处处长何文科、副处长李均熙，河海大学公共管理学院党委书记蒲晓东、院长朱新华、党委副书记朱志梅、副院长沈洪成等，区应急管理局班子全体成员参加座谈会。 座谈会上，区应急管理局党委书记、局长庄金中对河海大学一行的到来表示欢迎，并简要介绍了区应急管理局基本情况和近年来重点工作开展情况。他指出，随着河海大学常州新校区的启用，希望能够借助河海大学学科优势和人才优势，双方在安全生产、应急管理、自然灾害防治、人才培养等方面加强交流，拓宽合作领域，推动务实合作。同时，作为地方安全监管部门，区应急管理局将积极履职，全力做好河海大学新校区启用及今后正常运行期间的安全服务保障工作。 何文科首先对金坛各级各部门为保障河海大学常州新校区投入使用所提供的支持和帮助表示感谢，介绍了新校区前期建设阶段的基本情况和校园安全管理、实验室管理、危化品管理等安全工作情况，希望下一步能与金坛应急、环保、消防等部门加强交流合作，推动安全发展。蒲晓东介绍了公共管理学院的专业学科设置、办学特色等，表示河海大学相关专业设置与金坛实际和产业发展高度契合，将进一步发挥人才培养、学科建设、科学研究等方面优势，精准对接金坛发展需求，共同探索推进更好服务地方发展的合作项目，助力地方高质量发展。 双方还围绕盐穴资源综合利用安全规划、教学实践基地建设等具体工作进行了交流并达成初步合作意向。

8月24日下午，金坛区应急管理局与河海大学召开校地合作交流座谈会，双方围绕加强项目合作和人才培养等进行深入交流。河海大学资产与实验室管理处处长何文科、副处长李均熙，河海大学公共管理学院党委书记蒲晓东、院长朱新华、党委副书记朱志梅、副院长沈洪成等，区应急管理局班子全体成员参加座谈会。 座谈会上，区应急管理局党委书记、局长庄金中对河海大学一行的到来表示欢迎，并简要介绍了区应急管理局基本情况和近年来重点工作开展情况。他指出，随着河海大学常州新校区的启用，希望能够借助河海大学学科优势和人才优势，双方在安全生产、应急管理、自然灾害防治、人才培养等方面加强交流，拓宽合作领域，推动务实合作。同时，作为地方安全监管部门，区应急管理局将积极履职，全力做好河海大学新校区启用及今后正常运行期间的安全服务保障工作。 何文科首先对金坛各级各部门为保障河海大学常州新校区投入使用所提供的支持和帮助表示感谢，介绍了新校区前期建设阶段的基本情况和校园安全管理、实验室管理、危化品管理等安全工作情况，希望下一步能与金坛应急、环保、消防等部门加强交流合作，推动安全发展。蒲晓东介绍了公共管理学院的专业学科设置、办学特色等，表示河海大学相关专业设置与金坛实际和产业发展高度契合，将进一步发挥人才培养、学科建设、科学研究等方面优势，精准对接金坛发展需求，共同探索推进更好服务地方发展的合作项目，助力地方高质量发展。 双方还围绕盐穴资源综合利用安全规划、教学实践基地建设等具体工作进行了交流并达成初步合作意向。
