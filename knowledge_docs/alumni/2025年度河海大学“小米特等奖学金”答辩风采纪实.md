# 2025年度河海大学“小米特等奖学金”答辩风采纪实

> 来源: https://hhuedf.hhu.edu.cn/2025/1211/c26739a314762/page.htm

11月20日，河海大学2025年度“小米特等奖学金”答辩评审会在西康路校区夏金仙学生服务中心101多功能厅举行。小米集团人力资源部高校关系经理戎慧，学校相关职能部门领导、老师和学院专任教师共同担任专家评委。本次评审会还特别组织了35人的大众评审团，从学生视角对候选人的答辩情况进行现场打分，共同见证优秀学子的卓越表现。

戎慧在开场致辞中回顾了“小米奖助学金”项目的设立背景与发展历程。她表示，小米公益基金会成立六年来，始终致力于支持青年科技创新、助力人才培养，并在应急救灾等领域积极履行社会责任。她希望以此次奖学金评选为契机，进一步激发广大学子树立高远志向、扎实学识根基、勇于探索突破，未来为国家发展和社会进步贡献智慧与力量。

本次评选经个人申报、学院遴选、学校资格审核及初步评审，最终确定8名研究生和12名本科生进入特等奖学金答辩环节，围绕学生的学业成绩、科研创新、社会实践、公益服务及综合素质等多个维度展开全面评估。经评审委员会认真审议与综合评定，共有4名研究生与6名本科生脱颖而出，荣获2025年度河海大学“小米特等奖学金”。现在，就让我们一同走近这些候选人的精彩瞬间，感受他们的奋斗故事与青春风采！
