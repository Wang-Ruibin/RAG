# 50万元捐赠寄深情!刘俊校友捐资助力母校教育事业发展

> 来源: https://alumni.hhu.edu.cn/2025/0523/c26690a322645/page.htm

校友简介

刘俊，江苏泰州人，2004年本科毕业于河海大学土木工程专业。曾先后担任泰州医药城工程科科长、泰州医药高新区市政处处长，2017年创办泰州新润辰建设工程有限公司，现任公司董事长、总经理。

5月22日，河海大学泰州校友会常务副会长、土木工程专业2000级校友刘俊个人捐资50万元用于支持河海大学教育事业发展，捐赠签约仪式在河海大学校友之家举行。国内合作发展处处长、河海大学教育发展基金会副理事长兼秘书长赵冰，副处长李维出席捐赠签约仪式。

赵冰对刘俊校友心系母校、慷慨反哺教育的善举表示诚挚感谢。她强调，校友是学校发展最宝贵的财富，刘俊校友此次捐赠不仅饱含着对母校的深厚情谊，更是学校育人成果的有力印证，充分展现了河海人“饮水思源”的优良传统。她承诺，学校将严格规范捐赠资金的使用，确保每一份捐赠都能精准助力学校教育事业发展，激励更多河海学子茁壮成长、成为栋梁之材。

刘俊校友满怀深情地回顾了他在学校的求学时光。他感慨道，在河海求学的经历，不仅为他奠定了坚实的专业基础，更塑造了他“忠诚、干净、担当”的职业品格。即便身份不断转变，他始终坚守严谨务实的工作作风。谈及此次捐赠，他动情地表示：“能够为母校的发展贡献一份微薄之力，是我莫大的荣幸”。他衷心希望这份心意能为河海学子创造更优越的学习科研环境，同时呼吁更多校友携手并肩，共同书写母校发展的崭新篇章，让河海精神代代相传、熠熠生辉。

仪式上，赵冰处长代表学校与刘俊校友签订捐赠协议，并向其颁发捐赠证书和捐赠鸣谢牌。
