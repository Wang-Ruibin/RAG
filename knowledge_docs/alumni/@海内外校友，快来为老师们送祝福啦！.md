# @海内外校友，快来为老师们送祝福啦！

> 来源: https://alumni.hhu.edu.cn/2024/0902/c26691a287817/page.htm

数载求学路，浓浓河海情。

是我们一生最宝贵财富。

师恩难忘似海深，亲爱的校友们，在这个特别的日子里，让我们将心中的感激与祝愿拍成一段视频送给老师们，为老师们献上最诚挚的祝福！

海内外校友

即日起至9月5日

感念师恩

表达对老师们的感谢与祝福，可以讲述与老师的温暖瞬间，也可以抒发对老师的真挚情感。

1.视频时长控制在2分钟以内，单人录制或多人录制均可；

2.手机拍摄要求为横屏拍摄模式，摄像机拍摄采用16:9的比例， 人物画面、声音要足够清晰，尽量无杂音干扰；

3.视频无需添加背景音乐及字幕。

各校友分会广泛宣传，积极动员校友参与视频征集活动，并于9月5日下午6点前将相关视频资料发送至 hhuaa@hhu.edu.cn 电子邮箱。校友总会将对所收集的视频进行汇集整理，并通过网站、微信公众号等平台进行展示和播放。
