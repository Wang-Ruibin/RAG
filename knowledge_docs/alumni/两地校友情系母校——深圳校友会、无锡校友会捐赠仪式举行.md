# 两地校友情系母校——深圳校友会、无锡校友会捐赠仪式举行

> 来源: https://alumni.hhu.edu.cn/2025/1112/c26690a322639/page.htm

近日，深圳校友会、无锡校友会捐赠仪式在西康路校区举行。深圳校友会会长龚南、无锡校友会会长杨良生代表深圳校友会、无锡校友会分别向学校捐赠110万元支持学校事业发展。河海大学校长郑金海出席捐赠仪式。

郑金海对深圳校友会、无锡校友会慷慨捐资的善举表示感谢。他指出，两地校友会的捐赠不仅为学校事业发展提供了有力支持，更传递了校友们对母校的深切关怀。学校将妥善用好捐赠资金，切实提升办学质量，不辜负校友们的信任与厚爱，并期待与广大校友共同携手，谱写学校高质量发展新篇章。

仪式上，郑金海向深圳校友会、无锡校友会颁发捐赠鸣谢牌。

学校相关职能部门负责人，深圳校友会、无锡校友会理事会成员代表参加捐赠仪式。
