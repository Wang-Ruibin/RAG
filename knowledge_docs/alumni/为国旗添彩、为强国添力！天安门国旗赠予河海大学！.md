# 为国旗添彩、为强国添力！天安门国旗赠予河海大学！

> 来源: https://alumni.hhu.edu.cn/2025/0728/c26684a306050/page.htm

7月19日，在学校建校110周年倒计时100天之际，学校举行迎接天安门国旗仪式 。校长郑金海出席活动，校党委副书记、纪委书记孟新宣读国旗赠予证书，校党委副书记、副校长陆国宾主持仪式。

郑金海在讲话时表示， 在天安门广场升起的国旗具有唯一性，十分珍贵，这面国旗必将成为全体河海人接受爱国主义教育的旗帜 。他希望，全体师生校友以此次迎旗仪式为契机，赓续红色血脉，坚定理想信念，厚植家国情怀，做爱国主义精神的传承者。学校要始终坚持为党育人、为国育才，培养堪当民族复兴重任的时代新人，以实际行动向110周年校庆献礼。要依托水利特色，聚力深化改革，勇担时代重任，以服务国家科技自立自强为己任，在传承中扛牢使命，在创新中突破攻坚，在奋斗中展现作为，用守护江河安澜、破解治水难题的实干担当为国旗添彩、为强国添力。

孟新宣读国旗赠予证书

陆国宾主持仪式

河海大学深圳校友会会长龚南作为校友代表发言时表示，这面旗帜跨越千里，激励着河海人薪火相传、矢志报国。河海大学深圳校友会将继续发挥服务校友、凝聚力量、反哺母校、贡献社会的作用。

国旗护卫队代表高晔发言时表达了自己的激动心情，表示要守护好这份红色荣耀，切实将“爱国情 强国志”转化为“攻克水利难题 守护江河安澜”的实际行动，让青春在为祖国奉献中绽放光彩。

仪式上，国旗护卫队的6名同学迈着整齐矫健的步伐，将获赠国旗全幅展示。这面编号为2015-0143号的国旗，长5米，高3.3米， 曾于2015年10月27日学校建校100周年校庆日当天在天安门广场升起，对学校具有特殊的纪念意义、教育意义。

校领导、校党委常委、校长助理、校内各单位负责人、师生代表、校友代表共同礼迎国旗，传承红色基因，汇聚奋进力量。

6月27日，北京市人民政府天安门地区管理委员会向河海大学赠送天安门国旗。校长郑金海代表学校前往北京接受赠予，与天安门地区管委会党组成员、副主任姚辉（正厅局级）签署了天安门专用国旗赠予协议，并就学校基本情况、国旗展陈及使用计划等内容进行了介绍。
