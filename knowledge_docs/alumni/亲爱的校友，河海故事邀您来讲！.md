# 亲爱的校友，河海故事邀您来讲！

> 来源: https://alumni.hhu.edu.cn/2024/1209/c26691a296972/page.htm

百十河海，薪火相传，岁月不居，时节如流。母校的一草一木，一砖一瓦承载着河海人的青春记忆和成长足迹，校友们的成长发展、优势资源凸显着母校事业高质量发展的成效。

为广泛收集各方校友的珍贵记忆，宣传校友在各行各业的奋斗事迹，充分展示河海人的精神风貌，河海大学校友会微信公众号、河海大学校友会期刊《校友通讯》现邀请您一起讲述属于河海人的故事，让情感的溪流汇聚成海，让心灵的火花照亮前路，共同为母校百十校庆献礼！
