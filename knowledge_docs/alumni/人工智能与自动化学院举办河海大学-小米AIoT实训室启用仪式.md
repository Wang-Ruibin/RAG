# 人工智能与自动化学院举办河海大学-小米AIoT实训室启用仪式

> 来源: https://hhuedf.hhu.edu.cn/2025/1219/c26739a315217/page.htm

10月29日，河海大学-小米AIoT实训室启用仪式在河海大学常州校区举行。常州市京米教育科技有限公司，江苏峻源人力资源有限公司，金坛区委组织部、人社局、团区委，河海大学常州校区管委会、国内合作发展处、教务处等校内外领导、企业代表共同出席。仪式由人工智能与自动化学院党委书记董涌波主持。

河海大学常州校区管委会主任吴永强致欢迎辞，对与会嘉宾表示热烈欢迎，并强调了校企合作在推动人才培养与产教融合方面的重要意义。智能院院长刘小峰详细介绍了学院与小米集团的合作背景与成果。指出，此次共建AIoT实训室是深化产教融合、对接产业需求的重要举措，将为学生在人工智能前沿领域的实训学习提供优质的实践平台。

教务处副处长陈丹从学校层面介绍了河海大学在产教融合方面的探索与成效，强调了实训室在提升学生工程实践能力与创新能力方面的重要作用。京米科技负责人李梦林就“河海大学-小米AIoT实训室”项目进行了详细介绍，阐述了实训室在课程建设、技术研发等方面的规划与愿景。

京米科技李梦林与学校国内合作发展处处长、教育基金会办公室主任赵冰代表双方签订了捐赠协议。江苏峻源人力资源有限公司总经理沈涛与智能院院长刘小峰共同为实训室揭牌，标志着河海大学-小米AIoT实训室正式启用。

仪式结束后，与会嘉宾一同走访参观了新建成的AIoT实训室，并在实训室前合影留念。

河海大学-小米AIoT实训室是河海大学与小米集团校企协同、产教融合人才培养的重要载体，由小米集团提供品牌授权及课程授权、河海大学与常州市京米教育科技有限公司共同建设。实训室结合小米集团在软件、硬件的全球领先优势和力量，融合了数智化实践教学平台、小米AIoT实验箱+人工智能实训套件、小米机器狗等软硬平台和课程资源，提供全流程一体化实训服务，将为小米及产业生态提供人才供给，服务地方和行业产业发展。该实训室也是小米集团在常州授权建设的首个实训室，在校企协同人才方面具有突出示范作用。
