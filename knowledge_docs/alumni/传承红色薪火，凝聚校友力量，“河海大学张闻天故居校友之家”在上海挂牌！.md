# 传承红色薪火，凝聚校友力量，“河海大学张闻天故居校友之家”在上海挂牌！

> 来源: https://alumni.hhu.edu.cn/2025/0804/c26684a306153/page.htm

8月2日，在上海市浦东新区祝桥镇张闻天故居，“河海大学张闻天故居校友之家”正式挂牌！校长郑金海，张闻天外孙女张秀君，张闻天故居执行馆长李嵘，上海校友会会长石小强、秘书长李松等上海校友代表，2025级上海新生代表以及学校相关职能部门负责人参加活动。

郑金海指出，“河海大学张闻天故居校友之家”的挂牌，是学校百余年办学历史上传承弘扬爱国报国的红色基因、治水兴邦的历史担当和严谨务实的精神特质的又一有力实践。他强调，要让“张闻天故居校友之家”成为凝聚校友力量的温馨港湾，引导广大校友感悟张闻天同志勇于坚持真理、淡泊名利、奋斗不止的坚强信念，激励广大校友胸怀国之大者，服务国家、追求卓越，为强国建设、民族复兴伟业贡献属于河海人的智慧和力量。

上海校友会会长石小强表示，河海大学在沪校友人数已超过一万人，从大国工程总工到亚投行金融专家，遍布水利、金融、管理等多元领域，未来将通过“校友之家”持续助力校友成长。

故居活动现场同步举行了“河海大学爱国主义教育基地”挂牌仪式、2025级上海新生座谈会和校旗传递活动。郑金海、张秀君、石小强、李嵘共同为“河海大学爱国主义教育基地”“河海大学张闻天故居校友之家”揭牌。

上海校友会在张闻天故居捐赠建设“闻天图书角”，将陈列《张闻天选集》《张闻天论》等书籍，以及《南水北调工程中的河海人》《港珠澳大桥工程建设中的河海校友》等河海大学校友系列丛书，为校友们打造“行走的红色课堂”。

在沪，郑金海还率队走访了上海校友会，并与校友们座谈交流。郑金海向校友们详细介绍了学校在学科建设、人才培养、科学研究以及师资队伍等方面取得的成就以及学校庆祝建校110周年相关工作情况。与会校友围绕加强校友联络、服务校友发展等方面畅所欲言，并期望未来能与母校在科研攻关、人才培养、协同创新等方面强化交流合作，为母校高质量发展贡献力量。
