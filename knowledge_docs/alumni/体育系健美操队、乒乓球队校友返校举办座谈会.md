# 体育系健美操队、乒乓球队校友返校举办座谈会

> 来源: https://alumni.hhu.edu.cn/2025/1111/c26687a309915/page.htm

近日，体育系健美操队、乒乓球队校友座谈会分别在江宁校区举行。体育系党委书记刘江、主任唐鹏以及师生代表、校友等齐聚一堂，共忆往昔、共话未来，为母校百年华诞献上最诚挚的祝福。

在健美操队校友座谈会上，唐鹏深情回顾健美操队的发展历程，感谢校友对母校的深厚情谊。刘江代表学校接受了校友们赠送的纪念礼物。活动以“百十风华 薪火共舞”为主题，在《我们的跃动年华》宣传片中拉开序幕。视频中历届队员的训练与比赛画面唤起了在场所有人的美好回忆。

校友分享环节中，林峻、胡欣、叶云飞、席瑞阳和孙晶玉洁等校友代表分别讲述个人成长经历，感念健美操队带来的深远影响。现役队长林奕航表达了对学长学姐的敬意与传承决心。

当晚，健美操队校友与现役队员联袂在西操场“河海文化夜”中献上精彩表演。新老队员默契配合，以充满活力与美感的舞姿展现了河海体育精神的代代相传，将全天活动推向高潮。

在乒乓球队校友座谈会上，唐鹏充分肯定了球队成绩与贡献，回顾了学校在场地、教练和赛事方面的持续支持，并鼓励校友继续为母校发展助力。活动以“传承·凝聚·展望”为主题，通过短片回顾、领导致辞、校友分享、恩师寄语、在校生发言和赠礼仪式等环节，温情呈现了球队的发展历程与情感传承。专题短片带领全场重温球队从简陋训练场到现代化球馆的发展历程，定格了历代队员训练、比赛与获奖的珍贵瞬间。

王猛、卢宥臻、王羽等校友代表依次分享与球队的成长记忆，感谢教练指导和领导关怀，坦言球队经历塑造了自己的品格。未能到场的郭平老师、王建民老师也通过视频寄语，勉励新老队员传承乒乓精神。顾巍翀老师与校友互动追忆往事，展现代代相传的师生情。

在校生代表顾哲文表示将以学长学姐为榜样，努力为校争光。活动最后，郝思稼校友向体育系赠送象征传承的花瓶，将现场温情推向高潮。

校友座谈活动贺展演凝聚了校友们对母校的深厚情感，也充分彰显了河海大学体育文化的深厚底蕴与蓬勃生机，展现了河海体育精神的代代相传。
