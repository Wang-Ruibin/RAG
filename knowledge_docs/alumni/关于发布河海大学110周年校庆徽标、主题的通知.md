# 关于发布河海大学110周年校庆徽标、主题的通知

> 来源: https://alumni.hhu.edu.cn/2024/1101/c26691a290873/page.htm

河海大学建校 110 周年校庆徽标 :

校庆徽标设计说明： 徽标融合了数字 “ 110” 、“水”的字型、龙形图案、鲲鹏图案、浪花等诸多元素， 充分彰显了河海大学办学特色，体现了 110 年办学元素，整个徽标造型主题鲜明、简洁明快、寓意深刻、喜气庄重。

河海大学建校 110 周年校庆主题：

大哉河海百十载，兴水育才创一流。

河海大学建校 110 周年校庆徽标，是河海大学 110 周年校 庆期间使用的统一标识，版权归河海大学所有。学校各单位、师生、校友在举办庆祝河海大学建校 110 周年活动中，应正确使用校庆徽标和主题。
