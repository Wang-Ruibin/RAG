# 关于面向我校校友征集个人著作、科研成果、手稿等文献资料的通知

> 来源: https://alumni.hhu.edu.cn/2018/0608/c26691a183755/page.htm

亲爱的校友们： 我校建校以来，名师荟萃、学者云集、硕果璀璨，同时培养了数十万名河海学子，他们为社会的科技进步、文化发展做出了巨大贡献。为了长久保存和展示河海师生及校友的风采与学术贡献，弘扬河海的光荣传统，保护好河海精神和文化传承的载体，不断激励河海学子学习先贤的治学精神、高尚品德与严谨学风，图书馆正在筹建“‘河海人’文库”。现广泛征集“‘河海人’库”作品，并将作为我校图书馆特色馆藏文献的一部分，永久珍藏。 一、征集范围 凡在我校不同历史阶段学习和工作过的校友（毕业生和结业生、访问学者、进修生、曾在我校工作过的教职工等），历年来在国内外公开出版的著作（包括著、编、译、审）、教材、工具书、论文集、艺术作品集、科研报告、教案、手稿、书画作品等各种文献资料。 二、征集和捐赠方式 1、到馆捐赠： 请捐赠者在工作日上午8:30—12:00、下午2:00—5:00，将捐赠资料直接送至：（1）南京市江宁区佛城西路8号河海大学江宁校区图书馆文献编目部（A202室）；（2）南京市鼓楼区西康路1号河海大学图书馆资源建设部（9楼）；（3）南京市鼓楼区西康路1号河海大学图书馆办公室（705室）。 2、信函邮寄： （1）地址：南京市江宁区佛城西路8号河海大学江宁校区图书馆文献编目部；联系人：高老师；联系电话：（025）58099451、17714414056；邮编：211100；邮件上请注明“捐赠图书”。 （2）地址：南京市鼓楼区西康路1号河海大学图书馆办公室；联系人：王老师；联系电话：（025）83786330、13851941846；邮编：210098；邮件上请注明“捐赠图书”。 3、拜访征集： 捐赠者可通过电话或邮件联系图书馆，图书馆将安排专人登门拜访、征集，联系电话：（025）58099451、17714414056（高老师），邮 箱： tsgbmb@hhu.edu.cn 。 4、出版信息提供： 提供能够购买到的作品的出版信息（书名、著者、出版社、出版时间、ISBN号等），以便我馆收集采购信息。联系电话：（025）58099451、17714414056（高老师），邮箱： tsgbmb@hhu.edu.cn 。 三、征集时间 自通知之日起，常年接受捐赠。 四、相关说明 1、恳请广大校友大力支持，协助做好学校“‘河海人’文库”的建设工作。 2、征集的全部资料由图书馆设置专门场地，设立“‘河海人’文库”，永久收藏、展示，并供校内外读者使用。 3、请您在作品的扉页签名或盖章，使赠书版本更具价值。 4、图书馆将向捐赠者颁发捐赠纪念证书。 5、捐赠图书编目入藏后，便可以在书目查询系统中查询到馆藏信息。 6、其他问题亦可以联系图书馆办公室，联系电话：（025）83786330。 河海大学图书馆、河海大学校友办 2018年5月28 日
