# 农业水利工程专业2002级校友毕业20周年返校活动圆满举行

> 来源: https://alumni.hhu.edu.cn/2026/0513/c26687a326905/page.htm

岁月如歌，薪火相传；廿载逐梦，故园重归。日前，农业科学与工程学院2002级农业水利工程专业30余名校友怀揣着对母校的眷恋、对恩师的感恩、对同窗的思念，重返校园，举办毕业20周年返校聚会活动，共忆青葱岁月，共话成长感悟，共绘发展蓝图。

“廿载韶华·水韵情深”感念师恩座谈会在江宁校区举行，一场跨越二十年的重逢，在温情与敬意中拉开帷幕。农业科学与工程学院党委书记刘兴平、院长徐俊增，时任2002级农水专业班导师诸一骅，以及陈菁、朱成立、俞双恩、郭相平、缴锡云、郝树荣、邵孝候、于永海、邵光成、冯宝平等十余位当年的任课老师受邀出席，与校友们围坐一堂、畅叙情谊。

座谈会上，徐俊增首先代表学院对各位校友的返校表示热烈欢迎，随后详细介绍了学院和农水学科的发展历程与现状。他说，河海大学农水专业已走过70余年风雨历程，为国家农村水利事业发展输送了近万名优秀人才。校友是学院最宝贵的财富，学院的发展离不开每一位校友的关心与支持，期待各位校友常回家看看，继续为学院发展建言献策、贡献力量。

“时光荏苒，二十载转瞬即逝，但当年在课堂上的谆谆教诲、在校园里的欢声笑语，依然清晰如昨。”座谈会上，校友们打开了记忆的闸门，深情回望求学时光。大家纷纷回忆起当年在毕业设计上的钻研、在操场上的奔跑、在宿舍里的畅谈，追忆着老师们严谨治学的态度、无微不至的关怀，言语间满是感恩与眷恋。校友们也逐一分享了自己毕业后的工作与生活经历。二十年来，他们带着母校的嘱托与“抱朴守真，求实创新”的农水精神，深耕水利、农业、能源等各个领域，不少人已成为行业骨干，在各自岗位上担当重任，用专业能力为国家经济社会发展、为农村水利事业进步贡献着河海力量，用实际行动践行着“功成不必在我，功成必定有我”的责任与担当。畅谈中，校友们感慨万千，既惊叹于母校和学院二十年来的跨越式发展，也珍惜着彼此历经岁月沉淀的深厚情谊。大家纷纷表示，无论走得再远、飞得再高，母校永远是心灵的港湾，老师永远是前行的灯塔，同窗情谊永远是最珍贵的财富，衷心祝愿母校基业长青、桃李芬芳，祝愿学院学科发展蒸蒸日上、再创辉煌。

情系母校，薪火相传。为表达对母校培育的感恩之情，座谈会上，2002级农水专业校友向学院捐赠了一台开发版的四足智能机器人。该机器人具备先进的运动控制与交互能力，配备了深度相机与激光雷达等感知技术手段，可用于学院人工智能教学、面向农业机器人的技术应用开发、科技社团活动及科普展示，助力学院深化学科交叉创新，为培养复合型农业水利人才增添助力。该智能机器人也是校友与母校之间深厚情谊的生动见证。

此次毕业20周年返校活动，不仅让校友们重温了青春记忆、收获了重聚的喜悦，更搭建了校友与母校、校友与老师之间沟通交流的桥梁，进一步凝聚了校友力量，深化了校友情谊。相信在未来，2002级农业水利工程专业的校友们将继续与母校同心同向、同频共振，以更优异的成绩回报母校，助力学院和母校在服务国家和战略、推动行业发展的道路上书写更加精彩的篇章。
