# 北京金河水务建设集团捐资120万元设立奖学金

> 来源: https://alumni.hhu.edu.cn/2025/1015/c26690a322642/page.htm

10月14日，“北京金河水务奖学金”捐赠签约仪式在河海大学西康路校区举行。北京金河水务建设集团董事长兼总经理周付春、河海大学校长郑金海出席捐赠仪式。仪式由河海大学国内合作发展处处长、校友会办公室主任、教育基金会办公室主任赵冰主持。

郑金海代表学校对北京金河水务建设集团的慷慨捐赠表示诚挚的感谢。他介绍了河海大学在学科建设、师资力量、人才培养等方面发展的近况，以及学校建校110周年系列纪念活动筹备与开展情况。他指出，北京金河水务建设集团此次捐资设立奖学金，不仅彰显了企业“治水兴邦、赋能教育”的责任担当，也为校企双方搭建了协同育人、共促行业发展的坚实桥梁。学校将以此为契机，进一步深化与金河水务在水资源治理、智慧水务等领域的产学研融合，加速科研成果转化、培育更多高素质水利人才，为保障国家水安全和经济社会可持续发展贡献更大力量。

作为河海大学校友，周付春深情回顾了在母校的求学岁月与成长经历，对母校给予的专业知识培育和精神文化滋养表达诚挚谢意。他介绍了北京金河水务建设集团的发展历程、战略布局与未来发展规划等。他表示，“北京金河水务奖学金”的设立，既承载着既承载着企业反哺教育、支持水利人才培养的责任，也凝聚着公司与学校共同推动水利事业发展的期盼。希望通过奖学金激励更多学子深耕水利专业领域、志存高远、脚踏实地，积极投身水利事业，为国家水利建设贡献青春和智慧。

仪式上，双方共同签署捐赠协议。郑金海代表学校向北京金河水务建设集团颁发了捐赠证书和鸣谢牌。

双方还围绕人才培养、科技创新、校企合作进行了交流。

北京金河水务建设集团相关人员，河海大学相关职能部门及院系负责人、学生代表参加捐赠仪式。

北京金河水务建设集团原名北京市第一水利工程处，成立于1958年，是首都水利建设的骨干力量，荣获了鲁班奖、大禹奖、长城杯等数十项奖励，连续十年获得全国优秀水利企业荣誉称号，是全国水利市场和北京水利市场AAA级信用单位，拥有水利水电壹级、市政壹级、环保壹级、装饰壹级、地基处理壹级等20余项资质。是一家以水利施工为主，涵盖投资设计、工程施工、技术服务、运营养护等水利全产业链的综合性集团，设有工程施工、投资运营、生态科技、房地产物业、管养服务、新能源六大业务板块，年综合经营额过百亿元。

金河水务集团立足传统水利工程，实施科技兴企战略，在水生态、污水处理、环保疏浚及淤泥资源化、管道非开挖、高性能防渗防水材料、智慧水务、智能建造等领域卓有成效，是中国水利企业协会水环境分会会长单位，北京市水利工程协会秘书长单位，荣获国家级工程建设奖、大禹水利科技进步奖、北京市科技进步奖等奖项。

金河水务集团积极践行国企担当，勇担社会责任，在2008年汶川地震、2012年北京“7.21”特大暴雨、2023年海河“23.7”流域性特大洪水、2025年海河“25.7”区域性特大洪水的抢险救灾中发挥了重要作用。

周付春，男，汉族，1974年8月生，河南开封人，中共党员，硕士，高级经济师，1996毕业于河海大学人力资源管理专业。现任北京金河水务建设集团有限公司董事长兼总经理，任昌平区政协委员、中国水利企业协会水环境治理分会会长、北京开封企业商会名誉会长、北京水利学会监事长、北京市水利工程协会副会长、中国水利企业协会理事等职务。
