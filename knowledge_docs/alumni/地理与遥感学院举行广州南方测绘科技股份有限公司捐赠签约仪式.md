# 地理与遥感学院举行广州南方测绘科技股份有限公司捐赠签约仪式

> 来源: https://hhuedf.hhu.edu.cn/2025/1111/c26739a309931/page.htm

11月1日，河海大学地理与遥感学院与广州南方测绘科技股份有限公司举行“聚力创新，智绘未来”捐赠签约仪式。南方测绘公司副总裁、教育事业部总经理郭宝宇、沪宁分公司总经理袁大伟、教育事业部总工寻林辉、沪宁分公司教育事业部总监任大勇、集团沪宁分公司市场经理于武斌；地理与遥感学院院长周丰、教育基金会办公室副主任李维、地理与遥感学院副院长杨英宝参会，会议由学院党委书记孔祥冬主持。

周丰肯定了南方测绘在设备装备、软件研发等领域取得的成就。期望双方在人才培养方面，强化教育教学实践，提升学生的一线动手能力；在项目合作方面，联合开展涉水项目研究、重大工程咨询，充分发挥地理学学科优势，提高技术在应用场景中的贡献度；在社会服务方面，面向行业、区域及学术共同体的需求，聚焦关键领域实现技术突破，推动双方的长期合作切实落地。

郭宝宇指出，学院在地理学领域的办学实力突出，人才培养成果显著。南方测绘自成立以来，始终秉持服务教育、助力创新的理念，积极推动校企合作，此次合作既是双方共同推动行业发展、培育专业人才的重要举措，也是南方测绘深化教育领域合作的关键一步。南方测绘将充分发挥自身在技术、设备、实践资源等方面的优势，为学院教学科研与学生培养提供切实支持

会上，双方共同签署了捐赠协议。此次捐赠涵盖高端虚拟仿真实验设备等教学资源，将为学院实践教学体系升级、人才培养质量提升注入强劲动力。双方在全国地理科学类一流本科专业建设研讨会期间举行元宇宙信息地理实训基地揭牌仪式。

此次捐赠是学院推动校企合作以及产教融合的重要成果。未来，学院将以此为契机，充分整合南方测绘的技术与资源优势，进一步优化人才培养方案、加强实践教学平台建设、提升学科科研创新能力，为地理学领域培养更多高素质创新型人才，同时也为推动校企协同发展、服务国家战略需求贡献更多力量。
