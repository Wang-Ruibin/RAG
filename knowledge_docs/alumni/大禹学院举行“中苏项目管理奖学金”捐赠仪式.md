# 大禹学院举行“中苏项目管理奖学金”捐赠仪式

> 来源: https://hhuedf.hhu.edu.cn/2025/1219/c26739a315215/page.htm

10月25日，大禹学院“中苏项目管理奖学金”捐赠仪式在河海大学江宁校区举行。中苏项目管理（江苏）有限公司董事长刘海侠校友，大禹学院党委书记李英、院长李国芳、副书记、副院长金光球出席捐赠仪式。仪式由李英书记主持。

李国芳代表学院对中苏项目管理（江苏）有限公司支持学院教育事业的善举致以衷心的感谢，并介绍了大禹学院在拔尖创新人才培养——学生创新实践等方面的情况。她表示，此次奖学金的设立将为学院人才培养工作注入新动能，学院也将与校友企业保持密切联系，依托产学研合作打通“人才培养-实践锻炼-社会服务”的闭环培养体系。刘海侠分享了公司的发展历程、战略布局与未来发展规划等。她表示，设立“中苏项目管理奖学金”，既是企业主动反哺教育、践行支持水利人才培养社会责任的具体行动，也承载着公司与大禹学院携手并进，共同为水利事业发展凝聚力量、夯实人才培养基础的美好期盼。李英与李国芳共同向刘海侠颁发捐赠证书与鸣谢牌。双方围绕人才培养、科技创新、校企合作等方面进行了交流。

未来，大禹学院将以该奖学金为纽带，聚焦水利拔尖创新人才培养核心任务，联合校友企业共建实践创新平台，持续培育兼具创新思维与行业竞争力的水利拔尖创新人才，为水利事业高质量发展提供坚实的人才保障。

企业简介

中苏项目管理（江苏）有限公司2022年由苏州立杰招标代理有限公司及江苏鸿卿项目管理有限公司合并成立，目前公司地址位于苏州工业园区九章路69号理想创新大厦 B 座 604 室，注册资金1000万元人民币；经过多年的发展壮大，中苏项目管理（江苏）有限公司已经由过去的地区性单一业务公司，发展为立足江苏，布局全国，以全过程咨询服务为主的现代化企业。公司主营工程造价咨询、招标代理、工程监理、征地拆迁咨询、财务审计、工程项目管理、经济评价、司法鉴定、项目全过程咨询等综合咨询业务。公司具备工程造价咨询、工程招标代理等相关资质，建立了完善的工作质量控制制度，通过了ISO质量、环境、职业健康安全管理体系认证。并于南京市、宿迁市、苏州工业园区、苏州姑苏区、连云港市、泰州市、淮北市、贵州省、江西省、成都市等地区成立了分公司和分支机构。公司拥有一批理论知识全面，实践经验丰富的专业技术人员，其中注册造价工程师36人，一级建造师10人，注册监理工程师5人，招标师8人；高级工程师10人，工程师、经济师等中级职称18人，各种专业配套齐全，90%以上具有大学以上学历；公司还聘请了建设领域术造诣深、工程经验丰富的教授、专家成立了专家委员会，为工程咨询中的疑难问题提供专家服务。秉承质量第一、客户至上、精益求精、永无止境的服务理念，以及诚信、求真、高效、创新的企业精神，我们将永葆初心，牢记企业责任与使命，不断提升服务态度，提高服务质量，为项目保驾护航，为客户创造价值，为社会可持续发展助力赋能，致力成为备受信赖的国际化综合型专业工程咨询服务平台。
