# 广东河海联合创始人校友捐资赋能学校教育事业发展

> 来源: https://hhuedf.hhu.edu.cn/2025/1019/c26739a308612/page.htm

10月18日，广东河海工程咨询有限公司毕业于河海大学的八位联合创始人捐赠签约仪式在河海大学西康路校区举行。广东河海工程咨询有限公司董事长、河海大学广东校友会会长、1986级农田水利工程专业校友孙栓国，河海大学校长郑金海出席仪式。仪式由河海大学国内合作发展处处长、校友会办公室主任、教育基金会办公室主任赵冰主持。

郑金海代表学校对广东河海工程咨询有限公司联合创人的慷慨捐赠表达了诚挚的感谢。他指出，河海大学始终以培养水利英才、服务国家战略需求为使命，近年来在学科建设、科研创新与人才培养方面取得显著进展。作为校友企业中的优秀代表，广东河海工程咨询有限公司不仅深耕水利领域行业、凭借卓越的专业实力与责任担当赢得认可，更以实际行动诠释了河海人的使命情怀。孙栓国校友始终关心母校发展，个人曾捐资设立“广东河海农水基金”助力家庭经济困难学子成长成才。此次八位联合创始人共同发起的捐赠，更以集体行动彰显了河海校友“情系母校、反哺育人”的深厚情怀。他表示，学校期待以此次捐赠为起点，进一步深化与公司在人才培养、科研攻关、实习实践等领域的合作，携手打造校企协同育人的标杆模式，为水利行业高质量发展培育更多优秀人才。

孙栓国深情回顾了在河海大学的求学与成长历程，感慨母校不仅给予了他丰富的知识，还让他收获了珍贵的友谊和面对挑战的勇气。公司自成立以来，始终依托水利行业优势稳步发展，而企业的每一步成长，都离不开母校深厚的教育根基与人才支撑。他表示，履行社会责任、行业责任和员工责任是企业发展的应有之义；反哺母校、回馈水利事业，是全体联合创始人共同的初心。未来，公司将持续深化与母校的联动合作，通过科学研究、科技成果转化、人才培养等方面实现双向奔赴、双向赋能，为推动水利行业高水平人才培养、助力国家水利事业发展贡献更多力量。

仪式上，双方共同签署捐赠协议，郑金海代表学校向孙栓国校友颁发了捐赠证书和鸣谢牌。水文水资源学院、水利水电学院、农业科学与工程学院、地理与遥感学院、商学院等相关学院领导，分别向范立柱、李刚、林耀臣、李建军、臧学义、魏立山、孙旭等七位捐赠校友逐一颁发了捐赠证书。

水文水资源学院党委副书记陈培玲为范立柱校友颁发捐赠证书

水利水电学院党委书记娄健为李刚校友颁发捐赠证书

农业科学与工程学院党委书记刘兴平为林耀臣校友、李建军校友颁发捐赠证书

地理与遥感学院党委书记孔祥冬为臧学义校友、魏立山校友颁发捐赠证书

商学院党委副书记杨天为孙旭校友颁发捐赠证书

“广东河海农水基金”获奖学生代表、2022级土地整治工程专业本科生任静怡向孙栓国校友及广东河海工程咨询有限公司表达了由衷的敬意与感谢，并承诺将以优秀校友的奋斗精神为榜样，更加刻苦地钻研专业知识，未来主动投身水利事业、服务国家发展，用实际行动回馈校友的殷切期望。

学校相关职能部门，相关学院负责人及学生代表参加捐赠仪式。
