# 感恩母校！俞超群校友捐赠千架无人机表演点亮河海百十校庆夜空

> 来源: https://hhuedf.hhu.edu.cn/2025/1219/c26739a315221/page.htm

近日，终身教育学院2019级水利水电工程专业校友、浙江新昌县农林水集团党委书记、董事长俞超群捐赠无人机表演，献礼河海大学百十华诞。捐赠仪式在西康路校区河海馆举行。河海大学副校长陈达出席仪式并致辞，学校宣传部、教育基金会办公室、终身教育学院、环境学院等单位负责人共同参加。

陈达代表学校向俞超群表示衷心感谢。他指出，校友是学校发展的重要力量，俞超群校友的捐赠，体现了河海人饮水思源、反馈母校的文化传承，更展现了校企协同、资源共享的丰硕成果。学校将持续搭建校友发展平台，凝聚各方力量，共同服务国家战略与行业进步。俞超群回顾了在河海大学的学习经历，表达了对母校培养的感恩之情，并表示将继续推动校企合作纵深发展，助力母校人才培养与科研创新。陈达向俞超群颁发捐赠证书及鸣谢牌。此次捐赠不仅是校友情系母校的体现，也是终身教育学院与环境学院协同推进校友工作、实现资源共享的重要成果，彰显了河海大学多层次、多维度服务国家战略与行业发展的能力。仪式后，双方围绕产学研协同、人才培养与项目合作等议题展开座谈。

夜幕中，无人机编队以精准的灯光明暗变化，渐次呈现河海大学办学历程中的重要图景：从张謇先生筚路蓝缕的创校历程，到西康路、江宁、常州三校区的地标建筑，每一帧光影都承载着河海人的共同记忆。表演最终定格于“热烈庆祝河海大学建校110周年”的绚丽字样，以现代科技致敬百十厚重历史，为校庆系列活动增添了一道璀璨风景。

现场师生、校友纷纷驻足观看，夜空中的流光溢彩与地面的掌声欢呼交织。表演通过线上平台同步直播，与海内外校友共享这一难忘时刻。

俞超群始终心系母校发展，并以实际行动支持学校育人事业。2023年，在其积极推动下，浙江钦寸水库有限公司与河海大学共同打造产学研协同发展平台，为人才培养与技术革新注入持续动能。

同年12月，河海大学继续教育校友会浙江分会于钦寸水库正式成立，俞超群当选会长，为校友事业发展与母校建设搭建起坚实桥梁。
