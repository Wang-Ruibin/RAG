# 我校校友何光华当选2025年“大国工匠年度人物”

> 来源: https://alumni.hhu.edu.cn/2026/0303/c26684a323324/page.htm

3月2日，由中华全国总工会、中央广播电视总台联合主办，甘肃省总工会承办的2025年“大国工匠年度人物”宣传活动正式揭晓。 我校2000届电气工程及其自动化专业校友、国网无锡供电公司电缆运检中心主任、党支部副书记何光华成功入选。

何光华扎根电缆施工一线20余载，始终秉持匠心，攻坚行业技术瓶颈。针对传统电缆敷设接头多、隐患大的行业痛点，她带领团队历时8年攻关，在全球首创“高落差高压电缆线路无损施工技术”，将单段电缆无损敷设长度从400多米延长至1500米。该项技术于2019年荣获国家科学技术进步奖二等奖，已在全国电力、石油等行业规模化应用，并成功输出至俄罗斯、新加坡等10余个“一带一路”国家。

“大国工匠年度人物”宣传活动自2018年启动以来，已成功举办六届。本届评选经广泛推荐、严格评审，从制造业、电力、航天等多个关键行业的50位入围人选中遴选出10位年度人物。
