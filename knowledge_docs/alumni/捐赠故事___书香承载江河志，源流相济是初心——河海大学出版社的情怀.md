# 捐赠故事 | 书香承载江河志，源流相济是初心——河海大学出版社的情怀

> 来源: https://hhuedf.hhu.edu.cn/2026/0421/c26739a324985/page.htm

江河奔涌处，文脉永续时，正是在这样深厚绵长的文化源流中，河海大学出版社始终将社会效益放在首位，坚守出版初心。自2019年起，河海社连续在教育部和省委宣传部组织的社会效益评价考核中获评“优秀”；其中，在省委宣传部2021年与2024年度的考核中均取得满分，位列全省第一（并列）。近年来，河海社入选了数十项国家出版基金项目、国家“十四五”重点图书出版物规划项目、国家社会科学基金中华学术外译项目等国家级重点项目，2次获评“国家出版基金优秀项目承担单位”，4次入选“中国图书海外馆藏影响力出版100强”。数百种图书荣获中国出版政府奖提名奖，江苏省新闻出版政府奖、科技部全国优秀科普作品、自然资源部优秀科普图书、交通运输部优秀科普图书、徐迟报告文学奖、“中国最美的书”、“苏版好书”、“长江好书”等国家、省部级及行业奖项；多部图书入选权威推荐榜单；多部图书获《人民日报》、《光明日报》、中央电视台、新华社等主流媒体报道，多篇报道阅读量超百万。

初心为磐：以校友丛书为媒，镌刻河海精神印记

百十年来，一代代河海人将“大哉河海奔前程，毋负邦人期”的期许化为步履，他们的身影遍布国家命脉所系之处：在滔滔江河之畔奠基立业，在群山峻岭之间穿针引线，亦在异国他乡的工程现场挥洒智慧。记录河海人奋斗的足迹，传颂他们感人的故事，是对河海集体记忆的珍贵书写，更是永续“河海精神”的必要之举。正因如此，“河海大学校友系列丛书”在学校的主导与擘画下，应时而著，并蔚然成系。在“河海大学出版社校友系列丛书”的出版历程中，河海大学出版社始终怀持敬畏之心，深度参与丛书策划构思、内容打磨与宣传推广的全流程。以专业之力精雕细琢，让一代代校友的奋斗热忱与家国情怀，跃然于字里行间。这些书籍记录事迹也传递精神，连接过往也照亮前路。它们如同一条条无形的纽带，将天南海北的河海人紧密相连，让“河海精神”生生不息。从建校80周年的《河海人的风采》，到华东水利学院建院50周年的《群星璀璨》《春华秋实》，从建校100周年的《河海大学校友风采》到如今的《港珠澳大桥工程建设中的河海校友》《深圳改革开放40年河海人的风采》《乘风破浪正当时——河海校友企业家风采（第一辑）》《南水北调工程中的河海人》，再到建校110周年的《壮哉，港建！》《一生一事——我和夏金仙老师的故事》，这套丛书已成为河海人共同记忆的珍贵载体。

值得一提的是，河海大学出版社向全省170所高校13467名辅导员（捐赠数字不包括河海大学）捐赠了《一生一事：我与夏老师的故事》一书。该书被列为全省高校辅导员专题学习的重要读物，以推动全省高校辅导员学习夏金仙老师“择一事、终一生”的学工精神，助力师德传承与职业精神建设。捐赠活动被中国教育报、中国新闻出版报、新华日报交汇点、南京大学官网等报道。

水到渠成：以公益捐赠为翼，反哺育才杏坛沃土

如果说出书是对文化传承使命的践行，那么捐赠则是情怀的回响。河海大学出版社深知，自己的根脉深深扎在母校的学术沃土之中。怀着反哺之诚，出版社将捐赠视为对这片沃土最直接的回应，让从这里汲取的养分，重新浇灌未来的生长。2015年，河海大学出版社向学校捐赠200万元，为学校学科建设、人才培养注入温暖动能。2025年，河海大学出版社捐赠2200万元，以实际行动再次助力学校教育事业发展。数字的背后，是出版社与学校同舟共济的坚定承诺，这份反哺，源于对母校的赤子之爱，更源于对“育人即育未来”的深刻理解。河海大学出版社将经营所得，化为助力学校教育事业发展的温暖力量，实现了从“文化的记录者”到“教育的共建者”的角色升华。
