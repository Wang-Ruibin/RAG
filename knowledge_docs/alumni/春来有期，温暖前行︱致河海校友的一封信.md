# 春来有期，温暖前行︱致河海校友的一封信

> 来源: https://alumni.hhu.edu.cn/2020/0306/c26691a202071/page.htm

亲爱的河海校友们：

展信安好！

庚子年春天如约而至。当前，全国防控新冠肺炎疫情进入关键时期，一系列疫情防控措施有效实施，各族人民凝聚起和衷共济、群防群治的强大合力，共同抗击新冠肺炎疫情的蔓延。在此我们向所有奋战在抗疫前线的医务工作者以及支持抗击疫情的各地校友组织、校友，致以崇高的敬意！

面对困难，我们休戚与共，在疫情防控的主战场，我们都是参与者。与此同时，我们时刻惦念着每一位校友，你们和家人的健康平安是我们最大的希望。针对疫情，希望你们积极响应国家号召，从自我做起，做好安全防护，鼓励家人一道，共同配合疫情防控。保持乐观心态，积极传递正能量，为战胜疫情作出力所能及的支持，携手打赢这场疫情防控阻击战。

春来有期，温暖前行。鉴于当前国内相关医疗物资紧缺，为更好地救治病患，抑制疫情，我们倡议校友们继续伸出援手，分享防护物资信息，拓宽采购渠道，汇聚爱心，凝聚力量，全力以赴，为坚决打赢疫情防控阻击战贡献一份力量。待春暖花开，摘下口罩，我们在河海，一起温暖相拥！
