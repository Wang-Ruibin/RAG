# 最高奖金10万元！首届河海大学全球校友创新创业大赛，等你来报名！

> 来源: https://alumni.hhu.edu.cn/2024/1122/c26691a295352/page.htm

10月27日，第一届河海大学全球校友创新创业大赛正式启动！大赛设置创新创业成果展、行业高峰论坛、走近科创板实训、走进上市公司和央企等专项活动和综合交流活动，以赛聚智、以赛引才、以赛筑梦，助推校友经济高质量发展，打造“政产学研金”融合发展的创新创业生态！大赛奖金最高10万元，心动不如行动，快来一起看看吧！

0 2 大赛时间

2024年11月-2025年10月

0 3 组织机构

指导单位： 河海大学

主办单位： 河海大学校友会

承办单位： 河海大学南京校友会、 河海大学校友企业家联谊会

协办单位： 河海大学各地校友会、 河海大学创新创业学院、河海大学产业技术研究院

0 4 参赛对象及条件

大赛重点征 集 节水与水治理、新能源、新材料、双碳环保、数字经济与人工智能、智能装备、智慧农业 等领域项目，具体参赛条件如下：

1.参赛项目核心成员应为河海大学校友或师生。

2.参赛项目包括河海大学地方研究院、院（系）科技成果转化项目，以及河海校友担任创始人、股东、高管或参与投资的项目。

初创组 ：尚未注册公司或公司注册时间不超过2年（2022年12月1日后），有完整创业计划书，具备创新性，同时具备落地发展条件。

成长组： 已注册公司，且注册时间超过2年（2022年11月30日前），项目具有创新性技术、产品或经营服务模式，具有较高成长潜力。

0 5 赛程安排

提交项目报名表及相关资料。

由专家评委对报名项目分行业、分组别进行初步筛选，确定进入线下赛的项目名单。

通过现场路演的方式，由参赛项目向评委展示项目内容、商业模式、团队实力、市场前景、财务数据等方面内容。

行业赛获奖项目、符合直通车条件项目参加半决赛，优秀项目进入总决赛。

项目按初创组、成长组分组进行比赛，决出最终名次。总决赛后即时举行颁奖典礼，期间开展创新创业成果展等活动。

0 6 奖项奖金和激励政策

行业赛： 一等奖2名，二等奖4名，三等奖6名（分初创组、成长组，颁发证书）

总决赛： 特等奖2名，一等奖4名，二等奖6名，三等奖8名（分初创组、成长组，颁发证书和奖金）
