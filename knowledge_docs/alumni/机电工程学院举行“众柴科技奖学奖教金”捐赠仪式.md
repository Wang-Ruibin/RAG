# 机电工程学院举行“众柴科技奖学奖教金”捐赠仪式

> 来源: https://hhuedf.hhu.edu.cn/2025/1219/c26739a315218/page.htm

10月28日，“众柴科技奖学奖教金”捐赠仪式在河海大学常州校区举行。成都众柴科技有限公司执行董事、机电工程学院1994级梅燕校友，机电工程学院党委书记高洪娟，副院长王延杰、苑明海，师生代表等参加仪式。仪式由学院党委副书记、副院长张亿全主持。

梅燕校友深情回顾了在母校求学的历程和职业成长经历，他表示，四年求学时光为其后续发展打下坚实的专业基础。在母校110周年校庆之际，以设立奖学奖教金的方式反哺母校，希望能够激励更多师生投身水工金属结构领域前沿研究，传承河海大学“艰苦朴素、实事求是、严格要求、勇于探索”的精神，为智慧水利建设贡献力量。

教师代表蔡伟感谢梅燕校友的慷慨善举，表示将深耕教学科研一线，为水工金属结构行业培养更多高素质专业人才。2023级机械专业硕士张佳星以自身在水工金属结构设计领域的实习经历，表达了对梅燕校友的敬佩，并表示将以校友为榜样，潜心钻研人工智能与水工结构安全评估结合的课题，未来积极投身水利事业，不负校友与学校的期望。

高洪娟代表学院向梅燕校友致以衷心感谢，并表示梅燕校友的捐赠是校友反哺母校的生动实践，彰显了校友对母校的深厚情谊与责任担当，为全院师生树立了榜样。师生将以此次仪式为新起点，将这份感恩与激励转化为奋进的力量，学院也将进一步加强校友工作，搭建更多学院与校友交流合作的平台，汇聚校友智慧与力量，为学校发展注入源源不断的动力。

校友简介

梅燕，男，1974年12月出生，河南省信阳市固始县人，高级工程师，1998年毕业于河海大学机电工程学院金属结构与焊接专业，毕业后先后任职于水利部水工金属结构质量检验测试中心、雅砻江流域水电开发有限公司，现任成都众柴科技有限公司董事长，河海大学四川校友会副会长，九三学社社员，全国水工金属结构专业委员会委员。参与主编国家GB标准1项、国家能源局NB行业标准4项，申请专利50多项（其中发明专利10项），研发的水工金属结构设备在线监测系统填补了国内市场空白。
