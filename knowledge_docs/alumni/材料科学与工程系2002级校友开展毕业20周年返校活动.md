# 材料科学与工程系2002级校友开展毕业20周年返校活动

> 来源: https://alumni.hhu.edu.cn/2026/0514/c26687a326987/page.htm

日前，材料科学与工程系2002级校友开展毕业20周年返校活动，重温求学岁月，共叙同窗情谊。材料科学与工程学院党委书记陈伟刚，时任授课教师蒋林华、王泽华等，时任辅导员汤鸣鸿、赵燕容等与校友座谈交流。

陈伟刚代表学院对校友们的返校表示热烈欢迎，感谢校友们一直以来对学院和专业发展的关心与支持，介绍了学院近年来在学科建设、人才培养、科研创新等方面的发展成果，并希望校友们继续关心关注支持学校和学院发展，在科研合作、成果转化等方面贡献智慧和力量。

校友们分别分享了毕业后的职业发展与人生感悟：在科研一线攻坚克难，用专业能力诠释河海材料人的担当；在工程建设领域奋战，参与港珠澳大桥等大国工程，将母校所学付诸实践；在市场浪潮中拼搏，书写着属于自己的精彩；或在全新的领域里探索前行，践行当年的初心与誓言。

老师们认真倾听每一位校友特殊的“青春答辩”，对校友们的职业发展寄予殷切期许，反复叮嘱无论工作多忙碌，都要劳逸结合、保重身体，才能更好地追逐梦想、守护家庭、成就自我。温情时刻，校友们手捧鲜花，走向当年的任课老师、辅导员，献上最诚挚的敬意与祝福。一束束鲜花，承载着二十年来未曾淡去的师生情谊；一个个拥抱，诉说着学子对恩师教诲的深深感激。没有华丽的辞藻，没有盛大的仪式，却在一言一行间，彰显着最朴素、最真挚的情感。

在西康路校区的林荫道上，校友们一路走、一路看、一路聊，二十年前的求学时光历历在目。教学楼、实验室、宿舍楼下、操场边，每一处角落都留存着青春的印记，每一段回忆都饱含着真挚的情怀。当年课堂上的专注聆听、实验室里的潜心钻研、宿舍里的深夜卧谈、运动场上的热血拼搏、毕业时的依依不舍，都是彼此心中最珍贵的珍藏。

随后，校友们沿着江宁校区熟悉的路线漫步，一路探寻青春足迹。在校园的一草一木间，彼此诉说着当年的求学往事。在致用楼117教室的门牌前驻足凝视，思绪瞬间被拉回二十年前的课堂，仿佛又看到了当年背着书包、奔赴教室的自己。运动场上，昔日并肩奔跑、挥洒汗水的少年，如今再度集结，褪去职场的沉稳，重拾青春的热血。

夕阳西下，校友们依依不舍地漫步在校园里，再次回望熟悉的教学楼、实验室，再次与同窗、恩师道别。一句句简单的道别，承载着最深厚的情感，有对青春岁月的眷恋，更有对未来的期许。

时光荏苒，廿载春秋；初心未改，情谊弥坚。毕业20周年返校，是一场青春的重逢，更是一次初心的回望。校友们祝愿母校再创辉煌，在新时代绽放更加耀眼的光芒！校友们也将带着青春的记忆与母校的期许，在各自的领域继续发光发热，书写更加精彩的人生篇章。
