# 校友系列丛书之《壮哉，港建！》新书发布会在河海大学举行

> 来源: https://alumni.hhu.edu.cn/2025/1028/c26684a309183/page.htm

10月25日下午，《壮哉，港建！》新书发布会在河海大学西康路校区举行。校长郑金海、1980级军港建筑专业校友崔琦、徐明洪、林育平，1980级力学专业校友郭建中，校原党委副书记郑大俊等出席活动。

郑金海强调，军港建筑专业自创立之初便肩负“为国防解决迫切问题”的使命，一批批港建校友为国家国防事业、大国重器建设作出了“惊天动地事”，为学校赢得了广泛赞誉。激励河海学子接过前辈的接力棒，接续加入国防建设工作，勇担重任和使命，做到传承和创新。

郭建中代表编写组介绍书籍编撰初衷、撰写经过。他表示，创作过程中多次被港建人的赤诚、坚守、卓越和担当感动，港建人的人生壮怀激烈，港建人的事业波澜壮阔，港建人的信仰国家至上，使命如山，壮哉，港建。

林育平代表港建专业校友回忆了在港建楼学习的青春岁月，他感谢学校的高度重视和支持，感谢为这本书付出心血的校友，也期待“港建精神”在一代代河海学子心中薪火相传。

会上，进行了新书发布，校领导、作者代表向师生代表赠书。

校友代表、相关职能部门负责人、港口海岸与近海工程学院师生代表等参加活动。
