# 校长郑金海看望河南校友并座谈交流

> 来源: https://alumni.hhu.edu.cn/2025/0908/c26684a306890/page.htm

7月13日，校长郑金海赴河南郑州看望校友并座谈交流。

郑金海代表学校向河南各界校友致以诚挚问候，对长期以来关心学校发展的各界校友表示衷心感谢。他向校友们介绍了近年来学校发展情况和取得的成就，并介绍了学校庆祝建校110周年相关工作。他强调，校友是学校发展历程中的宝贵财富，校友组织是广大校友延续深情厚谊、携手共谋发展的重要平台。他希望，河南校友继续关心支持学校发展，加强交流与联系，共同搭建情感联络、互助合作、资源共享的平台，并诚邀校友们金秋十月荣归母校，共襄百十盛举。

座谈会上，与会校友踊跃发言，介绍了自己的求学经历和工作情况，围绕学校学科建设、人才培养、产学研融合、校友母校共赢发展等内容进行了深入交流。

河南校友代表、学校相关职能部门和学院负责人参加活动。
