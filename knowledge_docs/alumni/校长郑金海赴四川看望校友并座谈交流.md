# 校长郑金海赴四川看望校友并座谈交流

> 来源: https://alumni.hhu.edu.cn/2026/0215/c26684a323127/page.htm

2月11日，校长郑金海在四川调研期间，看望在川校友并座谈交流，向广大校友致以新春祝福。

郑金海感谢四川校友会及广大在川校友长期以来对学校学科建设、科学研究、人才培养等方面工作的关心和支持，并代表学校向校友们致以诚挚问候。他介绍了学校近年来的建设发展情况、百十校庆系列活动成果，以及在拔尖创新人才培养、关键技术研发、重大工程支撑、科研平台建设、高层次人才引育、校地校企合作等方面取得的成绩，希望四川校友会继续发挥桥梁纽带作用，加强交流互动，服务校友发展，助力学校与四川的产学研融合和“学校校友发展共同体”建设，共同服务国家战略实施和四川地方经济社会发展。

四川校友会名誉会长王源致欢迎词，并介绍了四川水利能源发展的有关情况，表示四川校友会将继续发挥联结学校与四川水利、能源事业深度合作的平台作用，广泛凝聚校友情谊，共促校地共赢发展。

中国电建成都院特聘专家、国家卓越工程师、全国工程勘察设计大师王仁坤在发言中介绍了自己投身水利领域工作40余年的工作经历和人生感悟，感念学校的培养和关心支持，并表达了愿与母校及校友们并肩前行，共同为雅下工程等国家重点工程高质量开发建设贡献河海力量。

河海大学国内合作发展处处长、校友会办公室主任朱乔在发言中表示四川校友会各项工作走在了全国前列，并介绍了学校校友工作有关情况。四川校友会秘书长乐成军主持座谈，并汇报了四川校友会在凝聚校友力量、助力学校、校友与地方共同发展等方面的工作进展和规划。

与会校友纷纷表达了对母校培养的感激之情，分享了在各自岗位上推动行业发展、服务社会经济的实践与思考。校友们表示，将在母校的引领下，继续弘扬河海精神，为母校发展、行业进步与地方经济社会建设贡献更多力量。

学 校相关职能部门、学院负责人，教师代表，四川校友会理事会相关成员，在川校友代表等参加活动。
