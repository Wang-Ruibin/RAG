# 校长郑金海赴新疆看望校友并座谈交流

> 来源: https://alumni.hhu.edu.cn/2025/0819/c26684a306354/page.htm

8月14日，校长郑金海在新疆调研期间，看望在疆校友并与校友代表座谈交流。

郑金海向新疆校友们致以亲切的问候，介绍了学校近年来在“双一流”建设、人才培养、科研创新等方面的工作进展和成绩以及学校建校110周年相关工作进展。他对新疆广大校友在水利工程、生态环保、边疆建设等领域作出的重要贡献表示感谢，对新疆校友会在联络校友、服务校友等方面的工作表示肯定。他指出，学校建设发展取得的成就离不开新疆校友们的鼎力支持，诚挚邀请校友们金秋十月荣归母校、共商发展。

新疆校友会秘书长买合木提江·卡地尔代表新疆校友会第二届理事会汇报相关工作，重点介绍了新疆校友会在组织建设、校地合作等方面工作情况，详细汇报了新疆校友会庆祝建校110周年校旗传递、校友企业走访等活动组织开展情况。

座谈会上，校友们满怀对母校的感恩踊跃发言，深情回顾在母校求学期间的难忘岁月。大家纷纷表示，母校艰苦朴素、实事求是、严格要求、勇于探索的校训精神深深影响着每一位河海人，未来将继续立足岗位、奋发进取，为母校事业发展和新疆建设发展作出新的贡献。

学校相关职能部门负责人、援疆干部代表以及20余位在疆校友代表参加活动。
