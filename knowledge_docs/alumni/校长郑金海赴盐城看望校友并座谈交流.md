# 校长郑金海赴盐城看望校友并座谈交流

> 来源: https://alumni.hhu.edu.cn/2026/0325/c26684a323944/page.htm

日前，校长郑金海带队赴盐城调研，与盐城校友座谈交流。

郑金海代表学校向盐城校友致以诚挚问候，对盐城校友会长期以来关心支持母校建设发展、助力地方水利事业进步表示衷心感谢。他介绍了学校近年来在人才培养、学科建设、科研创新、社会服务、校地合作等方面的发展成果，强调校友是母校最宝贵的财富，是学校高质量发展的重要支撑，并希望盐城校友会继续发挥纽带作用，凝聚校友力量，推动学校与校友、地方共赢发展。

盐城校友会相关负责人汇报了在校友联络服务、组织建设、活动开展等方面情况以及未来工作规划，表示盐城校友会将持续加强校友联动，广泛凝聚校友力量，助推互利共赢和协同发展。

与会校友纷纷追忆在河海的求学时光，分享各自在水利勘察设计、市政建设、教育、金融、法律等不同岗位的奋斗历程与实践感悟，感念母校的培育之恩，为母校建设发展建言献策，表示将始终心系母校，发挥优势，牵线搭桥，为母校在盐招生贡献力量，并期待与母校携手并肩、共同发展，为地方、行业发展做出更多贡献。

调研期间，郑金海一行专程赴江苏省盐城中学交流座谈招生宣传、人才培养等事宜。学校相关职能部门、学院负责人，盐城校友会相关负责人、校友代表等参加活动。
