# 校长郑金海赴辽宁、吉林、黑龙江调研并看望校友

> 来源: https://alumni.hhu.edu.cn/2025/0729/c26684a306069/page.htm

7月23-25日，校长郑金海率队赴辽宁、吉林、黑龙江水利行业单位调研并与当地校友座谈交流。

郑金海代表学校向东北各界校友致以诚挚的问候，向一直以来关心支持学校建设发展的校友们表示衷心的感谢。他介绍了学校近年来在科学研究、学科建设、人才培养、校地融合、产学研合作、校友工作等方面的工作进展和成就，以及学校庆祝建校110周年相关工作情况。

郑金海对校友们作出的贡献表示感谢，他强调校友是学校的宝贵财富和璀璨名片，希望校友们与母校共同携手为水利行业高质量发展和东北全面振兴贡献智慧与力量，并诚邀校友们校庆之际荣归母校，共襄盛举。

座谈会上，辽宁、吉林、黑龙江校友们纷纷表达对母校培养的感恩和深情厚谊。校友们分别围绕加强校友联络、服务校友发展、庆祝母校百十华诞、推进学校与校友发展共同体建设、联合服务行业及地方发展等方面畅所欲言，表示将会持续关心母校发展，为母校发展贡献力量。

郑金海一行还走访了水利部松辽水利委员会、黑龙江省水利厅、辽宁省水利厅、吉林省水利水电勘测设计研究院等校友所在单位，就强化行校合作对接与各单位进行了深入交流，希望通过多方产学研深度融合，共同助力服务国家水利行业高质量发展和新时代东北全面振。

学校相关职能部门、学院负责人，辽宁、吉林、黑龙江校友代表等参加活动。
