# 校长郑金海赴连云港看望校友并座谈交流

> 来源: https://alumni.hhu.edu.cn/2026/0422/c26684a325544/page.htm

4月16日-17日，校长郑金海带队赴连云港调研，与连云港校友座谈交流。中国工程院院士胡亚安出席活动。

郑金海代表学校，对胡亚安院士亲临中学授课、大力支持本科招生宣传工作表示感谢，向连云港校友致以诚挚问候，并对连云港校友会长期以来关心支持母校建设发展、积极服务地方经济社会发展表示衷心感谢。他强调校友是学校办学最深厚的底气与最大的资源，并希望连云港校友会深化校友工作内涵，助力“学校校友发展共同体”建设，带领广大校友立足各自岗位、深耕行业领域，以实干实绩为母校赢得更广泛社会声誉。

连云港校友会相关负责人汇报了在校友联络服务、平台搭建、资源互动等方面工作成绩，希望与母校在科研项目合作、校企课题攻关、专家授课交流、共建实习基地和人才培养等领域加强对接，实现资源共享、互利共赢。

与会校友深情回顾在校求学时光，分享在各自行业领域的奋斗历程与成绩，并表示将始终心系母校，全力支持母校各项事业发展。

在连 期间，郑金海一行专程赴连云港市水利规划设计院有限公司、江苏省新海高级中学围绕校企合作、本科招生宣传、人才培养等进行交流。学校相关职能部门、学院负责人，连云港市本科招生宣传组负责人，连云港校友会相关负责人、校友代表等参加活动。
