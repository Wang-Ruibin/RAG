# 校长郑金海赴陕西调研并与校友座谈交流

> 来源: https://alumni.hhu.edu.cn/2025/0815/c26684a306272/page.htm

8月9日，校长郑金海赴陕西看望校友并与校友座谈交流。

郑金海向陕西校友致以亲切的慰问，他介绍了学校近年来的建设发展情况、110周年校庆相关工作进展，以及学校定点帮扶工作情况。他感谢陕西校友对学校各项工作的支持和关心，特别指出，陕西校友积极投身定点帮扶工作，是校友助力学校事业发展的生动实践，更展现了陕西校友对社会发展的责任担当。他希望陕西校友继续关心、支持学校，助力构建学校校友发展共同体，并诚挚邀请校友们金秋十月荣归母校，共襄盛举、共叙发展。

座谈会由陕西校友会副会长辛乾龙主持，副会长兼秘书长郑伟、副会长申卫华分别介绍了陕西校友会近年来在校友服务联络取得的成绩以及未来的工作计划，并特别介绍了河海大学110周年校旗传递（陕西站）活动。名誉会长满斌、副会长刘琰等参会校友代表们依次发言，对母校的培养表示感激，并分享了他们在陕西的奋斗故事、河海情怀和河海精神传承实践以及对母校未来发展的美好期望。

学校相关职能部门、院系负责人，学校定点帮扶干部，陕西校友代表等参加活动。
