# 校领导看望丹麦、德国校友

> 来源: https://alumni.hhu.edu.cn/2025/0603/c26684a303801/page.htm

近日，校党委副书记、纪委书记孟新率河海大学代表团出访丹麦和德国，并看望校友。

孟新向在丹麦、德国的校友们致以亲切问候。她分享了学校近年来的发展情况，感谢校友们一直以来对母校的关心和支持。她表示，校友与学校始终是休戚与共的发展共同体，希望校友们持续加强与母校的合作交流，继续为母校建设发展出谋划策，并诚邀校友们荣归母校。

交流结束后，孟新与校友们合影留念，并参加丹麦、德国校友举行的庆祝建校110周年校旗传递活动。

学校相关职能部门和学院负责人，在丹麦、德国的校友代表参加活动。
