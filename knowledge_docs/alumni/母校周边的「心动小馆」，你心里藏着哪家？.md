# 母校周边的「心动小馆」，你心里藏着哪家？

> 来源: https://alumni.hhu.edu.cn/2025/0822/c26691a306402/page.htm

亲爱的校友们，还记得吗？每逢周末，约上同学去西康路校区旁的华清园餐馆点上几样家常菜，就像在学校吃到了 “家的味道”；期末复习季，在金良酸菜鱼点一大份酸菜鱼，辣得冒汗却笑得开怀。毕业散伙饭那天，江宁校区川人川菜老板多送的那瓶饮料，和满桌的饭菜香一起，成了记忆里最难忘的味道。天冷时与三五好友到常州校区旁的金山城火锅聚餐，热热闹闹的烟火气，暖了整个青春的冬天。那些藏在母校周边的小餐馆、小吃店，或许店面不大，却承载着我们最鲜活的校园时光。
