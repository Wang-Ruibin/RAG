# 水利水电学院2002级校友开展毕业20周年返校活动

> 来源: https://alumni.hhu.edu.cn/2026/0513/c26687a326903/page.htm

日前，水利水电学院2002级两百余名校友从全国各地重返母校，以“廿载水之子，江河再汇流”为主题，举办本科毕业20周年返校活动。水工、热动、农水专业校友师生座谈会分别在西康路校区、江宁校区同步举行。水利水电学院党委书记娄健，农业科学与工程学院党委书记刘兴平、院长徐俊增，新能源学院院长许昌，水利水电学院副院长冉启华，时任水利水电学院党委副书记余达淮、副院长郑源、水电系主任蔡付林、热动系主任屈波、辅导员高玉琴等参加活动。

娄健代表学院对校友们的归来表示热烈欢迎，系统介绍了近二十年来学校、学院的发展情况，细数学院长期以来的办学育人成果。与会教师与校友们亲切交流，共同回忆二十年前的课堂学习与校园生活，重温师生情谊，并为校友们送上殷切寄语与诚挚祝福。

座谈会上，校友们踊跃发言，汇报了二十年来的工作历程与成长收获，纷纷表示将始终坚守校训精神，持续关注和支持母校与学院发展，为国家水利水电事业高质量发展贡献力量。座谈会后，校友们漫步校园，重访教学楼、学生宿舍、学生食堂等熟悉场所，切身感受母校的发展变化。

返校期间，校友们相聚在西康路校区开展主题团建活动。拔河比赛、夹球挑战、一圈到底等趣味竞技项目，唤醒了大家沉睡二十年的校园青春记忆。校友们热情高涨、默契协作，在轻松欢乐的氛围中重温校园时光，进一步凝聚了同窗情谊，现场气氛热烈融洽。时任辅导员高玉琴重温了师生相处的温暖时光，为校友们送上真挚的祝福与期许。活动结束，全体人员合影留念，定格跨越二十载的美好相聚，共同为母校、母院送上真挚祝福。

校友们重走熟悉校园路，重温炽热求学梦，再聆谆谆恩师语，共话绵长同窗情，在阔别二十载的校园里，续写属于河海水电人的青春与荣光。

此次相聚不仅是一场青春的重逢，更是一次力量的凝聚。学院持续打造的“校友 - 学院 - 母校发展共同体”，将进一步拉近校友与母校的距离，凝聚校友力量，深化校友情怀，让江河情怀奔涌不息，让水电精神代代相传。

愿全体2002级校友历尽千帆，初心不改，常回母校看看，携手并肩为中国水利水电事业的高质量发展，持续贡献河海水电人的智慧与力量！
