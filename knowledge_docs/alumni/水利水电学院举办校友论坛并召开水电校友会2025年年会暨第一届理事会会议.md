# 水利水电学院举办校友论坛并召开水电校友会2025年年会暨第一届理事会会议

> 来源: https://alumni.hhu.edu.cn/2025/1107/c26687a309812/page.htm

10月26日，水利水电学院举办校友论坛并召开水电校友会2025年年会暨第一届理事会会议。原水电系主任沈祖诒，原水电系副书记、1975级水动专业校友姜晓平，中国人民武装警察部队水电指挥部总工程师、1963级水工专业校友梅锦煜将军，中国工程院院士、1984级水工专业校友唐洪武，全国工程勘察设计大师、1980级水工专业校友刘志明，中国电建川藏铁路指挥部副指挥长、1986级水工专业校友吴越等出席会议。水利水电学院各地、各年级80余位校友代表，学校相关部门领导、学院班子成员、师生代表等参加本次大会。会议由水利水电学院党委书记娄健主持。

水利水电学院院长王媛代表学院向长期以来始终心系母校、支持学校、学院建设与发展的各位校友表示，并介绍了学院在科技创新、人才培养、国际交流等领域取得的成绩。期盼校友们坚守初心，勇担使命，在水利事业发展中再立新功；常念母校，情系学院，在产学研合作中深化共赢；薪火相传，引领后学，为国家强国建设、民族复兴伟业贡献河海水电人的智慧和力量。

在校友论坛阶段，唐洪武强调校友是学院最宝贵的资源之一，要注重校友文化建设，通过校友企业合作等方式，拓展学院发展资源，提升学院办学实力。一要秉承务实作风、甘于奉献，学院的发展离不开一代代水电人脚踏实地、埋头苦干的精神传承；二要加强高质量人才培养，着力打造一支具有工程素养和创新能力的高素质人才队伍；三要优化学科布局，突出优势特色，强化交叉融合，提升学科综合实力和核心竞争力，推动科研成果高水平产出。

刘志明从个人求学工作经历、项目经历出发，分享自身感悟心得。从人才培养、学院发展与行业趋势三个方向，为学院发展建设提出宝贵建议。

吴越建以“以水为师，笃行致远”作为主题分享了自己毕业后的工作经历并给出关于项目管理要素、高素质复合型人才培养与团队管理经验的心得体会。

在水电校友会2025年年会暨第一届理事会会议阶段，娄健宣读了河海大学校友会发布的《关于河海大学水电校友会第一届理事会成员审定结果的公告》，宣布河海大学水电校友会第一届理事会正式成立。

水工1978级校友陈映坚，水电校友会会长陈映坚发言表示水电校友会将尽职尽责，联系校友、服务校友、宣传母校、奉献母校。

水电校友会秘书长、水工2003级校友杨志帅表示，校友会将汇聚发展合力，积极发挥职能，真正成为情感的纽带、合作的平台、发展的引擎。

水电校友会副会长、1979级水动专业校友沈一鸣，1981级水工专业校友金惠英，1994级水工专业校友黄剑，水电校友代表、2008级水工专业校友刘亚琴，水电校友会副秘书长、2010级水工专业校友张召依次发言，他们深情回忆在校求学岁月，表达对母校培育的感恩之情，表示将继续关注支持学院发展，更好服务广大校友，积极投身水利事业建设。

新一届理事会成员一致表示，将恪尽职守、凝心聚力，全力服务校友成长，积极汇聚校友资源，为学院与学校的蓬勃发展贡献智慧与力量，共同开创校友工作新篇章。

新一届理事会谨此向广大水电校友发出诚挚倡议：让我们紧密依托水电校友会，加强彼此之间的联系与交流，恪守河海大学“艰苦朴素、实事求是、严格要求、勇于探索”的校训，继承和发扬追求卓越的水电精神。携手共进，共同谋划行业发展大计，将深厚的校友情谊转化为高质量发展动能，为新时代水利事业贡献河海水电的智慧和力量！

活动的举办进一步加强了学院与广大校友间的紧密联系，为校企合作、科研创新搭建了重要交流平台。学院将以此为契机，持续激发办学活力，不断提升影响力，凝聚校友与社会各界力量，共同为国家水利水电事业高质量发展作出新的更大贡献。

经水利水电学院推荐，水电校友会广泛征求意见，河海大学校友会审核，产生水电校友会第一届理事会成员。名单如下:

对水电校友会理事会对吴钢（水工85级）、皋唱（水工87级）、甄莉萍（水动87级）、吴静（水工90级）、柴洪斌（水经90级）、郑福寿（水工91级）、赵大海（水工95级）、祁峰（水动99级）、吴国颖（水动02级）、刘亚琴（水工08级）等10位校友在水电校友工作中做出的热心奉献给予衷心的感谢。
