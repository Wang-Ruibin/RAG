# 水利水电学院水动专业1981级校友举办毕业40周年返校活动

> 来源: https://alumni.hhu.edu.cn/2025/1111/c26687a309920/page.htm

四十周年再聚首、共话同窗河海情。在庆祝河海大学建校110周年之际，水利水电学院水动专业1981级校友举行毕业四十周年返校活动。校友们一路走访了苏州高等研究院与常州校区，最终回到西康路校区。大家欢聚一堂，共叙同窗情谊，重温青春岁月，并亲身见证了母校的跨越式发展与历史变迁。

10月17日下午，校友们参观河海大学苏州高等研究院并举办座谈会。苏州高等研究院负责人从学科建设、人才培养、科研成果、校企合作等方面详细介绍了苏州高等研究院建设发展，并诚邀广大校友继续关心支持母校和研究院建设，积极为母校高质量发展建言献策。座谈会上，校友们纷纷回忆当年的求学生活，对母校日新月异的发展变化感到由衷自豪，结合自身工作经历分享了成长体会，表示将一如既往地关注母校发展，持续为学校“双一流”建设贡献力量。

在常州校区，校友们在图书馆南大门前合影留念。新校区建设落成与校友重返校园，生动展现了学校办学历程的变迁与时代发展的步伐。校友们参观了常州校区图书馆、若水广场等地，登上公共教学楼E楼顶观景台俯瞰新校区全貌，实地了解新校区在教学科研、生活配套等方面的建设成就，学校的发展给校友们留下了深刻印象。

西康路校区承载着校友们青春的记忆，是求知与奋进的起点。校友们相约西康路1号大门前合影留念，共同兑现四十年前的青春之约。 随后，校友们重游故地，从熟悉的工程馆、图书馆到曾经的宿舍、体育场，一处处旧景勾起了无数往昔回忆，让人深切感受到岁月的沉淀与情感的延续。在西康路校区教工之家，校友们聆听了校、院领导关于学校发展的介绍，大家畅叙情谊，共话母校未来。

四秩辉映水韵长，踔厉奋发谱新章。本次水动1981级毕业40周年校友返校活动既是情感的回归、精神的交流，更是凝聚力量、共谋发展的盛会。通过座谈交流、实地参观，校友们更加了解了母校的发展现状，增强了与母校的联系，也为学校校友工作注入了新的活力。未来，学院将持续深化与校友的联系与合作，进一步推动校友资源与学校发展深度融合，共同谱写母校高质量发展的崭新篇章。
