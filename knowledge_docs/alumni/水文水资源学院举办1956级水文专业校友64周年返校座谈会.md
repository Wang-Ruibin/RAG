# 水文水资源学院举办1956级水文专业校友64周年返校座谈会

> 来源: https://alumni.hhu.edu.cn/2025/1111/c26687a309912/page.htm

金秋十月，丹桂飘香。近日，水文水资源学院1956级水文校友毕业64周年返校座谈会于西康路校区召开。水文水资源学院党委书记吴志勇出席并主持座谈会，与阔别校园数十载的校友们共忆青春岁月，同话学院发展。

吴志勇代表学院向远道而来的校友们致以最诚挚的问候，他的话语中满含对前辈们的敬意：“时隔64年，各位校友重返母校，这份对河海的深情令人动容。欢迎大家回家，感谢您们始终心系学院发展！”随后，吴志勇围绕学院历史沿革、机构设置优化、师资队伍建设、国际合作交流、近年来的亮点成就以及杰出校友事迹等多个方面，全面介绍了学院的发展情况。从水文学科的优势巩固，到科研平台的不断升级；从青年教师的培养与支持，到学生创新能力的提升，吴志勇的讲述条理清晰、数据详实，让校友们直观感受到母院日新月异的变化。

交流环节，校友们纷纷打开话匣子，分享着自己与河海的故事。原华东水利学院水文系党总支书记 徐健校友 望着熟悉又陌生的校园，感慨万千：“再次踏进校园，真是既亲切又惊喜！教学楼翻新了，实验室更先进了，学院取得的这些成就，让我们这些老校友打心底里自豪！”他深情回忆起当年入学时水文专业的班级设置，提及那些曾悉心教导自己的恩师，言语间满是怀念。随后，他还就学院当前的学科发展、人才培养等问题与吴志勇展开了深入交谈，为学院的未来发展建言献策。

边淑颖校友则难掩激动之情，她眼眶微红地说：“回到母校，就像回到了自己的家！当年在学校学到的扎实基础知识，是我一辈子的财富，让我在工作岗位上能够从容应对各种挑战，也得到了行业内的广泛认可。”她特别感谢学院组织此次座谈会，让自己有机会与老同学们重逢，“看到这么多熟悉的面孔，看到大家在水利行业发光发热、为国家建设贡献力量，我真的太高兴了！”边淑颖校友还分享了一个温暖的细节：自己的家庭早已成为名副其实的“河海之家”，家中先后有8位成员考入河海大学，传承着河海人的精神与情怀。最后，她动情地祝福在场的每一位校友：“愿大家都能身体健康，幸福长寿！”

在众多校友中，有一位特殊的身影格外引人注目——刘召兰校友。为了参加此次座谈会，也为了赴约母校110周年校庆，她专程从日本赶来，跨越山海的距离，只为回到阔别已久的校园。谈及此次返校的心情，刘召兰校友眼中闪烁着光芒：“得知要举办毕业64周年座谈会，又恰逢母校110周年校庆，我无论如何都要回来！河海不仅教会了我扎实的专业知识，更培养了我做人的道理和良好的道德品格，这些都让我受益终身。”她回忆起在校时的点点滴滴，从课堂上的认真听讲，到实验室里的反复钻研，再到与同学们的互帮互助，每一个细节都历历在目。“这些年，我始终以自己是河海人为荣，时刻关注着母校的发展。看到学院如今的成就，我真的无比骄傲！”刘召兰校友的话语朴实而真挚，她跨越国界、不远万里返校的举动，不仅展现了对母校的深厚情谊，更让在场的师生深受感动，成为现场温暖感人的一幕。

座谈会在温馨而热烈的氛围中圆满结束，随后校友们共同前往学生食堂就餐，体验在校水文学子的大学日常。

此次返校活动，不仅为老校友们搭建了重温青春、畅叙友情的平台，更让他们感受到了母校的关怀与温暖。而以刘召兰校友为代表的校友们对母校的深情眷恋，也将激励着学院全体师生接续奋斗、推进高质量发展。
