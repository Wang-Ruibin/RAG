# 河你一起，云端相聚 河海校友庆祝母校建校110周年云活动上线啦！

> 来源: https://alumni.hhu.edu.cn/2025/1004/c26691a308170/page.htm

云聚河海，遥寄相思。

让你我共同点亮一颗温暖的星，

以这跨越山海的思念之光，

汇聚成河，奔腾入海，

为母校百十华诞送上最深情的祝福。

活动1：点亮全球

让我们以母校之名，

点亮世界每一个角落。

详细操作步骤：

1.进入“ 河海校友 ”小程序；

2.点击【服务】-【校庆活动】-【点亮全球】进入活动；

3.点击【我要点亮】授权获取位置 - 点击【送祝福】为母校送祝福 - 点击【邀请好友点亮】即可生成海报，邀请好友参与活动。

活动2：云合影

天涯未远，镜头为证。

快来与母校同框，

将所有的思念与祝福凝聚于此。

详细操作步骤：

1.进入“河海校友”小程序；

2.点击【服务】-【校庆活动】-【云合影】进入活动；

3.点击【进入活动】选择合影背景 - 点击【上传照片】上传本人照片生成合影 - 点击【保存到相册】可保存合影到相册 - 点击【分享给好友】邀请好友一起参与活动。

活动3：云征集

一张老照片，

定格一段青春。

期待您分享的老照片，

让青春的记忆在光影中重现。

详细操作步骤：

1.进入“河海校友”小程序；

2.点击【服务】-【校庆活动】-【云征集】进入活动；

3.点击【进入活动】可查看校友上传的老照片 - 点击【我要上传】可上传老照片 - 点击【分享】邀请好友一起参与活动。
