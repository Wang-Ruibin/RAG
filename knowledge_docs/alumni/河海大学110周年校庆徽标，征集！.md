# 河海大学110周年校庆徽标，征集！

> 来源: https://alumni.hhu.edu.cn/2024/0720/c26691a286409/page.htm

2025年10月27日是河海大学建校110周年华诞。为展示河海百余年办学经验与办学特色，传承弘扬河海精神文化，经学校研究决定，开展河海大学110周年校庆徽标征集，现将活动的具体安排通知如下：

河海大学110周年校庆徽标

全体师生员工，离退休教职工，广大校友，校外各界人士。

1.作品应充分彰显河海大学办学特色，体现1915-2025百十年办学元素，展示河海大学精神与文化。

2.作品要主题鲜明，简洁明快，寓意深刻，富有美感，构思新颖，喜气庄重。

3.作品的设计应易于识别与传播，适于做延展性设计，便于在校庆专题网站、相关宣传报道、校庆纪念品、校园氛围营造以及校庆活动的展示推广中应用。

4.来稿均须为未公开发表过的原创作品，如作品涉及抄袭、借用等侵权行为由作者本人承担相关后果，与征集单位无关。

1.信息填写

应征者请填写并提交《河海大学110周年校庆徽标征集作品报名表》（见下方附件）， 务必将作者个人信息、作者声明、设计理念（300字以内）等信息填写齐全。

附件：河海大学110周年校庆徽标征集作品报名表.docx

2.格式要求

采用电子版提交作品，需提供A4规格的黑白稿、彩色稿（注明CMYK色值）；logo图案面积不得小于5cmx5cm；提交时请提供设计源文件、矢量图（AI、TIF或PSD格式）；同时提供JPG格式（不小于3M），除艺术设计外的文字请注明字体。

3.投稿方式

请将稿件与《河海大学110周年校庆徽标征集作品报名表》一起发送至电子邮箱：dzb@hhu.edu.cn。请在邮件标题中注明“河海110周年校庆LOGO+作者姓名”字样。

4.投稿时间

即日起至2024年8月31日

1.评审

征集截止后，学校将组织专家对应征作品进行评选，确定最终评选结果和获奖作品。学校将从获奖作品中进行遴选，酌情加以修改或组合完善后，产生最终推广使用的河海大学建校110周年校庆徽标。

2.奖励

（1）一等奖1名（奖金3000元）

（2）二等奖2名（奖金2000元）

（3）三等奖3名（奖金1000元）

（4）优秀奖10名（奖金500元）

除以上奖励外，同时对获奖者颁发相应证书。以上奖金均为税前奖励金额，税费由获奖者个人承担。

1.所收到的应征作品一概不予退还，应征者请自留底稿。

2.所有入围作品的知识产权和使用权均归河海大学所有，未经允许，任何组织和个人不得擅自使用。河海大学有权对入围作品进行任何形式的使用、开发、修改、授权、深化设计、宣传展示、许可或保护等活动。除获奖奖金外，不再支付给创作者其他费用。

1.河海大学主页：http://www.hhu.edu.cn/，可以提供河海大学相关背景资料，欢迎浏览。

2.咨询联系人：瞿老师、陈老师 联系方式： 025-83786236，83786777

3.本次征集活动由党政办公室、宣传部负责解释。
