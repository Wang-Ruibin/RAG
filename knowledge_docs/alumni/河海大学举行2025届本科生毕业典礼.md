# 河海大学举行2025届本科生毕业典礼

> 来源: https://alumni.hhu.edu.cn/2025/0620/c26684a304649/page.htm

6月19日，我校隆重举行2025届本科生毕业典礼。河海大学杰出校友、泸州老窖股份有限公司总经理林锋作为特邀嘉宾出席典礼。校党委书记杨桂山，校长郑金海，校党委副书记、纪委书记孟新，校党委副书记、副校长陆国宾，副校长沈扬，各学院及有关部门负责人、教师代表和5190名本科毕业生参加了典礼。沈扬主持典礼。

校党委书记 杨桂山 为学生 颁发证书

毕业典礼上，郑金海发表题为《自信自强致高致远》的演讲。他向圆满完成学业的各位毕业生同学表示最热烈的祝贺，向为毕业生们成长成才倾注无数心血的父母家人和教职员工们致以崇高的敬意和衷心的感谢。他说，四年前，同学们怀揣着对未来的憧憬、对知识的渴望、对梦想的执着，意气风发地踏入了河海大学的校门。四年来，同学们与时代同频，与祖国共振，与河海共成长，用辛勤的汗水浇灌梦想，以不懈的拼搏铸就荣光，“中国灵魂、全球视野、河海特质”已深深融入同学们的血脉，成为最鲜明的精神标识。

郑金海发表题为《自信自强 致高致远》的演讲

郑金海还对广大毕业生提出三点希望：厚植家国情怀，做心怀“国之大者”的追光者。希望同学们赓续“红帽子大学”的红色基因，将“天下有溺犹己溺”的济世情怀融入血脉，让青春为中国式现代化建设闪耀光芒，在教育强国建设中勇担“河海使命”，彰显“河海作为”。淬炼格物之光，做穿透迷雾的深潜者。身处伟大变革时代，希望同学们锤炼“深潜”之功—在信息洪流中保持独立思考，在利益诱惑前坚守价值底线，在时代浪潮中精准定位自我坐标，穿透浮华锚定本质，久久为功，彰显河海智慧。永葆热爱热忱，做坚守理想的奋斗者。希望大家守护好对事业的热爱、对生活的热忱、对亲友的温暖。让心中的理想在热爱中找到人生的真谛与价值。他表示，母校永远是每一位毕业生的精神原乡和情感港湾，衷心祝愿大家鹏程万里，前程似锦，一生幸福。

典礼嘉宾、1991级人资及行政管理专业杰出校友林锋向毕业生们表示最诚挚的祝贺，向母校和老师们致以最崇高的敬意。他结合自身学习和工作生活经历与毕业生们分享四点感悟：深植报国之志、点燃青春理想，践行河海精神、勇于扎根基层，坚持终身学习、拥抱未知挑战，永远向阳而生、追求价值人生，在求知问道中锤炼品格，在创新创造中开拓视野，以凌云之志谱写属于青年一代的壮美诗篇。

典礼嘉宾、1991级人资及行政管理专业杰出校友林锋致辞

教师代表、电气与动力工程学院院长张犁教授向毕业生们顺利完成学业表示祝贺，对毕业生为母校付出的努力和作出的贡献表示感谢，并寄语毕业生以信仰为舵，以创新为帆，在各自领域成就栋梁之材，让“大哉河海奔前程，毋负邦人期”的铮铮诺言响彻云霄，共同绘就“天下安澜，向海图强”的河海新篇。

教师代表、 电气与动力工程 学院 院长 张犁 教授 寄语

毕业生代表水电学院水利水电工程专业曹诗佳同学和人工智能与自动化学院人工智能专业孙昊同学表达了对母校的依依惜别之情和对老师们的感激之情，表示在人生新征程上将始终秉承河海校训精神，将河海精神注入时代的浪潮，到祖国和人民最需要的地方发光发热，为实现中华民族伟大复兴的中国梦贡献智慧与力量。

毕业典礼上，郑金海向林锋校友颁发了河海大学荣誉证书。

校党委副书记、副校长陆国宾，副校长沈扬宣读了优秀毕业生名单。校领导和嘉宾向毕业生代表颁发毕业证书、学士学位证书，向优秀毕业生代表颁发荣誉证书。
