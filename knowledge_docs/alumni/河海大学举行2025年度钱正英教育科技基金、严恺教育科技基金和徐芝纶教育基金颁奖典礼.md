# 河海大学举行2025年度钱正英教育科技基金、严恺教育科技基金和徐芝纶教育基金颁奖典礼

> 来源: https://hhuedf.hhu.edu.cn/2025/1219/c26739a315216/page.htm

10月25日，我校举行2025年度钱正英教育科技基金、严恺教育科技基金和徐芝纶教育基金颁奖典礼，校长郑金海出席颁奖典礼。典礼由副校长陈达主持。

郑金海强调，钱正英、严恺、徐芝纶三位院士是水利工程与教育领域的精神丰碑，他们的家国情怀与科学精神，早已融入学校的办学基因，共同构筑了河海人的精神底色。以三位院士命名的三大基金已成为激励师生见贤思齐的重要标杆，在全国教育界、水利与海洋工程界、力学界享有崇高声誉，产生了积极深远的影响。在建校110周年之际举行颁奖典礼，这既是对学校百年办学传统的传承，也是对三位院士精神的深切致敬。他勉励获奖者以三位院士为楷模，承泽大师荣光、赓续奋斗精神，把个人理想追求融入强国建设、民族复兴伟大征程，共同书写无愧于时代、无愧于人民与历史的河海新篇章。

“钱正英水利终身成就奖”获得者、中国工程院院士陆佑楣通过视频发表获奖感言。他表达了对学校建校110周年的祝福，并表示河海大学不仅赋予他专业知识，更培育了他“我饮河海一滴水，我献祖国一生情”的家国情怀，这份荣誉属于所有奋战在水利事业一线的建设者们。

“严恺科技奖”获得者、国家卓越工程师，全国水运工程建造大师，中交天津航道局有限公司首席顾问顾明分享了在重大水运工程一线的技术攻坚经历，表示将牢记科技报国的初心使命，潜心研究、追求卓越，努力将科研成果转化为推动国家富强和民族复兴的科技动力。

“徐芝纶奖学金”获得者、环境学院2022级本科生李章钰表示将扎根水利事业，勇担时代使命，以实际行动传承院士们的科学精神与家国情怀。

此次钱正英教育科技基金、严恺教育科技基金和徐芝纶教育基金共评出79名获奖者，覆盖清华大学、大连理工大学等全国11所水利类重点高校优秀学生，以及中国水利水电科学研究院、中国长江三峡集团上海勘测设计院有限公司等多家单位的资深专家、学者和科技工作者。

江河奔流，滋养万物而不息星火相传，照耀前路而弥新在新时代的浪潮中河海人将接力续写“治水兴邦”的辉煌传奇！
