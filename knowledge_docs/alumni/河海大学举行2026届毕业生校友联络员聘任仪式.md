# 河海大学举行2026届毕业生校友联络员聘任仪式

> 来源: https://alumni.hhu.edu.cn/2026/0612/c26684a327881/page.htm

6月10日，河海大学举行2026届毕业生校友联络员聘任仪式，副校长陈达出席活动。学校相关职能部门负责人，学院校友工作负责人、联络员以及91名2026届毕业生校友联络员参加活动。

陈达对圆满完成学业、即将奔赴新程、主动担当奉献的毕业生校友联络员表示祝贺。他介绍了学校在校友组织联络、校友活动开展、校友发展服务等方面情况，希望毕业生校友联络员心怀理想、笃行实干，善用校友平台资源锤炼本领、增长才干；履职尽责、当好纽带，用心凝聚身边校友、用情传递母校牵挂；坚守本心、行稳致远，在磨砺中不断成长、在坚守中不断突破，书写不负青春、不负时代的精彩人生。

国内合作发展处处长、校友会办公室主任朱乔宣读了《关于聘任2026届毕业生校友联络员的通知》。

农业科学与工程学院党委书记刘兴平、南京校友会执行秘书长谢书磊、2022届环境科学专业校友徐中钰分别代表学院校友工作负责人、校友分会负责人、校友做交流发 言。

2026届毕业博士生罗煜宁、2026届本科毕业生冯泽渊代表受聘校友联络员发言，分享在校成长心得，畅谈履职设想与未来规划，表示将全力做好校友联络服务工作。

陈达、朱乔为2026届毕业生校友联络员代表颁发聘书。

校友联络员聘任是校友会加强校友联络服务、强化校友组织网络的重要举措。自2018年以来，校友会已聘任校友联络员近600人，他们在促进校友交流合作、提升母校声誉和校友凝聚力，助力“校友-母校发展共同体”建设等方面发挥了重要作用。
