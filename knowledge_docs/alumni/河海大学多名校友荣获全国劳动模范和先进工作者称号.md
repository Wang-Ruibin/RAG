# 河海大学多名校友荣获全国劳动模范和先进工作者称号

> 来源: https://alumni.hhu.edu.cn/2025/0508/c26684a302561/page.htm

4月28日，庆祝中华全国总工会成立100周年暨全国劳动模范和先进工作者表彰大会在人民大会堂隆重举行。中共中央总书记、国家主席、中央军委主席习近平出席大会并发表重要讲话。蔡奇宣读《中共中央、国务院关于表彰全国劳动模范和先进工作者的决定》。据初步统计，我校校友林锋、韦有波、王晓芳、吴春雷荣获“全国劳动模范”称号，校友刘瑞国荣获“全国先进工作者”称号。

林锋，1993届人事劳资及行政管理专业校友，现为泸州老窖股份有限公司党委副书记、总经理，正高级经济师、高级营销师、高级技师。林锋躬耕酒业三十余年，带领泸州老窖荣获“全国脱贫攻坚先进集体”“全国五一劳动奖状”等荣誉，入选“中国上市公司价值百强”等榜单，个人曾获“四川省劳动模范”“中国酒业营销金爵奖年度杰出人物”“四川数智先锋人物”等荣誉。

韦有波，2005届无机非金属材料工程专业校友，现为中铁十八局集团第二工程有限公司党委副书记、总经理。韦有波扎根施工一线二十余年，参与建设多项国家重大工程，领衔的“一拖二式”劳模创新工作室获批国家专利23项，发表核心期刊论文15篇、优秀工法5项，获国家优质工程奖、詹天佑奖等奖项。个人曾获“全国铁路总工会‘火车头奖章’”“全国五一劳动奖章”等荣誉。

王晓芳，2005届无机非金属材料工程专业校友，现为上海电气上重铸锻有限公司大型铸锻件研究所副所长，高级工程师。长期致力于核电、火电等国家重大工程装备用大型锻件材料及制造工艺的研发工作，攻克了“华龙一号”“国和一号”核电机组、600MW快堆、高温气冷堆等关键项目的“卡脖子”难题，显著提升了我国大型锻件极端制造能力。

吴春雷，2006届热能与动力工程专业校友，现为广西投资集团来宾发电有限公司设备管理部主任，高级工程师。吴春雷持续突破技术改造、智能运维等领域难题，打破多项国外技术垄断，曾获“自治区劳动模范”“广西工匠”等荣誉。

刘瑞国，2006届水利工程专业校友，现为辽宁省河库管理服务中心水文情报预报中心正高级工程师。刘瑞国长期从事防汛抗旱水文情报与分析、水文预报与预警、洪水调度与会商、旱情监测与预警、智慧水利等技术工作，曾获“全国水旱灾害防御工作者”“全国水利系统先进工作者”等荣誉。

全国劳动模范和先进工作者每五年评选一次，本次评选共有1670人被授予全国劳动模范称号，756人被授予全国先进工作者称号。
