# 河海大学常州校区1983级校友举行毕业四十周年返校活动

> 来源: https://alumni.hhu.edu.cn/2026/0618/c26687a328130/page.htm

日前，河海大学常州校区（原水电部常州机械制造职工大学）1983级校友集体返校。常州校区管委会副主任崔锦铭、机电工程学院党委副书记张亿全，朱炳麒、胡德飞、胡钢、上官国华等老教师受邀参加座谈会。

崔锦铭代表常州校区对校友返校表示热烈欢迎，并从办学历史、校区布局、学科专业、师资队伍、科研平台及校地融合等方面，全面介绍了常州校区的发展概况。张亿全与校友们一起回顾了学院四十载发展历程，向校友们对学院的关心表示衷心感谢。1983级老班长、党支部书记杨棨深情表示，校友们虽年事已高，却依旧情系母校、心怀眷恋，专程返回母校。校友们还特意带回了珍藏的毕业证书、成绩单等老物件，睹物思情、追忆往昔。几位老教师与校友们一起回忆在校期间的学习生活点滴，共叙师生情谊，由衷为学校的稳步发展感到欣慰与自豪。现场交流气氛轻松温馨，真挚感人。

座谈会后，校友们参观了校园，实地感受崭新的校园、优良的环境和浓厚的学风。

此次校友返校，搭建起母校与校友情感联结的暖心桥梁，彰显了校友们饮水思源、不忘师恩的深厚情怀。
