# 河海大学教育发展基金会召开第四届理事会第六次会议

> 来源: https://hhuedf.hhu.edu.cn/2026/0421/c26739a324986/page.htm

12月29日，河海大学教育发展基金会召开第四届理事会第六次会议。会议由基金会理事长郭继超主持，基金会理事、监事出席会议。

副理事长兼秘书长赵冰作《河海大学教育发展基金会理事会2025年工作报告》。报告回顾了2025年基金会在资源拓展、项目管理、品牌建设等方面的工作成果以及未来工作规划。与会理事听取并审议了工作报告、捐赠资金使用、基金保值增值等议题。理事会充分肯定了基金会2025年工作成效，认为基金会在筹资创新、项目管理、内部治理等方面的探索实践，高度契合了国家关于社会组织助力教育事业高质量发展的要求，有效汇聚了社会力量赋能学校发展。与会人员围绕资源拓展、公益项目品牌建设、风险防控机制等方面开展深入研讨。郭继超理事长向全体理事、监事的履职尽责和大力支持表示感谢。他强调，2026年是“十五五”规划开局之年，也是基金会乘势而上、深化发展的关键之年。结合学校“十五五”规划编制要求，他对基金会2026年工作提出四点希望：一是坚持党的全面领导，扎实推进基金会等级评估工作，持续提升内部治理专业化水平；二是锚定学校高质量发展战略，深化资源拓展与协同合作，编制捐赠项目指南，推动社会资源与学校发展需求的有效链接；三是强化精细管理与服务升级，完善捐赠荣誉体系与回访机制，提升捐赠黏性与公益文化浸润力；四是凝聚品牌力量，讲好河海公益故事，提升基金会影响力。未来基金会将进一步提高政治站位，以凝心聚力、务实笃行的工作作风，充分发挥社会组织服务教育发展的独特优势，为学校教育事业高质量发展、为加快推进教育强国建设贡献更大公益力量。
