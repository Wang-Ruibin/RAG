# 河海大学校友会第五届会员代表大会暨校友会理事会换届会议召开

> 来源: https://alumni.hhu.edu.cn/2025/0623/c26684a304796/page.htm

6月22日，河海大学校友会第五届会员代表大会暨校友会理事会换届会议成功召开。河海大学校长郑金海，河海大 学原校长、第四届理事会会长徐辉，河海大学原党委副书记、第四届理事会副会长郭继超，第四届理事会成员代表，第五届理事会理事、监事候选人以及校友代表等参加会议。

徐辉向长期以来关心支持学校校友工作的各地校友、第四届理事会全体理事、监事表示感谢。他表示，校友是学校最宝贵的财富，学校与校友携手共赢是最根本的战略选择，校友会是校友工作最坚强的组织保障。希望新一届理事会肩负起团结校友、服务校友、促进学校发展的重任，使校友工作成为学校高质量发展的强劲动能。

会议听取、审议并通过了郭继超所作的校友会第四届理事会工作报告。他介绍了校友会近年来在校友组织体系建设、校友联络和服务、校友活动开展及校友发展支持等方面取得的成绩和工作经验，并建议新一届理事会紧紧围绕学校中心工作，进一步完善校友工作体系，持续提升校友服务质效，实现校友与母校事业发展同频共振。

会议审议并通过了第四届理事会财务报告和监事工作报告、新修订的《河海大学校友会章程》，选举产生了第五届理事会理事、监事。

随后，第五届理事会召开第一次会议，选举产生校友会新一届负责人。郑金海当选为会长，赵冰当选为秘书长。

郑金海代表学校、校友会向关心支持学校事业发展的广大校友、各校友组织致以诚挚的慰问，向为校友会发展倾注心血、作出重要贡献的第四届理事会致以衷心的感谢。他指出，校友会是学校事业发展的“战略支撑点”，校友会与校友是学校历史新进程的“命运共同体”，理事会及理事是学校、校友事业内涵发展的“坚定推进者”。他表示，新一届理事会将恪守章程，继续秉持“服务校友、服务学校、服务社会”的宗旨，用真情款待校友，广泛倾听校友的意见和建议，为校友创造更多成长、发展机会；用温情善待学生，充分发挥校友榜样力量，引导学生成长成才。他强调，校友的成长成就是学校最大的荣光，要以更暖的情怀凝聚校友，加快推进学校校友发展共同体建设，实现“母校助力校友发展，校友反哺母校建设”的良性循环与内涵共生。

会后，与会人员合影留念。会议由国内合作发展处处长、校友会办公室主任、教育基金会办公室主任赵冰主持。
