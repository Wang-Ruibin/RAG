# 河海大学校友会负责人任职前公示

> 来源: https://alumni.hhu.edu.cn/2025/0624/c26691a304843/page.htm

根据《关于开展全省性社会组织负责人任职前公示工作的通知》（苏社管联办〔2018〕1号）关于落实“加强对社会组织负责人任职监督，充分发挥社会组织服务国家、服务社会、服务群众、服务行业的作用”的工作要求，参照社会组织相关法规和文件规定，现对下列同志拟担任河海大学校友会负责人情况予以任职前公示：

郑金海，男，1972年2月出生，汉族，中共党员，博士研究生学历，现任河海大学校长、党委副书记，拟任河海大学校友会会长。

赵冰，女，1978年12月出生，汉族，中共党员，硕士研究生学历，现任河海大学国内合作发展处处长、校友会办公室主任、教育基金会办公室主任，拟任河海大学校友会秘书长。

公示时间为：2025年6月22日至2025年6月28日。对公示对象如有异议，请于公示期间与河海大学校友会联系。

联系地址：江苏省南京市西康路1号河海大学河海馆906室，联系电话025-83787397。
