# 河海大学能动校友会理事会换届暨动力学科发展论坛

> 来源: https://alumni.hhu.edu.cn/2025/1107/c26687a309814/page.htm

10月25日，河海大学能动校友会理事会换届暨动力学科发展论坛顺利召开。校友会办公室副主任、教育基金会办公室副主任李维，电气学院领导班子成员、校友代表、师生代表等参加。

在换届大会阶段，郑源教授首先对与会嘉宾表示热烈欢迎。李维宣读了学校《关于同意水动校友会更名的批复》《关于同意河海大学能动校友会换届的批复》，同意将“河海大学水动校友会”正式更名为 “河海大学能动校友会”，以进一步顺应水动、热动及能动专业的学科发展历史及未来建设需求。

会议审议并表决通过了《河海大学能动校友会章程（修订）》，明确了校友会的宗旨、业务范围、会员权利与义务，以及组织机构的运行机制。经民主选举，大会产生了新一届理事会及领导机构，由邵宜祥担任会长，王丰担任秘书长。

邵宜祥对上一届理事会的辛勤付出表示衷心感谢，并表示新一届理事会将紧密围绕校友会宗旨，积极搭建多元交流平台，深化校友与母校在人才培养、科研合作、资源共享等方面的联系，切实发挥校友会的桥梁与纽带作用。王丰阐释了开展秘书处工作的基本思路，表示将着力提升秘书处的服务效能，畅通校友联络渠道，组织开展形式多样的品牌活动。

会议同时举行了捐赠签约仪式，江苏智格高科技有限公司、南京丰道电力科技有限公司、绿迹（南京）能源科技有限公司等捐资注入“电力基金”“动力基金”，并捐赠仪器设备，支持学校、学院事业发展。
