# 河海教师荣获“宝钢优秀教师特等奖提名奖”！

> 来源: https://hhuedf.hhu.edu.cn/2025/1211/c26739a314759/page.htm

11月15日2025年度宝钢教育奖在上海揭晓我校环境学院华祖林教授荣获“宝钢优秀教师特等奖提名奖”华祖林，环境学院二级教授，博士生导师，全国高校黄大年式教师团队“水环境模拟与生态修复教师团队”带头人，享受国务院政府特殊津贴专家。主持国家一流本科课程、江苏省高等学校重点教材、江苏省优秀研究生课程与教材、教育部主题案例等9项，获2023年、2024年江苏省优秀博士学位论文指导教师，主持获得江苏省优秀毕业设计团队奖2项。主持国家自然科学基金重点项目、国家自然科学基金联合基金重点项目、以及重大工程项目等60余项，成果获得国家科学技术进步奖二等奖，教育部科学技术进步奖一等奖等省部级以上奖励近十项。

此外，学校还有10名师生获奖。其中，水文水资源学院李琼芳教授、外国语学院张海榕教授、港口海岸及近海工程学院陶爱峰教授荣获“宝钢优秀教师奖”。

土木与交通学院博士研究生王涛、本科生杨焯，港口海岸与近海工程学院博士研究生曹占雪，环境学院博士研究生柳冲冲，电气与动力工程学院本科生胡梦锴，地球科学与工程学院本科生周子煜，商学院本科生李淑琪荣获“宝钢优秀学生奖”。

宝钢教育基金会于2005年5月获民政部批准成立，宝钢教育奖被公认为我国高等院校中分布面广、奖励总额多、荣誉度高、深受师生看重的全国性奖项之一。今年来自全国115所高等院校的282名教师、509名学生荣获宝钢教育奖。其中，有10名教师获得“优秀教师特等奖”，5名教师获得“优秀教师特等奖提名奖”，25名学生获得“优秀学生特等奖”。
