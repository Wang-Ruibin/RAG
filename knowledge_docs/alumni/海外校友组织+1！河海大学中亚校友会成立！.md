# 海外校友组织+1！河海大学中亚校友会成立！

> 来源: https://alumni.hhu.edu.cn/2025/0815/c26684a306275/page.htm

8月10日，河海大学中亚校友会成立仪式在哈萨克斯坦首都阿斯塔纳举行。校长郑金海出席仪式并致辞。学校相关职能部门、学院负责人，新疆校友会负责人，中亚校友代表等参加活动。

郑金海向在中亚地区工作生活的校友们表示亲切的问候，向中亚校友会的成立表示热烈的祝贺。他介绍了学校近年来在开放办学、学科建设、科研合作等方面的成绩，重点介绍了学校与中亚地区的交流合作情况，希望中亚校友会强化增进校友联系，整合拓展校友资源，推动中亚高校、科研机构及企业等与母校、校友在关键技术联合攻关、科研成果转化等领域的深度合作，实现校友事业发展和母校高质量发展的双向奔赴，为全面推动共建“一带一路”高质量发展贡献河海力量。

会议听取了中亚校友会的筹备情况报告。国内合作发展处处长、校友会办公室主任赵冰宣读《关于同意成立河海大学中亚校友分会的批复》。会议选举产生了河海大学中亚校友会第一届理事会，1998级电力系统及其自动化专业校友塔拉甫·加盘当选为会长，2003级水利水电工程专业校友塔拉甫汗·帕提汗当选为执行会长，2000级水利水电工程专业校友哈尔肯·克里木当选为秘书长。

新当选的中亚校友会会长塔拉甫·加盘表示，将充分发挥好链接校友、母校的桥梁纽带作用，构建“校友—母校—区域发展”三维合作网络，积极助力母校国际化进程和中亚地区繁荣发展。

新疆校友会会长阿力甫江·阿不里米提在交流发言时表示，新疆校友会将与中亚校友会紧密联系，深化校友合作，携手为校友事业和母校发展注入新动能。

成立仪式后,中亚校友会举行了庆祝建校110周年校旗传递活动。
