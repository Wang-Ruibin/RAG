# 港口海岸与近海工程学院1975级校友举办入学50周年返校活动

> 来源: https://alumni.hhu.edu.cn/2025/1111/c26687a309917/page.htm

近日，在河海大学喜迎百十华诞之际，海洋水文75级校友重返母校，深情相聚。河海大学原校长张长宽、港口海岸与近海工程学院院长龚政、党委书记张雪刚、海洋学院党委书记胡忠华出席座谈会，座谈会由李兴法校友主持。

张雪刚首先代表学院对各位校友在学校百十校庆之际入校50周年重返母校表示热烈的欢迎，并对大家一直以来关心支持学院的发展表示衷心的感谢！张雪刚从师资队伍建设、人才培养、学科建设、科学研究和社会服务贡献等介绍了学院的发展情况及近年来取得的丰硕成果。胡忠华则围绕海洋学院独立建院10年来取得的成绩作了详细汇报，展现了学院蓬勃发展的新局面。两位书记诚挚邀请校友们“常回家看看”，继续关心和支持母校发展。

1975级校友孙庭兆深情回顾了昔日同窗共读的岁月，并对大家在母校百十年校庆之时重返校园表示由衷的欣慰。随后，作为当年的老师，张东生亲切寄语在场校友，叮嘱大家在工作与生活中注意身体，祝愿各位健康长寿、阖家幸福，充满关怀与牵挂。

座谈会后，校友们参观了修缮一新的西康路校园，并前往常州校区参观，继续重温青春记忆，畅叙同窗情谊。大家也分享了各自的事业发展与人生体悟，表达了对学校前景的美好祝愿。
