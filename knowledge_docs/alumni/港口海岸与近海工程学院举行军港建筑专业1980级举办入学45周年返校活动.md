# 港口海岸与近海工程学院举行军港建筑专业1980级举办入学45周年返校活动

> 来源: https://alumni.hhu.edu.cn/2025/1111/c26687a309921/page.htm

在庆祝河海大学建校110周年之际，港口海岸与近海工程学院1980级军港建筑专业校友重返母校，共叙情谊，追忆青春岁月。港口海岸与近海工程学院党委书记张雪刚、院长龚政出席校友座谈会，与校友们亲切交流，座谈会由1980级校友林育平主持。

座谈会伊始，校友们共同观看了河海大学最新宣传片，母校厚重的历史底蕴与新时代取得的辉煌成就令在场校友感慨万千，自豪不已。张雪刚代表学院对校友们的归来表示最热烈的欢迎。他表示，校友是学院最宝贵的财富和发展的坚实力量，感谢大家多年来始终心系母院、鼎力支持。龚政向校友们详细汇报了学院近年来在师资队伍建设、学科创新发展、科学研究突破、高素质人才培养等方面取得的丰硕成果，描绘了学院蓬勃发展的崭新蓝图。

畅谈环节，校友们抚今追昔，踊跃发言。大家为母校和学院日新月异的变化感到振奋，纷纷表示无论身在何方，根永远在河海，未来将一如既往地关注和支持母校发展，与母校同心同向，携手共赴新征程。

座谈会后，校友们怀着激动的心情一同参观了当年学习和生活过的港建楼。在熟悉的场景前，往昔的求学时光仿佛历历在目，同窗之情愈发深厚。
