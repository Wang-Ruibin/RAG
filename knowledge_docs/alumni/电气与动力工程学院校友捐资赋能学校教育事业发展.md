# 电气与动力工程学院校友捐资赋能学校教育事业发展

> 来源: https://alumni.hhu.edu.cn/2025/1024/c26690a322640/page.htm

10月20日，电气与动力工程学院举行校友捐赠集中签约仪式。南京软核科技有限公司、南京灿维科技集团有限公司、1999级全体校友为支持学校、学院事业发展，注资“电力基金”并捐赠仪器设备。1996级校友、南京软核科技有限公司总经理马春生，1997级校友、南京灿维科技集团有限公司董事长蒋海军，1999级校友代表宋鹏飞、王蓓蓓、薛高洋、鲍晓玮、鲍进，河海大学国内合作发展处处长、校友会办公室主任、教育基金会办公室主任赵冰，电气院班子成员、教师代表等参加签约仪式。仪式由电气与动力工程学院党委书记吴洪彪主持。

赵冰感谢校友们情系河海，为推动学校、学院发展慷慨解囊，注入强劲发展动力。

张犁代表学院向校友们致以诚挚的感谢。他表示，校友们的善举生动诠释了河海学子“饮水思源”的优良传统，不仅是对学院物质上的支持，更是对学院全体师生精神上的巨大鼓舞。

马春生校友、蒋海军校友、薛高洋校友在发言中深情地回顾了在母校求学的青葱岁月。他们表示，母校的悉心培养是自己事业发展的基石，今日的成就离不开当年老师的教诲和母校的滋养。

赵冰分别与捐赠方代表签订捐赠协议，并颁发捐赠证书。张犁与1999级校友代表共同为会议室冠名揭牌。

学院自启动“电”亮未来，“动”力无限——真情回馈母校行动以来，校友们纷纷响应，形成了学院校友相互支撑、彼此成就，合力谋划长久发展的良好氛围。
