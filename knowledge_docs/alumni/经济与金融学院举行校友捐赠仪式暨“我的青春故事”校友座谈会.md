# 经济与金融学院举行校友捐赠仪式暨“我的青春故事”校友座谈会

> 来源: https://alumni.hhu.edu.cn/2025/1125/c26687a313114/page.htm

近日，经济与金融学院举行校友捐赠仪式暨“我的青春故事”校友座谈会，2003级校友张秋禹，2013级校友田苗、蒋涵，2020级校友何嘉熙，经济与金融学院党委书记李娜、院长庞庆华、教师代表等参加。座谈会由经济与金融学院党委副书记赵光好主持。

庞庆华代表学院向校友们致以诚挚的欢迎，他介绍了学院的建设发展情况，以及在经济金融领域的深厚办学根基、丰硕的人才培养成果。他感谢校友们一直以来对学院的关心和支持，并表示捐赠体现了校友们对母校和学院的深切情谊，为学院人才培养、学科建设注入了宝贵动力，学院将以此为契机进一步优化育人环境、搭建成长平台，实现学院创新创业教育的“新发展”。

张秋禹代表2003级国际经济与贸易专业捐赠3万余元。他分享了毕业后的发展历程与感悟，感恩母校与校友会的支持，并表示大学期间获得的学习能力、思维方式对自身职业发展带来了很大帮助，为其毕业后投身光伏行业筑牢根基。这次反哺母校，希望能够搭建更优质的发展平台，帮助更多师生实现科研成果转化，为推动学校人才培养、成果落地贡献校友力量。

田苗、蒋涵捐赠30万元设立“新和思路发展项目”。蒋涵回顾了他从新疆远赴江苏求学，在国家创业政策与学校支持下在校创业的经历。他表示，学校和老师给予的充分包容是他能够兼顾创业与学业的关键，毕业近10年，他始终心系母校，恰逢学校建校110周年，希望通过捐赠的形式帮助学院搭建校内创业平台，为学生提供创新创业资金扶持，同时在未来进一步持续深化校企合作，搭建学校与新疆的交流通道，助力学校人才培养。

何嘉熙分享了本科和研究生期间的求学经历，并希望今年4月捐赠的桂花树能够以桂寄情，传递“百年树木，百年育人”的精神，作为校友与学校、学院之间的情感链接，在为校园增添绿意的同时见证学院学子的茁壮成长。

李娜为校友们颁发捐赠证书、捐赠鸣谢牌，并向校友们反哺母校的行为致以衷心感谢。她表示，广大校友的关注与支持是学校和学院发展的重要动力，而学院是校友最坚实的后盾。学院将进一步优化拔尖创新人才培养，为行业输送更多高素质复合型人才，在对接国家需求、服务社会中不断推动校友与学院、校友与校友间形成紧密的支撑体系，凝聚校友情谊，共促学院与个人协同发展。

校友们以一笔笔满怀情谊的捐赠，表达了对学院发展的深切支持。未来，经济与金融学院将进一步整合产业资源与实践优势，加强金融科技等实践教学平台建设，提升学科科研与产业需求的对接能力，聚焦经济金融领域创新创业人才培养，为学院学子双创能力培养提供更加丰富的土壤，为深化校企协同、服务国家金融发展战略贡献更多力量。
