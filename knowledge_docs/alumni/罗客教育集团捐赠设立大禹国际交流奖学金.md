# 罗客教育集团捐赠设立大禹国际交流奖学金

> 来源: https://hhuedf.hhu.edu.cn/2025/1219/c26739a315220/page.htm

10月27日，罗客教育集团捐赠仪式在河海大学西康路校区举行。罗客教育集团董事长仓之毅、集团江苏分公司总监陆智明、集团江苏分公司国际化项目主任刘燕婉，河海大学副校长宋翔洲、教育基金会办公室主任赵冰、大禹学院党委书记李英、院长李国芳参加捐赠仪式。仪式由大禹学院党委书记李英主持。

宋翔洲代表学校对罗客教育集团的捐赠表示诚挚的感谢。他介绍了河海大学在学科建设、师资力量、人才培养、国际交流合作等方面的发展情况，以及学校建校110周年系列纪念活动。他指出，罗客教育集团通过赋能高校国际化彰显企业责任，与河海大学“中国灵魂、全球视野、河海特质”的人才培养核心理念高度契合，此次捐赠既是对学校国际化建设的坚定助力，更是对河海学子国际视野培养的深切期许。希望双方今后以此次捐赠为契机，深化合作、共谋发展，进一步探索校企协同育人新模式，共同为国家教育事业发展与区域经济社会进步，培养更多具有全球竞争力的人才。

仓之毅介绍了集团的发展历程及其在国际教育领域的丰富经验。他表示，罗客教育集团始终坚持以助推高校国际化为使命，致力于为学生提供优质的海外学习资源。目前，罗客教育集团已为国内60所高校提供资金捐赠，广泛对接国内超过250所高校，与国（境）外超过60所高层次大学、国际组织和世界500强企业建立了紧密联系。他期望此次与河海大学携手，共同助力学校国际化人才培养与战略发展。

李国芳对罗客教育集团捐资设立大禹国际交流奖学金，支持大禹学院拔尖创新人才培养的善举表示衷心的感谢，并介绍了学院在学生国际化培养方面的举措和成效。她表示此次捐赠为学院扩大学生国际交流规模、拓宽学生国际视野提供了宝贵的支持。

最后，双方还围绕学生国际化培养、国际交流合作等方面进行了交流。
