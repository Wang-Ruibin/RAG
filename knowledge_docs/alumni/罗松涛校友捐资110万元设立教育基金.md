# 罗松涛校友捐资110万元设立教育基金

> 来源: https://alumni.hhu.edu.cn/2025/1014/c26690a322643/page.htm

罗松涛，1978年7月出生，江苏宿迁人，港航一级注册建造师、高级工程师，2000年毕业于河海大学港口与航道工程专业，毕业后任职于苏州市航道管理处，2004年至2009任苏州大通工程建设有限公司工程部经理，现任苏州港航建设工程有限公司总经理。担任政协苏州市姑苏区（园区组）第三届委员会委员，致公党苏州工业园区五支部主委、省社会服务委员会委员、市参政议政委员会委员。主持过多项省市重点工程的实施。曾获2019年度参政议政工作先进个人、2021致公党江苏省委脱贫攻坚先进个人、2022年度优秀人民建议二等奖、2023年度园区履行社会责任先进个人等荣誉。

10月10日，“苏州港航建设教育基金”捐赠签约仪式在河海大学西康路校区举行。河海大学副校长陈达，港口及航道工程专业1996级校友、苏州港航建设工程有限公司总经理、苏州校友会常务副会长罗松涛，中交上海航道局有限公司江苏交通建设工程分公司党委书记张骏，江苏壹路技术咨询有限公司总经理吴梓愚，苏州校友会执行会长雷秋生，学校相关职能部门、学院负责人出席。仪式由港口海岸与近海工程学院党委书记张雪刚主持。

陈达代表学校表达对罗松涛校友及苏州港航建设工程有限公司的衷心感谢。他指出，校友是学校发展的重要力量，罗松涛校友的慷慨捐赠体现了河海人回馈母校、支持教育的优良传统，展现了企业的社会责任与担当。他强调，学校将继续支持校友工作，推动校企合作走深走实，共同助力人才培养与学科建设。

罗松涛深情回顾了在河海大学的求学经历，对母校的悉心培养表达诚挚谢意。他表示，作为港口海岸与近海工程学院的毕业生，始终心系母院发展，此次以公司名义捐赠110万元设立教育基金，旨在激励优秀学子奋发向上，支持学院各项事业进步。他祝愿母校蒸蒸日上，再创辉煌。

仪式上，双方代表签署捐赠协议，陈达向罗松涛颁发了捐赠证书及鸣谢牌。

雷秋生代表苏州校友会发言，表示苏州校友会将持续凝聚校友力量，支持母校建设与发展。

港口海岸与近海工程学院院长龚政汇报了学院近年来的发展情况，表示“苏州港 航建设教育基金”的设立，将为学院发展注入新的动力。学院将以此次捐赠为契机，进一步凝聚广大校友资源，深化校企合作，共同谱写学院高质量发展新篇章。

座谈交流环节，双方围绕人才培养、科技创新、校企合作等内容展开了深入探讨。
