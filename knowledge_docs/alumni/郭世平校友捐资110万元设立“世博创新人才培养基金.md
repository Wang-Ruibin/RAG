# 郭世平校友捐资110万元设立“世博创新人才培养基金

> 来源: https://alumni.hhu.edu.cn/2025/1004/c26690a322644/page.htm

郭世平，1965年12月出生，江苏人，高级工程师。1985级河海大学水文地质与工程地质专业，曾任职于化工部矿山设计研究院、化工高等专科学校、化工部十三化建集团、盐业设计院工作，现任江苏世博设计研究院有限公司院长，省鉴定加固委员会会员、理事。获得地质勘探用取土组件发明专利，作为项目负责人完成市产学研联合研究项目“快速环保半刚性基层在连云港地区创新应用”；获得2017第四届紫金奖文化创意设计大赛优秀奖。‌

9月24日，“世博创新人才基金”签约仪式在河海大学江宁校区举行。江苏世博设计研究院有限公司院长郭世平，常务副院长刘伟，院长助理林昱；河海大学副校长邓成，国内合作发展处处长、河海大学教育发展基金会副理事长兼秘书长赵冰，地球科学与工程学院党委书记马孟珂，院长刘瑾，副院长赵燕容，副院长肖儒雅，党委副书记、副院长张玲彬 ，部分教师代表、学生代表参加签约仪式。

邓成对郭世平表示热烈欢迎和衷心感谢，并介绍了学校建设发展情况。他指出，郭世平校友的慷慨捐赠，既是对学校建校110周年的珍贵献礼，也体现了对母校发展的深切关怀与鼎力支持。作为水文地质与工程地质专业的优秀毕业生，郭世平校友始终心系母校，其个人成长是河海育人成果的生动写照，其善举更彰显了河海人“饮水思源、回馈母校”的深厚情怀。

郭世平深情回顾了在河海大学求学和成长经历，对母校的悉心培养表达诚挚谢意。他表示，此次设立“世博创新人才培养基金”，旨在支持河海大学地球科学与工程学院学科发展、青年拔尖人才培养、创新创业教育等事业发展工作。作为河海校友，他始终感念母校培育之恩，希望借此基金激励更多学子奋发成才，为学院发展注入全新活力。

刘瑾表示，“世博创新人才培养基金”的设立，不仅承载着郭世平校友的深情厚谊，更彰显了他对科技创新和人才培养的高度重视。未来，学院将依托该基金，以需求导向、前沿导向、成才导向为牵引，努力将学院建设成为具有鲜明特色的卓越人才培养高地和科技创新策源地。

会上，邓成向郭世平颁发捐赠证书及鸣谢牌。

随后，双方共同启动“世博创新人才培养基金”。

仪式由地球科学与工程学院党委书记马孟珂主持。
