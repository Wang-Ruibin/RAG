# 马克思主义学院举办“大中小学思政课一体化建设”校友论坛

> 来源: https://alumni.hhu.edu.cn/2026/0618/c26687a328128/page.htm

6月13日，马克思主义学院“大中小学思政课一体化建设”校友论坛圆满举办。学院领导、各地校友代表及全院师生代表欢聚一堂，回望学院十五载办学峥嵘岁月，共话校友温情，共探大中小学思政课一体化协同育人新路径。

开幕式

学院院长戴锐致开幕词，全面回顾了学院十五载发展历程，凝练学科建设、人才培养、教学科研等各项核心办学成果，表示未来学院将坚守为党育人、为国育才初心使命，奋力谱写学院高质量发展新篇章。

主题研讨

主题研讨环节，六位校友结合不同岗位实践，从多元维度分享了大中小学思政课一体化建设的经验与思考。

江苏警官学院马克思主义学院院长、2002级思想政治教育专业校友程婧，结合公安院校办学特色，围绕行业育人资源、本土红色文化资源整合路径，分享系统化忠诚教育体系的搭建经验与特色育人成果 。

东南大学马克思主义学院副教授、2014级马克思主义基本原理专业校友郝娜，结合一线课堂教学与调研成果，深度剖析大中小学思政课学段内容衔接的现实痛点，从顶层设计、教学方法、育人目标等层面提出破解方案。

中国矿业大学马克思主义学院副教授、2016级马克思主义基本原理专业校友单宗玥，聚焦思政课一体化评价体系的文化维度，探讨了构建兼具文化内涵与实践效能的一体化评价体系的思路。

连云港市赣榆区教育局教研室正高级教师、 1987级 政治教育专业校友王常亮，分享了区本教研层面开展卓有成效的、以“新质教研态”为抓手的教学改革新探索，讲述了“新质教研态”如何契合、赋能、引领中小学思政一体化。

江苏省涟水中学高级教师、 1989级 政治理论专业校友朱亚军，介绍了基层视角下大中小学一体化建设的现实困境，从跨界演课、思政品牌、同主题共思政的教研机制等方面开展探索，推动让不同学段的思政课真正实现“育人逻辑”的融通。

福建省福州第八中学教师、 2016级 思想政治教育专业校友何超仪，结合福州区域中学教学实践，剖析思政一体化从政策落地到课堂育人的现实困境，并针对性提出优化策略。

圆桌对话

圆桌对话环节，南开大学马克思主义学院博士生导师、2008级思想政治教育专业校友常宴会，浙江大学博士生、2018级思想政治教育专业校友陈炜枫，北京大学博士生、2019级思想政治教育专业校友彭兆龙受邀参与，与在校生进行了面对面交流。

在学院2023级马理论专业学生主持人李凯欣的提问下，三位校友结合自身求学、科研与职业成长经历答疑解惑，现场氛围温馨热烈。校友们追忆了在校求学的珍贵经历，分享了与恩师相伴成长、校园研学论道的暖心故事，也回顾了学院的特色育人模式，表示学院扎实的治学氛围与多元的实践平台，为个人成长成才筑牢了坚实基础。

教师话当年

学院孙其昂、刘爱莲两位资深教授受邀登台，回望学院办学峥嵘岁月，讲述马院优良治学传统，躬身垂范师德风骨。

孙其昂回顾了学院马理论学科从无到有的奋斗历程，重点讲述了硕博学位点申报的关键攻坚过程。他表示，学院十五载跨越式发展，是历届师生、校友多方接续奋斗、同心聚力的成果，鼓励青年师生要坚守初心，沉心学术研究。

刘爱莲系统梳理了学院从早期教研室、社会科学系到独立建院的完整发展脉络，分享了数十载一线思政教学、人才培养的暖心故事与优良班风学风建设成果。结合多年从教感悟，她寄语全体师生秉持谦逊务实、戒骄戒躁的态度，坚守实干底色，脚踏实地锤炼专业本领。

总结

学院党委书记黄世虎作总结讲话，向远道而来、倾情赋能的全体校友致以诚挚谢意，表示本次校友论坛兼具理论高度、实践深度与校友温度，并指出学院将以建院十五周年为新起点，持续搭建校友联动、校地协同、学段互通育人平台，深耕思政课教育教学改革，全力培育新时代栋梁之材。

弦歌十五载，薪火续华章。本次校友论坛立足学院建院十五周年重要节点，广聚校友智慧力量，凝聚多方协同育人共识。论坛既是对学院十五载办学育人成果的全面回望，也是深化思政课教学改革、推进大中小学思政课一体化建设的优质交流载体。展望前路，学院将始终坚守育人本源，持续盘活优质校友资源，不断夯实学段联动协作机制，在育人新征程中行稳致远。
