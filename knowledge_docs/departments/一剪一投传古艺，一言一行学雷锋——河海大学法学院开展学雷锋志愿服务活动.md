# 一剪一投传古艺，一言一行学雷锋——河海大学法学院开展学雷锋志愿服务活动

> 来源: https://law.hhu.edu.cn/2026/0526/c6346a327323/page.htm

今年的3月5日是第63个学雷锋纪念日，河海大学“学雷锋一条街”志愿服务活动于3月8日在西康路校区、金坛校区同步开展。为深入学习贯彻习近平总书记关于学雷锋志愿服务工作的重要精神,同时弘扬中华优秀传统技艺，普及法律知识，河海大学法学院于西康路校区开展“一剪一投传古艺，一言一行学雷锋”学雷锋普法志愿服务活动

本次活动创新将雷锋精神知识、法律常识问答与剪纸、投壶两项传统技艺相结合，打造趣味十足的沉浸式体验。活动现场，参与者在投壶，剪纸和答题中感受雷锋精神和传统文化的魅力；志愿者耐心指导，讲解答疑，用专业知识服务每一位参与者。现场还布置了雷锋气球、主题横幅，宣传海报，浓厚的学雷锋氛围感染着每一位参与者。

此次活动以寓教于乐的形式，打破了学雷锋活动的传统模式，在体验传统技艺的过程中加深 对雷锋精神的理解，又普及了法律知识，更引导大家将雷锋精神内化于心、外化于行，以实际行动践行助人为乐、奉献社会的理念。

未来，河海大学法学院将继续结合专业特色，开展更多形式新颖、内涵丰富的志愿服务活动，让雷锋精神代代相传，让法治理念深入人心，引导广大师生以青春之我、奋斗之我，为社会贡献青春力量。 （供稿：张静雯，审核：石依依、宋伟卫、刘成钢）
