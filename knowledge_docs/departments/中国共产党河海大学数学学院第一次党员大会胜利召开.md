# 中国共产党河海大学数学学院第一次党员大会胜利召开

> 来源: https://math.hhu.edu.cn/2024/0321/c18393a275910/page.htm

3月20日，中国共产党河海大学数学学院第一次党员大会在江宁会议中心隆重召开。学院143名师生党员参加会议，学院民主党派代表列席了会议。大会由学院陈华副院长主持，学校选举工作督导刘丹平书记到会指导。

大会在庄严的国歌声中拉开帷幕。 校督导组专员刘丹平同志 宣读了《关于同意数学学院党委委员和书记候选人预备人选的批复》。学院党委潘静书记向大会作 《凝心聚力谋发展，团结拼搏谱新篇，奋力开启数学学院高质量发展新征程》工作报告， 报告简要回顾并总结了学院成立以来的各项工作和主要成绩，分析了面临的机遇和挑战，确立了学院未来五年的发展思路与奋斗目标，制定了开创学院高质量发展新格局的工作任务。

会议强调，办好数学学院是学校领导对我们的殷切希望，是全体数学人的共同愿景，更是我们的神圣职责。站在新的历史起点上，我们肩负着时代赋予的使命，责任重于泰山，事业任重道远。

会议指出，学院要坚持立德树人根本任务，聚焦科技是第一生产力、人才是第一资源、创新是 第一动力，全面推进有组织科研，大力增强师资队伍建设，深度开展学科交叉融合，持续提升人才培养质量，加强学院优秀文化的传承创新，在更高起点上推进学院事业实现持续高质量快速发展。

大会审议并通过了党员大会选举办法和监票人、计票人名单。总监票人李锐同志主持了选举工作，大会以无记名投票的方式选举产生了第一届党委委员。经与会正式党员无记名投票、差额选举，刘尉、杨永富、陈华、潘静（按姓氏笔画为序）当选新一届中共河海大学数学学院委员会委员。

大会结束后，学院新一届党的委员会召开了第一次全体会议。会议选举产生潘静为党委书记。潘静书记对新一届党委的成立表示祝贺，希望各位党委委员积极发挥新一届党委的政治核心作用，团结奋进，踔厉奋发，共创学院高质量发展新征程，为将河海大学建设成为水利特色、世界一流高水平研究型大学，为强国建设、民族复兴贡献智慧和力量！
