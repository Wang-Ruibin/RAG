# 喜报 | 我院获2023年度“江苏省青年志愿服务行动组织奖”

> 来源: https://math.hhu.edu.cn/2024/0314/c18393a274894/page.htm

日前，江苏共青团公布了2023年度全省青年志愿者行动评选工作结果， 河海大学数学学院青年志愿者协会荣获2023年度“江苏省青年志愿服务行 动组织奖”。

一直以来，协会以“传志愿精神，品科学之美，伴少年成长”为理念，深入开展青少年科普志愿服务活动。立足专业特色，连续8年举办“启明星”科学小实验进社区志愿服务项目。科普志愿服务的足迹遍及十几个省，形成了完善的科普教学体系，共开展课堂500余场人次，服务少年儿童超过4000人次。

同时，协会举办“环保千理，科普同行”——大学生环境保护志愿服务项目，自2020年以来，在学校周边开展常态化开展服务60余次，寒暑期专题支教40余次，累计服务受益人数达到8000余人次，总计新闻报道80余篇，其中国家级15篇，曾作为典范，被“美丽中国·青春行动”全国青少年生态文明中心官方微信公众号报道，获江苏省高校青年志愿服务项目大赛银奖、生态环保创益大赛三等奖、河海大学第二届青年志愿服务项目大赛铜奖等奖项等荣誉。

此后，协会将继续结合专业特色，引导青年志愿者用实际行动奉行“传志愿精神 ，品科学之美，伴少年成长”的志愿服务理念。
