# 学院召开2026年春学期全院教职工大会

> 来源: https://math.hhu.edu.cn/2026/0313/c18393a323564/page.htm

为切实推进2026年度学院各项重点工作部署，进一步强化专任教师的责任意识与使命担当，学院于2026年3月11日下午2:00在励学楼B219组织召开2026年春学期全院教职工大会。鉴于常州校区教职工的实际工作情况，本次会议采取线上与线下同步进行的方式，确保全体教职员工准时参会、全员覆盖。会议由学院党委书记潘静主持 。

杨永富副院长首先就学院 “十五五”发展规划编制情况、2025年度年终考核结果、2026-2028年聘期岗位任务修订方案及2026年度工作要点的起草工作作出系统说明。他指出，学院已依照学校统一部署，完成数学学院“十五五”规划草案的编制并正式提交；新一轮聘期岗位任务修订工作即将启动，将严格遵循学校相关文件要求，科学拟定实施细则，并按规定程序及时发布；年度工作要点目前正处于研究论证与系统梳理阶段，将在广泛征求教师意见的基础上，经学院审议后正式公布实施。

随后，陈华副院长与杨建斌副院长分别就研究生培养质量提升、科研项目申报组织、本科教育教学改革推进以及学生管理服务优化等重点工作，进行了 简短 汇报，明确了本学期相关工作的总体思路、重点任务与具体实施路径。

最后，潘静书记作对新学期工作提出三点明确要求：第一，要牢牢把握 “十五五”开局之年的重大战略机遇，主动应对新一轮“双一流”建设带来的深刻变革与严峻挑战，聚焦学科建设、人才培养与科技创新等核心领域，全面提升学院综合竞争力；第二，要以关键指标为牵引，集中力量攻坚克难，力争实现跨越式发展；第三，必须严守政治纪律与职业规范，持续加强师德师风建设，坚决筑牢廉洁从教底线，着力营造风清气正、严谨治学的育人环境。
