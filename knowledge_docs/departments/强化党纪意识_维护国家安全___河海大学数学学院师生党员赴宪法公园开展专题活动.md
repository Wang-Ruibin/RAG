# 强化党纪意识 维护国家安全 | 河海大学数学学院师生党员赴宪法公园开展专题活动

> 来源: https://math.hhu.edu.cn/2024/0513/c18393a279481/page.htm

5月11日，为贯彻落实习近平新时代中国特色社会主义思想，深入开展党纪学习教育，弘扬宪法精神，我院组织师生党员赴南京宪法公园开展了专题学习活动。活动以“强化党纪意识，维护国家安全”为主题，学院党委书记潘静出席了活动，各支部党员代表和学生骨干参加了本次活动。

南京市宪法公园是南京市新的法治文化地标，是全国首个全景式展示中国共产党带领中国人民“寻宪、制宪、修宪、奉宪”伟大历史进程的宪法主题公园。公园由旗阵广场、宣誓广场、宪法宣传教育展、主题雕塑、共享绿谷等部分组成。全体参加师生，在熊惠康的带领下，宣读了宪法宣誓誓词。

在讲解员的带领下，全体人员参观学习了旗阵广场、宣誓广场和宪法宣传教育展，并通过实物和影像，接受宪法教育，感受宪法的神圣力量，培养和树立法治意识和纪律意识。

党员们在浓厚的宪法宣传氛围中，感受宪法为国为民的安全感。场馆各类设施生动地诠释着一个又一个历史故事，展示了一代又一代中国人对人民幸福生活与国家公正法治的高度重视和身体力行。

通过本次主题活动，让全体参与人员全民学习了在党的领导下，国家法制化建设进程，提高了党纪意识和法治意识，有助于进一步提高教育法制化水平。
