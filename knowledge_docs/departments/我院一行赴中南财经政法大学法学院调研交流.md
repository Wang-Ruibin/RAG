# 我院一行赴中南财经政法大学法学院调研交流

> 来源: https://law.hhu.edu.cn/2026/0518/c6356a327102/page.htm

为进一步加强院际合作，提升我院人才培养与学科建设水平，5月16日-17日，河海大学法学院党委书记刘成钢、院长李栗燕、副书记、副院长宋伟卫等一行赴中南财经政法大学法学院开展调研交流。

调研期间，双方举行了座谈交流会。中南财经政法大学法学院院长候猛，党委副书记、纪委书记兼行政副院长陈金川，经济法学系主任吴京辉，宪法与行政法学系主任谭冰霖，法学实验教学中心主任疏义红，法学实验教学中心实验员方玉珍参与座谈交流。座谈会上，法学院院长李栗燕围绕学院发展沿革、学科布局、师资队伍、人才培养特色及近年来取得的建设成效进行了交流分享。中南财经政法大学法学院候猛院长详细介绍了该院的学科优势、科研平台及教学改革经验。

随后，双方围绕学科建设、人才培养模式创新、学术科研合作等议题开展了深入座谈与合作洽谈。与会人员就推进院际协同育人、联合开展学术活动等方面交换了意见，并达成初步合作共识。双方一致表示，将以此次交流为契机，建立常态化沟通机制，共同探索法学教育高质量发展的新路径。

座谈结束后，我院调研团队参观了中南财经政法大学国家级法学实验教学示范中心。疏义红主任介绍了中心的发展历程、平台功能及实验教学模式，并引导参观了数字化模拟法庭、法律实务实训室等教学设施。

此次调研交流加强了两院之间的联系与互信，为后续深化合作奠定了良好基础。河海大学法学院将积极借鉴兄弟院校的先进经验，进一步推动学院人才培养和学科建设工作迈上新台阶。
