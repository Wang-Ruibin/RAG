# 我院刘磊教授在多智能体系统协同控制研究取得新进展

> 来源: https://math.hhu.edu.cn/2024/0318/c18377a275067/page.htm

简介

“结合自适应动态规划 (ADP) 和强化学习算法，研究了一种基于间歇事件触发策略 ( IET) 的多智能体系统平均一致性问题。设计了一种新的间歇事件触发条件，并建立了相应的分段微分不等式。利用所建立的不等式，得到了几个关于平均一致性的重要准则。在此基础上，继而研究了多智能体系统的最优一致性策略，并导出了纳什均衡意义下的最优间歇事件触发策略和相应的局部 Hamilton–Jacobi–Bellman 方程。最后，给出了最优策略的自适应动态规划及 Actor- c ritic 算法的数值实验。和单一的 I ET 或 A DP 算法相比， I ET-ADP 的融合策略在节省系统能量方面具有明显优势。所设计的 I ET- ADP 算法实现了事件触发机制和 ADP 算法的协同配合效应。此外，间歇事件触发策略的模型构建，对多智能体系统的稳定性研究具有重要作用，该方法还可推广应用于无人系统的智能控制和协同编队任务中。因此，课题组借助树莓派电路板、地面站、飞控等硬件及一系列软件搭建了户外无人系统飞行实验平台 ，并将设计好的控制算法部署到四旋翼无人飞行器的编队任务中，构建了一套从理论分析到数值模拟、软件在环再到实机应用的研究闭环。 ”
