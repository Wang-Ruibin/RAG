# 数学学院举行“ ie∞奖学金 ”捐赠仪式暨2004届数学与应用数学专业校友毕业20周年返校活动

> 来源: https://math.hhu.edu.cn/2024/0714/c18393a286179/page.htm

7月13日，我院举行了“ie∞奖学金”捐赠仪式暨2004届数学与应用数学专业校友毕业20周年返校座谈会。上海高驰进出口有限公司董事李树茂等40多名2004届数学与应用数学专业校友，河海大学国内合作发展处、校友会办公室、教育基金会办公室主任赵冰，数学学院党委书记潘静等相关部门、学院负责人，数学学院师生代表出席了捐赠仪式。仪式由数学学院副院长杨永富主持。

赵冰代表学校热烈欢迎校友的到来，对校友们长期以来给予母校的关心和支持表达了诚挚的感谢。她介绍了学校近年来在学科发展、人才培养、队伍建设、科学研究以及校友工作等方面发展成就和发展愿景。她强调，校友是学校声誉的最佳代言人，是支持学校发展最可信赖的力量。学校高度重视校友工作，着力搭建学校与校友、校友与校友合作交流平台，全方位构建学校与校友发展的共同体，期待校友与母校携手同行，共同发展，实现学校美誉度和校友认同感、自豪感的双向奔赴，并热忱邀请校友们在110周年校庆时，荣归母校。

潘静代表学院感谢李树茂校友的慷慨捐赠，并介绍了学院的发展历程、师资力量、学科建设及学生培养情况。她表示， 学院的建设发展离不开校友们的鼎力支持，“ie∞奖学金”是数学学院成立以来接受的第一笔校友捐赠设立的奖学金，这不仅诠释了校友心系母校、感恩母校的深厚情怀，也为学院的人才培养和长远发展注入了强劲动力。学院将努力为校友赋能、支持校友的发展，希望与校友携手共同谱写学院高质量发展新篇章。

李树茂毕业多年始终心系母校，时刻关心和支持母校发展，他回顾了自己在河海大学求学的美好时光，表达了对母校的深厚感情和回馈社会的强烈愿望。他表示，“ie∞奖学金”中的“i”既是数学中的虚数单位，也代表热爱；e”是数学中一个自然常数、无限不循环小数；“∞”是数学中无穷大的符号。“ie∞奖学金”就是希望能够汇集数学专业校友的点滴爱心，资助和奖励家庭经济困难的数学专业优秀学子，鼓励他们热爱所学专业、勇敢追梦，早日实现心中梦想，为学校和国家发展贡献自己的力量。

仪式上，李树茂与赵冰签署了捐赠协议，杨永富代表数学学院向李树茂校友颁发了捐赠证书。

随后，校友们与当年的任课教师开展了座谈交流，共叙同窗情谊，共谋学院发展。
