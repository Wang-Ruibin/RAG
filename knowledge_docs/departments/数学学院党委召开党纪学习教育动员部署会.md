# 数学学院党委召开党纪学习教育动员部署会

> 来源: https://math.hhu.edu.cn/2024/0429/c18393a279019/page.htm

4月26日，河海大学数学学院党委召开全体党员大会，对开展党纪学习教育进行动员部署。会议由数学学院党委书记潘静主持。

会议集中学习了习近平总书记关于开展党纪学习教育的重要讲话和重要指示精神以及上级有关部署要求，详细解读了学院党委关于开展党纪学习教育的实施方案。会议明确，自2024年4月至7月，在全党开展党纪学习教育是以习近平同志为核心的党中央作出的重大决策部署，是加强党的纪律建设、推动全面从严治党向纵深发展的重要举措。全体党员师生要切实把思想和行动统一到党中央决策部署上来，认真落实校党委安排要求，确保取得实实在在成效。

会议强调，一是提高政治站位，深刻认识党纪学习教育的重大意义。坚持高站位谋划、高标准推进、高质量落实，以强烈的政治责任、饱满的政治热情，扎实做好党纪学习教育各项工作。要将《条例》学习与学习贯彻习近平总书记关于党纪学习教育的重要讲话和重要指示精神贯通起来，把《条例》作为知纪明纪的必修教材和守纪执纪的行为准则。各党支部根据实际情况制定本支部的党纪学习教育计划表，统筹抓好理论学习中心组和读书班的学习，依托“三会一课”、主题党日等，推进以案促学、以训助学，高质量完成好党纪学习教育任务。二是紧扣任务核心，推动党纪学习教育各项任务落到实处。坚持融入日常、抓在经常，要科学把握好学习教育的目标要求和核心要义，确保学习教育组织到位、措施到位、落实到位。坚持“集体研学+个人自学”“权威解读+专题培训”“警示教育+案例剖析”“组织生活+实践锻炼”等多种方式有机组合，创新红色话剧等工作载体，抓好各领域全覆盖，把遵规守纪刻印在心，内化为言行准则，体现在履职尽责全过程。三是确保学习成效，把党纪学习教育与推进提升学科发展工作紧密结合。坚持做实“规定动作”，创新“自选动作”，与巩固拓展主题教育成果衔接起来，和做好学校第十四次党员代表大会筹备及支部建设工作统筹起来，紧密结合学校年度重点工作，做好党建业务同促进，务求党纪学习教育取得实效，教育引导党员干部将学习成效转化为做好深化人才培养改革、加强人才队伍建设、完善有组织科研等重点任务的强大动力。

会后,潘静书记为师生党员们作新修订的《中国共产党纪律处分条例》专题辅导报告。
