# 数学学院召开全院大会暨本科教育教学审核评估工作动员会

> 来源: https://math.hhu.edu.cn/2024/1024/c18393a290448/page.htm

10月23日，数学学院在励学楼B219报告厅召开全院教师大会。学院领导班子、全体教职员工出席会议。会议由学院党委书记潘静主持。

杨永富副院长对学院成功获批数学一级博士点向学院老师们的辛勤付出表达感谢，并围绕学科、师资、国际合作交流等方面对数学一级博士点成功获批后的建设工作做了详细部署。杨院长提出，为更好迎接即将到来的本科教育教学审核评估和学位点建设，学院要继续发挥基础学科特色，加强学位点建设成效；持续在高水平人才队伍建设工作上发力，筑牢学院发展人才之基；推动国际合作办学落地，争取学院发展实现新的突破。

为迎接本科教育教学审核评估，副院长兼副书记杨建斌介绍了学校及学院的相关准备工作，并呼吁全体教师认真对待，积极准备，全面提升人才培养质量。

最后，潘书记结合学院工作，强调教师们要对标对表，积极做好迎评准备工作，做到以评促建、以评促改、以评促管、以评促强，齐心协力做好本科教育教学审核评估和数学一级博士点建设，做人才培养、学科建设、学院发展的参与者、支持者和实践者。
