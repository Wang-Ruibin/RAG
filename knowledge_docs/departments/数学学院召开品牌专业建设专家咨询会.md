# 数学学院召开品牌专业建设专家咨询会

> 来源: https://math.hhu.edu.cn/2025/0428/c18393a302075/page.htm

为更好推进江苏高校品牌专业建设，加强专业内涵与特色，4月27日，数学学院召开品牌专业建设专家咨询会。江苏师范大学原党委副书记、校长周汝光，合肥工业大学国家名师朱士信，西安电子科技大学国家名师刘三阳，上海财经大学数学学院院长、教指委委员徐定华，国防科技大学全国模范教师、教指委委员李建平等五位校外专家出席会议。会议由副院长杨永富主持，院长助理朱露和刘尉、信息与计算科学系主任刘向阳、本科教学秘书范舒文参加会议。

专家组充分肯定了数学学院品牌专业建设工作在专业内涵、特色与成效等方面的进展，并对国家一流专业、省品牌专业建设等方面提出三点建议：一是把握国家对基础学科特别是数学的重视机遇，增加对数学学科的投入，促进学科快速发展；二是进一步加强数学学科与学校特色优势学科专业的深度融合，以融合促增长；三是加大高层次人才引进和培养力度，提升学科整体实力。

助理刘尉简要介绍了数学学院江苏高校品牌专业建设情况，包括专业概况、代表性成果、建设目标与思路、建设任务与预期成果、保障措施等。

“数学与应用数学”专业为国家级一流专业建设点，依托学校“双一流”学科，在加强数学专业人才培养的同时，大力开展支撑学校优势学科的交叉创新型人才培养，凸显了“数学JIA”的专业特色。

本次品牌专业建设专家咨询会的圆满召开有助于进一步剖析专业建设的现状和问题，明确了未来发展方向。数学学院将充分吸收咨询会上的专家意见和建议，进一步深化专业建设和学科建设，不断推动数学学科和专业高质量发展。
