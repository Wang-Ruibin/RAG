# 数学学院召开第一议题学习暨学院党委中心组（扩大）学习会

> 来源: https://math.hhu.edu.cn/2024/0927/c18393a289379/page.htm

9月23日上午，数学学院召开第一议题学习暨学院党委中心组（扩大）学习会，学院党委书记潘静主持，学院党委委员、师生党支部书记、办公室党支部全体党员参加会议。

潘静书记在会上传达了党的二十届三中全会精神，她强调党的二十届三中全会，是全面推进强国建设、民族复兴伟业的关键时期召开的一次具有全局性、历史性意义的重要会议，会议审议通过的《中共中央关于进一步全面深化改革、推进中国式现代化的决定》开启了高质量发展的时代新篇。她指出，学院领导班子、党支部书记、系中心负责人要在学习宣传贯彻全会精神上走在前、作表率，将学习贯彻全会精神与学习习近平总书记关于教育的重要论述贯通起来，与学习贯彻习近平总书记关于加快发展新质生产力扎实推进高质量发展的重要论述和关于科技创新和基础研究重要讲话精神联系起来，与巩固拓展主题教育成果、慎终如始抓实抓好党纪学习教育结合起来，加快构建高质量党建工作体系，从思想、组织上引领学院事业高质量发展，进一步带动教育科技人才一体化推进的新局面，以全局观念和系统思维谋划数学学科改革发展新思路新举措，从弘扬教育家精神、培育时代新人、构建科研新范式、聚焦人才引培、夯实文化根基、深化合作交流、提高管理效能等七个方面，持续深化各项改革任务落实，助推学院高质量发展。

会议学习了全国教育大会精神。潘静书记强调，河海大学作为中国水利高等教育的开创者和引领者，我们每一个河海人要深刻认识教育强国的 “六大特质”“五个重大关系”“五方面战略任务和重大举措”，紧紧围绕立德树人根本任务，结合学校第十四次党代会相关部署要求，以钉钉子的精神抓好下半年重点工作的贯彻落实和谋划好今后的工作。

数学学院党委坚决落实 “第一议题”制度，将学习习近平新时代中国特色社会主义思想、党的二十大精神等作为重点学习任务，结合推动学院事业发展的实际工作进行研学，使学习教育常态化、规范化、实效化。
