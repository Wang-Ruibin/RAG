# 杨桂山书记来我院调研指导工作

> 来源: https://math.hhu.edu.cn/2026/0410/c18393a324587/page.htm

4月7日上午，校党委书记杨桂山来我院调研指导工作。新加坡国立大学数学科学研究所所长、‌新加坡国家科学院院士‌沈佐伟出席调研座谈会并给予指导， 学院党委书记潘静、副院长杨永富、副院长陈华及教师党支部书记、系（中心）负责人参加调研座谈会。

会上，潘静书记代表学院对杨书记走访学院开展调研表示热烈欢迎，对沈院士长期关心和支持学院发展表示衷心感谢。杨永富副院长汇报了数学学院 “十五五”发展规划，重点 围绕学科建设、师资队伍、人才培养、科学研究、开放办学与社会服务等方面阐述了现状，分析了学院目前面临的外部与内部的需求和挑战，明确了未来五年的发展思路和落实举措。

杨桂山书记听取汇报后对 学院近年来取得的阶段性成果给予充分肯定。他指出，数学学院正处于关键的转型阶段，要充分抓住机遇，再上一个台阶。学院应明确 “十五五”到底要干什么，短板是否能补齐。最后，他对学院提出了四点要求： 一是优化学科布局 ，要结合数学博士学位点建设方案进行完善和修正，同时要加强学科间的交叉研究，坚定不移建设应用数学交叉研究中心，真正做到 “入主流、强特色”。 二是加强人才队伍建设 ，首先要把进人工作放在学院党政最核心的位置，尤其是成熟青年人才、海外优秀人才的引进；其次学院每年要派出 2-3名青年教师出国访问，加强对年轻人的培养；再次要加强团队建设，尤其要发挥好客座教授培养青年教师的作用；最后学院要推进教师的分类管理，严格区分教学为主型和教学科研型，完善两类教师的考核评价体系。 三是做好人才培养工作 ，设立本硕博贯通培养的拔尖创新班，并在保研、导师指导、出国交流等培养环节给予政策支持；要提高全校数学公共课的教学质量。 四是提高科学研究水平 ，学院要抓好基金申报工作，提高基金的资助率；要做好 “水系统数学建模与智能计算”实验室建设，可依托召开学术年会等形式加强研究与交流。

沈佐伟院士指出，数学学院在当前的转型发展时期，应做好两件事：一是做好人才引进工作，要想好怎么招人，人才是学院发展的关键；二是学科建设要立足学校发展，学院要解放思想、推动学科间的交流与合作，打造具有河海特色的数学品牌。

会上， 学院班子成员和教师代表也结合实际工作和存在问题分别作了汇报。
