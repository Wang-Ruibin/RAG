# 河海大学数学学院与东南大学数学学院联合开展 “树立和践行正确政绩观”主题教育学习研讨

> 来源: https://math.hhu.edu.cn/2026/0520/c18393a327178/page.htm

近日，河海大学数学学院与东南大学数学学院联合开展“树立和践行正确政绩观”主题教育学习研讨活动。东南大学马克思主义学院党委书记宋晓燕教授作专题导学。河海大学数学学院与东南大学数学学院全体院领导、系（中心）负责人及相关工作人员参加了学习研讨。

会上，宋晓燕教授以“牢固树立和践行正确政绩观 护航‘十五五’开局起步”为题，围绕政绩为谁而树、树什么样的政绩、靠什么树政绩、如何评价政绩四个方面，结合高校办学治校实际，作了一场系统深刻、贴合实际的辅导报告。

辅导报告后，河海大学数学学院与东南大学数学学院围绕“以正确政绩观引领指导‘十五五’工作”进行专题交流。双方分别结合学科特点，分享了在学科建设与交叉创新团队建设、师资队伍建设与高层次人才引育、拔尖人才培养与华罗庚班联培等方面的探索与实践。大家一致认为，要以此次学习教育为契机，把学习成果转化为推动工作的实际成效，以实实在在的政绩为学校高质量发展贡献学院力量。

下一步，学院党委将严格落实上级学习教育工作要求，坚持学深悟透、学查改一体、学用结合，以务实的作风、扎实的行动推动学习教育走深走实。
