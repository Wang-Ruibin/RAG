# 河海大学数学学院与马克思主义学院党委中心组开展党纪学习教育联学活动

> 来源: https://math.hhu.edu.cn/2024/0517/c18393a279697/page.htm

5月14日下午，数学学院与马克思主义学院党委中心组围绕党纪学习教育联合开展专题理论学习研讨会，通过领导干部的先学一步、学深一层，引领带动学院党纪学习教育走深走实。校党纪学习教育联络员李娜同志列席会议。

党委中心组联学会上，三位领导干部围绕党纪和师德师风带头导学领学。马克思主义学院党委副书记、副院长李宁以 “新中国成立以来党内纪律教育的实践理路”为题，从理论、历史和现实的维度带大家回顾和总结了新中国成立以来党内纪律教育的内容和特征；指出党的纪律逐步形成并不断健全，纪律建设日益发展并不断深入，为实现党的路线方针政策提供了可靠保证；强调大家以史为鉴，在党的纪律教育的历程中汲取经验启示。马克思主义学院党委书记黄世虎结合师德师风案例谈警示教育，要求严格落实教师的道德标准，注重防范师德师风重大风险点，强调以案为鉴，严于律己，慎独慎微。数学学院党委书记潘静从党的纪律视角解读教师“十项准则”负面清单，提出进一步将师德规范和党纪约束结合，以纪为鉴，强化师德师风建设，严守纪律规矩“底线”。

集中研讨交流环节，党委中心组成员结合两院工作实际，就基础学科管理事务推进中的共性问题、痛点难点问题开展交流、分享经验。通过交流互鉴，进一步深化了对党的纪律理论认识，强化了院际协作与互联互动。

学校党纪学习教育联络员李娜同志对此次联学活动给予了高度评价，肯定学院结合业务开展理论联学的方式。她强调学院在落实党纪学习教育工作过程中要以党纪学习教育推动实际工作，以实际工作深化党纪学习教育 。

在此次党纪联学活动中，领导干部带头示范端正学风、集中精力认真学习、联系实际深入研讨，对党内纪律教育的重要意义、深远考量和主旨要义有了更全面认识，进一步增强了纪律规矩和师德意识、提高了个人党性修养，达到了预期目的。下一步，要持续在学纪知纪上融会贯通、在明纪守纪上笃信笃行，切实把党纪学习教育成果转化为始终对党绝对忠诚的政治本色，转化为推动学院高质量发展的实际行动。
