# 河海大学数学学院举办国家安全教育暨党纪学习教育主题报告

> 来源: https://math.hhu.edu.cn/2024/0511/c18393a279406/page.htm

5月10日，为深入开展党纪学习教育和国家安全教育，夯实保密教育，我院举办了国家安全教育暨党纪学习教育主题报告。邀请到江苏警官学院现代警务研究中心副主任孟卧杰教授担任主讲，由院党委书记潘静主持。

孟卧杰教授在讲座中，结合大学生日常学习生活中的实际情况，深入浅出地讲解了国家安全的重要性，以及违反国家安全法所带来的严重后果。他通过鲜活的案例，让在场的每一位学生深刻认识到，国家安全并非遥不可及的概念，而是与我们每个人的日常生活息息相关。

孟教授分析了总体国家安全观视域下的国家安全形势，全新视角阐述了党纪学习教育，国家安全教育和保密教育之间的联系，尤其围绕国家安全教育通过深入浅出的国家安全知识讲解和生动的案例分析，让我们对新时代国家安全工作有了更进一步的认识，引导师生们增强维护国家安全的敏锐意识和本领能力，争做国家安全的维护者、引领者和践行者。

讲座中，孟卧杰教授还针对大学生常见的网络安全、信息安全等问题进行了详细讲解，提醒大家要增强自我防范意识，避免泄露个人信息，防范网络诈骗等违法行为。同时，孟卧杰教授还就如何识别并抵制境外敌对势力的渗透活动进行了指导，帮助同学们树立正确的价值观和世界观。

来自学院各年级的学生党员骨干和教师党员代表一百多人，参加了本次专题讲座。通过本次活动，帮助学院师生党员提高了国家安全意识，夯实了党纪学习教育，明确了自身在维护国家安全中的责任与担当，也为维护校园安全稳定奠定了坚实的基础。
