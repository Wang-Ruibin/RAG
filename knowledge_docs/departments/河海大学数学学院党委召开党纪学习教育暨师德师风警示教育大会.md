# 河海大学数学学院党委召开党纪学习教育暨师德师风警示教育大会

> 来源: https://math.hhu.edu.cn/2024/0517/c18393a279739/page.htm

为推动党纪学习教育走深走实 ，筑牢师德师风建设阵地，数学学院党委于 5月16日下午召开全院教职工党纪学习教育暨师德师风警示教育大会。会议由学院党委潘静书记主持。

潘书记首先介绍了教师思想政治和师德师风建设的总体要求，随后详细解读了新时代高校教师行为规范，会上重点从党的纪律角度解读 “十项准则”负面清单并以典型案例为反面教材开展了警示教育。

会议指出，当前学校正在开展党纪学习教育，作为基础学院的老师一定要从高质量发展的战略高度统一思想、提高认识，立足教学科研实际，精修业务水平，强化监督保障， 要在更高起点上推进学院事业实现持续高质量快速发展。最后会议强调全体教职工一要坚守初心，落实立德树人根本任务，加强师德师风建设，大力弘扬教育家精神， 做学生为学、为事、为人的大先生；二要坚定信心，提振发展自信，树立学科自信，结合 2024年重点工作，把遵规守纪融入日常工作， 锚定目标加压奋进 ,实干争先再谱新篇；三要坚持同心，全院上下要在“闯关大考”中、在重点任务前心往一处想，劲往一处使，师德师风是评价教师队伍素质的第一标准，广大教职工要把遵规守纪和安全发展印刻在心，守牢底线、红线，坚定不移朝着高质量发展目标迈进。
