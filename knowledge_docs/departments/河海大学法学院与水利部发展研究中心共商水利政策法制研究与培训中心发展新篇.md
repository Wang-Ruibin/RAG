# 河海大学法学院与水利部发展研究中心共商水利政策法制研究与培训中心发展新篇

> 来源: https://law.hhu.edu.cn/2025/1206/c6356a314518/page.htm

11月24日上午，河海大学法学院李栗燕院长、李祎恒副院长、吴志红副院长、李华琪副教授一行前往水利部发展研究中心围绕“水利政策法制研究与培训中心”的建设与发展座谈交流。水利部发展研究中心吴强主任、吴浓娣副主任、韩宁处长、贺骥处长、李淼处长、郎劢贤处长共同出席会议。

座谈会上，李栗燕院长首先代表河海大学法学院向水利部发展研究中心长期以来对政法中心工作的支持表示感谢，并系统回顾政法中心自成立以来的工作成效。她指出，水利政策法制研究与培训中心作为水利部发展研究中心与河海大学共同组建的研究机构，始终聚焦水利政策与法治建设的前沿问题，在理论研究、决策咨询及普法宣传等方面取得了较多成果。李祎恒副院长提出，面向未来，政法中心应进一步强化“政策—法制—培训”三位一体的发展格局，推动涉水法治研究向系统化、专业化迈进。吴志红副院长围绕河海大学法学院学科建设与人才培养成效作了系统介绍，法学院始终以“复合型、创新型法治人才”为培养目标，已初步构建理论与实践相融合的课程体系。

座谈中，水利部发展研究中心各处处长与来访的河海大学专家学者开展了卓有成效的交流沟通，并结合本部门职能，就涉水法治、人才培养、成果共推等具体合作路径提出建设性意见：一是希望深入构建人才培养模式，强化专业人才队伍建设；二是期待借助平台开展更多学术交流活动，推动政策与法制领域的交叉融合；三是强调充分发挥双方平台协同作用，实现资源共享与优势互补；四是从战略层面出发，进一步深化和拓展合作内容，推动双方合作向更高层次、更宽领域迈进。吴浓娣副主任建议，未来应进一步强化政法中心的理论探索与实践结合，凸显其应用价值，增强实务导向。吴强主任在总结讲话中，充分肯定了双方前期共建工作的成效，并对未来合作提出展望。他强调，双方应依托政法中心深化协同，以“五个联合”——联合开展研究、联合申报成果、联合培养人才、联合学科建设、联合组织专业培训——为抓手，共同谋划下一阶段发展，推动政法中心工作再上新台阶。

座谈在热烈、务实的氛围中进行，双方围绕政法中心后续换届安排、工作机制优化、合作内容细化等议题进行了深入交流，并在加强常态化联动、推动中心可持续发展等方面达成广泛共识。本次交流不仅巩固了双方的合作基础，也为政法中心在新时期的创新发展指明了方向。下一步，河海大学法学院与水利部发展研究中心将携手推进政法中心各项工作提质增效，共同为水利法现代化事业提供更强有力的保障和支持。
