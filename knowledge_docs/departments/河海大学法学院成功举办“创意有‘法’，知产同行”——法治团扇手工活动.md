# 河海大学法学院成功举办“创意有‘法’，知产同行”——法治团扇手工活动

> 来源: https://law.hhu.edu.cn/2026/0526/c6346a327321/page.htm

2026 年4月11日，河海大学法学院学生会与河海大学法治宣讲团联合举办“创意有‘法’，知产同行”知识产权普法主题法治团扇手工活动。本次活动以普及知识产权法律知识为核心，将普法教育与手工创作相结合，面向法学院学生开展，旨在提升青年学子知识产权保护意识，营造尊重原创、崇尚创新的校园氛围。

活动围绕知识产权科普与团扇手工创作两大板块有序展开。首先进行世界知识产权日与知识产权核心法律知识讲解，介绍4月26日世界知识产权日的起源与意义，解读《著作权法》《专利法》《商标法》三大核心法律，剖析校园中论文抄袭、设计盗用、软件盗版等知识产权“隐形陷阱”，并讲解固定证据、沟通协商、专业维权的合法维权途径。

接下来，进入团扇手工制作环节，志愿者同学介绍永生花团扇制作工具、构图技巧与粘合要点，引导同学们围绕著作权保护、拒绝盗版、知识产权保护等法治主题进行自由创作，搭配花材、绘画补充并书写法治宣言。

此次活动将法治知识与手工艺术深度融合，打破传统普法的单向输出模式，让知识产权保护理念以更生动、更具参与感的方式深入人心。活动不仅帮助同学们掌握实用的知识产权法律知识，强化学术诚信与原创保护意识，更以创意载体传递法治精神，助力河海大学法学院学子树立尊重知识、保护产权的法治观念，为培育创新型法治人才奠定坚实基础 。（供稿： 赵永清 ，审核：石依依、宋伟卫、刘成钢）
