# 河海大学法学院成功举办第十届江苏省大学生知识产权知识竞赛校内初赛

> 来源: https://law.hhu.edu.cn/2026/0526/c6346a327322/page.htm

2026 年4月26日，河海大学法学院在江宁校区、常州校区同步成功举办第十届江苏省大学生知识产权知识竞赛校内初赛。本次活动严格按照江苏省知识产权局、江苏省教育厅、共青团江苏省委赛事要求，依托组委会官方网络答题系统开展，旨在深入贯彻知识产权强国与强省建设纲要，提升我校学生知识产权素养，营造尊重知识、崇尚创新的校园法治氛围。

本次初赛严格采用组委会官方网络答题系统，统一于上午 9:00—10:30集中进行。为确保竞赛有序开展，参赛同学们提前抵达考场，有序完成身份核验、设备调试与系统登录等准备工作，江宁、常州两校区同步准时开赛，全程组织规范、公平公正、公开透明。考试过程中，两校区赛场始终安静有序，同学们全神贯注、凝神思考、认真作答，自觉遵守竞赛纪律，以沉稳的状态完成全部答题，充分展现出扎实的知识储备与严谨端正的学风风貌。

初赛结束后，学院依据官方公布的初赛成绩严格审核、公平遴选，并严格遵照赛事“参赛队伍须男女均有”的要求，最终确定由唐子涵、李益萱、刘智炜三位同学组成的代表队，成功晋级省级复赛。

此次两校区联动初赛，既圆满完成校内选拔任务，也做到以赛促学、以学促行，普及知识产权知识，厚植校园创新创造与产权保护的文化土壤，最后预祝三位同学在复赛中沉着应战、全力以赴，再创佳绩，为校争光！（供稿：陈颖，审核：石依依、宋伟卫、刘成钢）
