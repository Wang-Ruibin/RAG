# 河海大学法律援助中心参与的项目入选2025第二届“高质量党建引领乡村全面振兴”创新优秀案例

> 来源: https://law.hhu.edu.cn/2026/0513/c6356a326899/page.htm

近日，2025第二届“高质量党建引领乡村全面振兴”创新优秀案例揭晓，由中共南京市鼓楼区委热河南路街道工作委员会主持，河海大学法律援助中心参与的项目《党建筑桥促联心 多元协同聚合力——南京市鼓楼区热河南路街道“连心桥•联心调”党建引领矛盾纠纷多元化解实践》入选创新优秀案例。本次活动由新华日报社、中共江苏省委党校、群众杂志社、江苏省乡村振兴研究会、乡村干部报社、党的生活杂志社联合主办，面向全国党政机关、社会团体、企事业单位公开征集，南京仅此项目入选。

法学院党委自2023年始与南京市鼓楼区热河南路街道党工委合作共建，在河海大学法律援助中心执行主任范艳萍老师的组织与带领下，河海大学法律援助中心陆续与街道合作开展了“普法护江青年行”“观摩水上执法，感悟水上派出所，守护长江安澜”等主题党日活动，深入街道开展普法宣传，参与社区基层治理，为热南地区建设和民生服务作贡献。

本次申报案例紧扣乡村全面振兴要求，聚焦党建嵌入产业发展、基层治理、服务体系建设等重点领域，同时展现了“组织建在链上、服务沉在链上、资源聚在链上、作用融在链上”的产业链党建探索，充分体现了基层党组织战斗堡垒作用和党员先锋模范作用。（供稿：陈文琪；审核：刘成钢）
