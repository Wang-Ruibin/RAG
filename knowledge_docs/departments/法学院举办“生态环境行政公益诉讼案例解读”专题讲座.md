# 法学院举办“生态环境行政公益诉讼案例解读”专题讲座

> 来源: https://law.hhu.edu.cn/2026/0526/c11333a327324/page.htm

为深化学生对生态环境司法保护机制的认知，提升环境法治素养与实践能力，5月12日下午14:00，法学院在常州校区2号楼举办"生态环境行政公益诉讼案例解读"专题讲座，本次讲座由吴志红副院长主讲。法学院学生积极参与本次活动，现场氛围浓厚。

讲座伊始，吴院长首先带领同学们回顾了检察公益诉讼制度的发展历程。从2014年党的十八届四中全会提出探索建立检察机关提起公益诉讼制度，到2015年全国人大常委会授权开展试点，再到2017年民事诉讼法、行政诉讼法正式修改确立该制度，她系统梳理了公益诉讼从试点探索到全面推开的发展过程。她指出，目前检察公益诉讼已从最初的“四大领域”逐步扩展为“4+N”格局，涵盖未成年人保护、个人信息保护、反电信诈骗等多个领域，而生态环境始终是其中最重要的实践方向之一。吴院长还向同学们介绍了《检察公益诉讼法》的最新立法动态，引导大家关注公益诉讼制度的发展前沿。

在案例讲解环节，吴院长围绕立案、诉前程序与审判阶段，通过多个典型案例展现生态环境行政公益诉讼的实践逻辑。她以河南三门峡违建塘坝危害高铁安全案为例，介绍了“预防性公益诉讼”理念，说明检察机关如何在重大风险尚未造成实际损害前提前介入，通过诉前检察建议推动隐患治理；结合徐州危险废物非法倾倒案，分析了刑事诉讼、民事公益诉讼与行政公益诉讼协同运用的实践路径；围绕万峰湖流域生态环境受损案，讲解了最高人民检察院通过“一体化办案”破解跨区域生态治理难题的创新模式。此外，她还结合吉林德惠垃圾堆放案、海南三亚违法取用地下水案等案例，重点分析了行政机关“监督管理职责”的认定以及公益诉讼在司法监督中的特殊作用。吴院长指出，检察公益诉讼不仅关注案件胜诉，更关注公共利益是否真正得到修复和保护。

讲座最后，吴院长从制度层面对检察公益诉讼的功能定位进行了进一步思考。她表示，生态环境治理往往涉及跨区域、多部门协同，仅依靠单一机关难以彻底解决问题。近年来，检察机关与水利、生态环境、自然资源等部门不断加强协作，推动公益诉讼从“监督对抗”逐渐走向“协同治理”。监督行政机关依法履职是手段，维护社会公共利益才是最终目标。

整场讲座案例丰富、逻辑清晰，将法律条文与司法实践紧密结合，使同学们更加直观地理解了生态环境行政公益诉讼的制度价值与现实意义，也进一步激发了大家对于法治实践与公共利益保护的关注与思考。（供稿：哈德思，审核：石依依、宋伟卫、刘成钢）
