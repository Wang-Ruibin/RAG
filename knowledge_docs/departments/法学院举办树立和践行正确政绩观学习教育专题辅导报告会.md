# 法学院举办树立和践行正确政绩观学习教育专题辅导报告会

> 来源: https://law.hhu.edu.cn/2026/0513/c6356a326901/page.htm

为深入学习贯彻习近平总书记关于树立和践行正确政绩观的重要论述，5月12日，法学院党委邀请马克思主义学院周忠丽教授作“学习贯彻习近平总书记关于树立和践行正确政绩观的重要论述”专题辅导报告。学院领导班子成员、党委委员、师生党支部书记参加学习。

周忠丽教授紧扣“树立和践行正确政绩观”主题，结合习近平总书记关于树立和践行正确政绩观的重要论述，从“为谁树政绩”“树什么样的政绩”“靠什么树政绩”“如何评价政绩”四个方面系统阐释了正确政绩观的核心要义与实践路径，并结合实际案例生动展现了政绩观偏差的主要表现形式。报告主题鲜明、深入浅出，为党员、干部全面深入理解和把握习近平总书记关于树立和践行正确政绩观的重要论述提供了有力指导。

与会人员纷纷表示，通过本次专题辅导，深刻领会了习近平总书记关于树立和践行正确政绩观重要论述的丰富内涵与实践要求，今后要真正把学习成效转化为树立和践行正确政绩观的思想自觉、行动自觉，切实把学习成果融入服务师生、履职尽责全过程。（供稿：陈文琪；审核：刘成钢）
