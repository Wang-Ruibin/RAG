# 法学院联合土木院成功举办草坪音乐节活动

> 来源: https://law.hhu.edu.cn/2026/0618/c6346a328227/page.htm

为丰富校园文化生活，缓解学生学业压力，促进同学们之间的交流互动，2026年5月29日晚，由两院学生会联合主办的“春意为弦，心声作歌”草坪音乐节在西操场举行。晚风、草坪、音乐与青春交织，数百名师生于此夜共同奔赴一场充满艺术气息与温暖力量的视听盛宴。

暖场环节，蒋宇韬同学带来吉他弹唱《他夏了夏天》，轻快旋律让草坪逐渐升温。随后两位主持人热情登场，以“全员不设限，快乐无上限”的口号正式拉开音乐节序幕。龙狮社以《遇见醒狮》震撼开场，鼓点激昂、狮姿灵动，瞬间点燃全场。

演出精彩纷呈。李天宇、孙绍宁同学以萨克斯伴奏演绎《美丽的神话》与《恋人》，悠扬旋律为草坪铺上温柔底色；李啸冉、蒋宇韬、李佳颖等同学带来《心墙》《爱错》《我怀念的》等歌曲，深情唱腔引发全场跟唱。

国潮元素多次掀起高潮——龙狮社再度登台献演《盆青》，传统舞狮与现代灯光相得益彰；章河军同学以一曲箫乐《飞雪玉花》展现中式浪漫，古韵悠长。张瑜琛同学的英文热单《Beauty and a beat》注入动感活力，朱淏岩用《寂寞先生》带来深沉治愈，张伟伟的《水星记》温柔诉说宇宙级深情，唐拜尔迪以《别怕变老》鼓励青春热爱永不停歇。整场演出掌声与欢呼此起彼伏。

抽奖环节把现场气氛推向高潮，欢呼声与掌声交织在草坪上空，为下半场的演出注入了更多热情。随后，季雨璇、郑雅婷同学带来对唱《一笑倾城》，甜美默契的配合赢得阵阵掌声；穆斯塔帕的吉他弹唱《Shape of my heart》用经典旋律打动人心；章河军以一曲箫乐《飞雪玉花》展现中式浪漫，古韵悠长；张伟伟的《水星记》温柔诉说宇宙级深情；唐拜尔迪以《别怕变老》鼓励青春热爱永不停歇。

一曲终了，今夜所有的温柔、热血与感动在掌声与欢呼中圆满落幕。

本次音乐节为两院学子搭建了释放压力、展示才艺的舞台，更以音乐为桥拉近了同学们之间的距离，共同沉浸在充斥着乐声的氛围中。活动筹备充分、组织有序，现场氛围热烈而不失温馨，充分展现了河海学子积极向上、热爱生活的精神风貌。

未来，法学院学生会将会携手更多院系、社团、部门，推出更多有温度、有创意的校园文化活动，以艺术为桥、以青春为名，为校园生活注入源源不断的活力。（供稿：吴佳悦；审核： 石依依、宋伟卫、刘成钢 ）
