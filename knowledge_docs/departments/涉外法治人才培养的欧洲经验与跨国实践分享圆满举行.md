# 涉外法治人才培养的欧洲经验与跨国实践分享圆满举行

> 来源: https://law.hhu.edu.cn/2026/0312/c11333a323538/page.htm

3 月 5 日下午 2 点，河海法学大讲坛 154 期在行政楼 902 顺利开讲。本次讲坛特邀欧洲华人律师协会副会长、西班牙执业律师徐晶福先生担任主讲人，为法学院师生带来题为“涉外法治人才培养的欧洲经验与跨国实践分享”的精彩讲座，从海外执业视角解锁涉外法治人才成长密码。

徐晶福律师深耕商法、跨境收并购、投融资与国际贸易等领域，是马德里大律师峰会欧亚峰会首位亚裔主讲人，拥有西班牙马德里卡洛斯三世大学法律学士及法律与国际商法双硕士学历，兼具丰富的海外执业经验与多元的国际视野。讲座中，徐律师首先厘清涉外法律服务的中国特色概念，结合全国涉外律师人才库数据，指出当前国内高端复合型涉外法治人才的巨大缺口，尤其强调小语种涉外律师的稀缺性与发展潜力。

围绕涉外法治人才培养，徐律师分享了欧洲法律职业发展的实操经验，从律所职业分级、实习求职技巧到海外留学规划，内容干货满满。他特别强调，扎实的法律基础、优秀的英语能力是核心，主动沟通、合群协作的素养尤为重要，而关注时事、积累跨文化认知更是涉外法律服务的必备能力。针对简历制作与面试技巧，徐律师给出实用建议：简历精简至一页纸，突出法律相关经历；面试时放平心态，展现自身优势与适配性。

同时，徐律师结合中国企业“走出去”的发展趋势，分析了涉外法律服务的两大核心方向，讲解了绿地投资等跨境投资模式，强调从事涉外法律工作需兼具国际视野、商业惯例认知与当地文化尊重。他还介绍了中资律所在海外的发展现状，鼓励同学们把握国家涉外法治人才培养的战略机遇，积极参与实习、模拟法庭等实践活动，提升综合能力。

互动环节中，徐律师针对同学们提出的国内外法学就业差异、中资律所海外布局、实习经历取舍等问题逐一细致解答，并为有海外留学、涉外法律实习规划的同学提供了宝贵的资源与建议。现场师生积极提问，交流氛围热烈。

此次讲座为法学院学子搭建了与海外资深律师交流的平台，让同学们对涉外法治职业发展有了更清晰的认知，也为学院涉外法治人才培养提供了有益的欧洲经验参考。河海大学法学院始终坚持复合型法治人才培养路径，结合学校水利、环境等优势学科，不断拓宽学子专业视野，未来也将持续开展各类高水平学术讲座，助力学子成长为适应国家发展需求的优秀法治人才。
