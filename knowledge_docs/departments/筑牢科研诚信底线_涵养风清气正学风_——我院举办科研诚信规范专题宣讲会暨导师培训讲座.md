# 筑牢科研诚信底线 涵养风清气正学风 ——我院举办科研诚信规范专题宣讲会暨导师培训讲座

> 来源: https://math.hhu.edu.cn/2026/0629/c18393a330728/page.htm

为深入推进科研诚信建设，强化学术规范管理，压实研究生导师立德树人责任，涵养严谨务实学风，近日，我院举办“数学学院科研诚信规范专题宣讲会暨导师培训讲座”。学院副院长陈华教授作专题宣讲，学院研究生导师、近五年入职青年教师，以及硕士研究生和博士研究生参加学习交流。

陈华教授围绕科研诚信建设的重要意义、政策法规要求、典型案例警示、学术规范指导和责任体系落实等方面展开讲解。他指出，科研诚信是科技创新的生命线，是科研活动必须坚守的基本准则。对于数学等基础学科而言，科研诚信不仅体现在数据真实、结论可靠，也体现在定义清晰、证明严密、算法可复现、程序可核验等具体环节。他要求，全院师生应坚持“四个面向”，把学术追求融入国家科技自立自强和高水平人才培养大局之中。

本次讲座还就导师育人职责和师德师风建设进行了研讨和交流。陈华教授指出，导师是研究生培养第一责任人，应加强对学生选题、文献阅读、科研实施、论文写作、投稿发表等全过程指导，要及时掌握学生研究进展，帮助学生树立正确的学术观、科研观和成才观。同时，导师应坚持言传身教，规范署名秩序，尊重学生权益，营造平等、健康、严谨的师生关系。

本次科研诚信规范专题宣讲会暨导师培训旨在帮助全院师生明晰科研诚信规章制度，厘清学术不端边界，提升科研风险防控能力。今后，学院将持续加强科研诚信教育和过程管理，完善学院、导师、学生协同联动的责任体系，引导师生共同维护风清气正的学术生态，为学院高质量人才培养和高水平科研创新提供坚实保障。
