# 陈达副校长来我院调研指导工作

> 来源: https://math.hhu.edu.cn/2024/0312/c18393a274399/page.htm

3月8日，陈达副校长一行来我院进行调研指导工作，学院党委书记潘静、副院长杨永富、副院长陈华、院党委副书记曹松祥等学院领导及教师代表出席调研座谈会。

会议伊始，杨永富副院长汇报了数学学院的基本情况，并对科学研究、研究生培养、本科教学、师资队伍建设等学院2024年重点工作进行了详细介绍，特别是学院围绕一级博士点申报开展的相关工作和存在的困难，剖析了目前学院发展遇到的难点和未来规划。

潘静书记介绍了数学学院成立以来相关工作情况，并从数学学科的特点等角度，分析了当前学院在人才引进、科研工作等方面存在的问题与改进方向，围绕推动落实学院“数学和水利交叉研究中心”建设运行做了较为细致的汇报。

陈华副院长介绍了目前学院研究生培养方面的情况，提出要抓好研究生人才培养与扶持、教师引进与考核，客观评价了学院在科 研、基金项目等方面的成绩和存在的问题。陈华副院长与参会人员围绕考核评价、教学、科研情况等进行了热烈的讨论，并对学院发展和学科建设中的相关问题，充分交换了意见。

陈达副校长对学院取得的相关成绩给予了肯定，表达了学校对学院发展和数学学科建设的高度重视，并对学院近期的重点工作、难点问题提出了指导性意见，为学院发展指明了方向。同时，强调做好兄弟学校对比分析，发挥学科交叉优势，引进来与走出去结合，抓好科学建设，满怀信心应对挑战，脚踏实地共谋发展。

本次调研座谈会为学院发展谋划了整体布局，推动学院建设迈上新的台阶。学院将深入落实教育强国战略，在民族复兴伟业的道路上凝心聚力、争创佳绩。
