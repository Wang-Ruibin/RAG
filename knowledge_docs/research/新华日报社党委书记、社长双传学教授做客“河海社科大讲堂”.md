# 新华日报社党委书记、社长双传学教授做客“河海社科大讲堂”

> 来源: https://skc.hhu.edu.cn/2024/1030/c3871a290715/page.htm

10月26日，河海社科大讲堂在常州校区举办。新华日报社党委书记、社长，新华报业传媒集团董事长双传学教授应邀作《深入学习贯彻习近平总书记关于全面深化改革的一系列新思想、新观点、新论断》的专题讲座。副校长陈达出席并主持专题讲座。 双传学在报告中对深刻领会习近平总书记关于全面深化改革的一系列新思想、新观点、新论断的重大意义、丰富内涵和实践要求等方面进行了系统阐释和深入解读。他指出，习近平总书记关于全面深化改革的一系列新思想、新观点、新论断，是新时代全面深化改革理论创新的最新成果，是进一步全面深化改革的行动指南。学好用好这些新思想、新观点、新论断，对于统一和引领全党全国各族人民思想和行动，以更加奋发有为的精神状态进一步全面深化改革，具有重要意义。高举中国特色社会主义伟大旗帜，做习近平新时代中国特色社会主义思想的坚定信仰者和忠实实践者，坚定信心、同心同德，以身作则、以上率下，为全面建设社会主义现代化国家、全面推进中华民族伟大复兴作出应有的贡献！ 河海社科大讲堂是由社科处主办、相关文科院系及省部级科研基地承办的名家讲堂，旨在搭建高层次的学术交流平台，营造校园高品质人文学术氛围，进一步推动学校人文社会科学繁荣发展。本期由江苏省习近平新时代中国特色社会主义思想研究中心河海大学基地、马克思主义学院承办。 学校相关职能部门及文科院系负责人、师生代表等 180 余人参会。
