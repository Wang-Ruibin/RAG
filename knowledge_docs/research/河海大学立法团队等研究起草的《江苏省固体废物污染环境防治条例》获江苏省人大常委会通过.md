# 河海大学立法团队等研究起草的《江苏省固体废物污染环境防治条例》获江苏省人大常委会通过

> 来源: https://skc.hhu.edu.cn/2025/0315/c3871a299592/page.htm

近日，江苏省第十四届人民代表大会常务委员会第十二次会议表决通过了修订的《江苏省固体废物污染环境防治条例》（以下称《条例》），自2025年3月1日起施行。在此次《条例》修订工作中，河海大学法学院李义松立法团队与生态环境部南京环科所、南京市环科院、江苏省环境工程技术有限公司联合开展《<江苏省固体废物污染环境防治条例>修订前期研究》。 联合课题组在江苏省生态环境厅、江苏省固体废物监督管理中心的指导下，以江苏省法学会环境资源法学研究会为平台，汇聚全省理论与实务界的智慧力量，积极征求各方意见建议，广泛开展了实证调研，经过多次修改完善，完成并提交修订草案送审稿、说明、对照表及研究报告。送审后，李义松还以专家身份深度参与了省司法厅和省人大常委会的后续立法工作。 随着江苏经济快速发展，城镇化进程加快，每年固体废物产生量超过3亿吨，新形势对江苏固体废物污染环境防治的法律支撑提出了新的要求，现行《条例》迫切需要修订。新《条例》共9章82条，《条例》基于江苏的具体情况和实际需要，坚持问题导向，突出江苏特色，围绕“无废城市”建设目标，以工业固体废物、生活垃圾、建筑垃圾、农业固体废物、危险废物为重点，从规划、标准、基础设施建设、收运体系建设等方面作了全面规定，切实提升固体废物源头减量、资源化利用和无害化处置水平。 李义松，法学院教授，江苏省法学会环境资源法学研究会会长，带领团队主持完成国家社科基金、省部基金项目及政府招标项目70余项，包括地方立法研究项目10余项，其中8部江苏省地方性法规在省人大表决通过。
