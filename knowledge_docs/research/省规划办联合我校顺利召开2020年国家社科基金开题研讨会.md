# 省规划办联合我校顺利召开2020年国家社科基金开题研讨会

> 来源: https://skc.hhu.edu.cn/2020/1127/c3871a215906/page.htm

11月25日,省哲学社会科学规划办联合我校社科处在闻天馆209召开2020年国家社科基金开题研讨会，省规划办领导做开题指导发言,点评专家、2020年新立项的国家社科基金项目负责人以及文科学院师生参加了研讨会，会议由李娜副处长主持。 省规划办领导指出十九大后，社会进入一个新时期，处于历史新方位，应以习近平新时代中国特色社会主义思想为指导，用新思想新观点新论断，为全面建设社会主义现代化国家提供有力的理论支持和学理支撑。接着他对课题研究提出了要求：一是注意成果的原创性，创新是时代主题，也是考量社科研究成果质量的重要衡量标准；二是注意成果形式规范性，课题组应对参考文献、引用等规范性操作予以重视，规范性是衡量课题成果是否合格的重要因素；三是强调基础研究应结合理论前沿，应用研究结合国家发展重大需求，当下应结合十九届五中全会精神积极开展社科领域研究；四是注意课题成果意识形态问题；五是注重团队人才培养，带动青年人才发展；六是强调社科研究应注重构建3个体系，通过结合我校水优势学科和水特色研究，建立河海社科研究的学科体系、学术体系、话语体系。 项目负责人通过PPT介绍了基金课题研究目标、研究思路、研究方法、重点难点，点评专家在研究重点聚焦、研究思路合理性、研究操作可行性、研究难点攻破等方面提出重要观点与建议，鼓励课题组在专业精细化研究基础上进行学科交叉研究项目，负责人认真聆听专家的点评，不时的与专家进行互动交流。 本次开题研讨会顺利举办，为课题组进行后续课题研究做好了铺垫。社科处将长期关注并跟踪项目研究进展，及时做好项目管理服务工作，帮助项目负责人高质量完成课题研究，以期顺利结项！

项目负责人通过PPT介绍了基金课题研究目标、研究思路、研究方法、重点难点，点评专家在研究重点聚焦、研究思路合理性、研究操作可行性、研究难点攻破等方面提出重要观点与建议，鼓励课题组在专业精细化研究基础上进行学科交叉研究项目，负责人认真聆听专家的点评，不时的与专家进行互动交流。

本次开题研讨会顺利举办，为课题组进行后续课题研究做好了铺垫。社科处将长期关注并跟踪项目研究进展，及时做好项目管理服务工作，帮助项目负责人高质量完成课题研究，以期顺利结项！
