# 第四届长江经济带高质量发展论坛在我校举办

> 来源: https://skc.hhu.edu.cn/2025/0315/c3871a299591/page.htm

12月15日，第四届长江经济带高质量发展论坛在我校举办。江苏省人大常委会副主任、江苏省社科联主席曲福田，中国工程院院士、长江设计集团首席科学家钮新强，南京大学原党委书记、人文社会科学资深教授洪银兴，重庆市社科联党组书记、副主席李保海，湖北省社科联党组成员、副主席刘卫桥，江西省社科联党组成员、副主席许璟出席开幕式。开幕式由江苏省社科联党组书记、常务副主席张新科主持。校党委书记杨桂山、校长郑金海出席论坛。 曲福田在致辞中强调，推动长江经济带高质量发展，关键在创新、核心在机制。社科界要充分发挥理论优势，提高政治站位、深化理论研究，坚持创新驱动、增强发展动能，强化区域协同、实现协调共进，为党和政府科学决策、有效施策提供重要参考，为构建中国哲学社会科学自主知识体系、推动长江经济带高质量发展贡献更多智慧、更大力量。 郑金海在致辞中表示，推动长江经济带高质量发展是一项长期而艰巨的任务，论坛旨在贯彻落实党的二十届三中全会精神和习近平总书记关于推动长江经济带的重要论述，通过深入研究与探讨，汇聚各方力量与智慧，进一步加强交流与合作，共同探索长江经济带发展的新思路、新路径、新举措，为推动长江经济带高质量发展作出新的更大贡献。 李保海在代表省市社科联发言中表示，本次论坛是充分发挥社科界思想库、智囊团作用，推动中东西互动合作的具体实践。各省市社科联紧紧围绕长江生态环境保护修复、产业绿色转型升级、长江文化保护传承等重大问题，组织广大社科界广泛调查，深入研究，取得了一系列原创性研究成果，为长江经济带高质量发展贡献了社科力量。 主旨报告中，钮新强围绕国家水网高质量建设阐述了相关理论及思考。他提出应坚持“节水优先、空间均衡、系统治理、两手发力”的治水思路，坚持“四水四定”原则，以南水北调工程为骨干，以区域水网为补充，立足长远，科学构建国家水网，推动相关领域的科技创新，为重要经济区、重要城市群、重要能源基地、粮食主产区、重点生态功能区提供有力的水安全保障。 洪银兴从统一大市场建设的视角探讨长江经济带城市现代化。他认为，在现代化建设的新征程中，城市应发挥现代化的中心作用，建议从提高人口密度、提升城市的产业创新能力、推动中心和外围地区一体化和城市更新等途径来实现城镇的城市化和城市的现代化。 杨桂山的报告中，阐述了长江中游鄱阳湖与长江的江湖自然连通关系，介绍了对鄱阳湖湖区子湖、蝶形湖及入江水道等水文变化情况开展的研究，对改善鄱阳湖枯水年退水期和枯水期水位偏低的状况提出了多路径协同的建议。 中共中央党校（国家行政学院）经济学部副主任许正中教授、国务院发展研究中心资源与环境政策研究所研究员、中国自然资源学会副理事长、全国政协委员谷树忠，江苏省政府研究室原副主任、资源环境与发展研究院首席专家沈和，江苏省战略与发展研究中心主任王国亮，中国科学院南京地理与湖泊研究所副所长、研究员段学军分别从人才强国战略、生态环保产业、百强县案例、发展评估等角度分析了长江经济带高质量发展的动力机制。 论坛报告中，与会专家围绕论坛主题，共同研讨理论热点、跟踪实践前沿、开展学术交流，从长江水文化、河湖治理等角度阐述了长江经济带高质量发展的重要推力。 本届论坛以“以体制机制创新推动长江经济带高质量发展”为主题，由河海大学、南京农业大学、中国科学院南京地理与湖泊研究所主办。来自重庆、江西、湖北、江苏等沿江11个省市社科联、社科院、有关高校、新型智库、中国科学院相关研究所专家学者、师生代表等参加会议。
